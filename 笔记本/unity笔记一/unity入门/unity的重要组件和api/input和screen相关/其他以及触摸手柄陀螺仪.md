一，其他
1.是否有任意键和鼠标长按就执行一段代码
```
if (Input.anyKey)
{
    print("任意键就执代码");
}
```
2.是否有任意键或鼠标按下
```
if (Input.anyKeyDown)
{
    print("任意键就执代码");
}
```
应用场景就是我们可以用在我们按下就换游戏里面的按键
我们可以得到这一帧的按键
```
print(Input.inputString);
```
我们可以配合起来
```
if (Input.anyKeyDown)
{
    print("任意键就执代码");
    print(Input.inputString);

}
```

二，手柄的输入相关
1.我们可以通过一个函数得到手柄的所有的按键的名字
```
string[] strs=Input.GetJoystickNames();
```
2.某一个手柄按钮按下
```
if (Input.GetButtonDown("Jump"))
{

}
```
3.某一个手柄按钮抬起
```
if(Input.GetButtonUp("Jump"))
{

}
```
3.某一个手柄按钮长按
```
if (Input.GetButton("Jump"))
{

}
```

```
if (Input.touchCount > 0)//这里的意思是我们当检测到我们有多个手指输入
{
    Touch t1=Input.touches[0]; 这里的意思是我们的touch的类是一个数组

```
2.是否启用多点触控
```
Input.multiTouchEnabled = true;
```

四，陀螺仪
1.开启陀螺仪
```
Input.gyro.enabled = true;
```
2.获取重力加速度的向量，只读不可以改
```
print(Input.gyro.gravity);
```
3.获取旋转速度
```
print(Input.gyro.rotationRate);
```
4.获取陀螺仪的四元数
可以用来控制场景上的3d物体收到的重力影响
```
print(Input.gyro.attitude);
```
