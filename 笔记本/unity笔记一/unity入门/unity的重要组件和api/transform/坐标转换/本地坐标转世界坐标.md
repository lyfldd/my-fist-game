# Transform 坐标转换（本地坐标 → 世界坐标）

## 概念

本地坐标转世界坐标，是世界坐标转本地坐标的反向操作。当我们知道一个相对于某对象的偏移量（本地坐标），想知道它在世界空间中的绝对位置时使用。

> 此文件内容已整合到「世界坐标转本地坐标.md」，两套坐标的 API 均在其中，包括 `TransformPoint` / `InverseTransformPoint` / `TransformDirection` / `InverseTransformDirection` 的完整用法。

## 核心方法速查

```csharp
// 本地坐标（点）→ 世界坐标（点）
Vector3 worldPoint = transform.TransformPoint(localPoint);

// 世界坐标（点）→ 本地坐标（点）
Vector3 localPoint = transform.InverseTransformPoint(worldPoint);

// 本地方向向量 → 世界方向向量
Vector3 worldDir = transform.TransformDirection(localDir);

// 世界方向向量 → 本地方向向量
Vector3 localDir = transform.InverseTransformDirection(worldDir);
```
