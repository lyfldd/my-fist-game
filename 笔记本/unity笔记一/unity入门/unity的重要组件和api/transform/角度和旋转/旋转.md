# Transform 旋转
## 概念

Transform 提供两种旋转方式：自转（Rotate，在自身轴上旋转）和绕某点旋转（RotateAround）。旋转可以用欧拉角向量或四元数描述，Space 参数决定是相对于世界坐标系还是自身坐标系。


## 用法

### 1. 自转（Rotate）

**重载1：传欧拉角向量**
```csharp
```
// 重载1a：只传欧拉角向量（默认相对自身坐标系）
transform.Rotate(new Vector3(1, 10, 0) * Time.deltaTime);

// 重载1b：传欧拉角向量 + 相对坐标系
transform.Rotate(new Vector3(0, 45, 0), Space.World);
transform.Rotate(new Vector3(0, 45, 0), Space.Self);

**重载2：绕单轴旋转（最常用）**
```csharp
```
// 重载2a：绕Y轴转10度，相对自身坐标系
transform.Rotate(Vector3.up, 10f * Time.deltaTime);

// 重载2b：绕Y轴转10度 + 相对坐标系
transform.Rotate(Vector3.up, 10f * Time.deltaTime, Space.Self);  // 默认
transform.Rotate(Vector3.up, 10f * Time.deltaTime, Space.World);

// 重载2c：绕世界Y轴转
transform.Rotate(Vector3.up, 10f * Time.deltaTime, Space.World);

**重载3：分别传XYZ三个轴的角度（Space.World）**
```csharp
```
transform.Rotate(0, 10f * Time.deltaTime, 0);  // 等价于绕世界Y轴转

**重载4：绕任意轴（世界空间）旋转**
```csharp
```
// Axis 是世界空间轴，angle 是角度，Space.World
transform.Rotate(axis, angle, Space.World);


### 2. 绕某点旋转（RotateAround）

```csharp
```
// 重载：绕 pivot 点，沿 axis 方向旋转 angle 角度
transform.RotateAround(Vector3.zero, Vector3.up, 10f * Time.deltaTime);

参数含义：
- **第一个参数（pivot）**：旋转中心点的世界坐标
- **第二个参数（axis）**：旋转轴的世界方向（必须是单位向量）
- **第三个参数（angle）**：角度（度），正值逆时针


### 两种方式的区别

| 方式 | 作用 | 典型场景 |
|------|------|----------|
| `Rotate` | 自身旋转，在自身轴上原地转动 | 角色转身、风车旋转 |
| `RotateAround` | 绕某个外部支点做圆周运动 | 月球绕地球、轮盘绕轴心 |


## 坑点

- `Rotate` 的欧拉角重载**是增量旋转**（在当前旋转基础上叠加），不是绝对设置。
- `RotateAround` 需要持续在 Update 中调用才有效（每帧转一点），因为它也是增量的。
- 如果对象有父对象，`Space.World` 下 `RotateAround` 的行为会更复杂（因为父对象的旋转也会影响结果）。
- `Rotate` 用 `Space.Self` 时，每次旋转都基于当前旋转后的轴方向（欧拉角增量叠加），可能产生累积误差。


## 应用场景

| 需求 | 推荐方式 |
|------|----------|
| 角色原地转身 | `transform.Rotate(Vector3.up, turnSpeed * Time.deltaTime, Space.World);` |
| 螺旋上升（子弹螺旋） | `transform.RotateAround(pivot, Vector3.forward, spiralSpeed * Time.deltaTime);` |
| 开门动画 | `transform.Rotate(Vector3.up, openAngle, Space.Self);` |
| 月球绕地球 | `moon.transform.RotateAround(earth.position, Vector3.up, moonSpeed * Time.deltaTime);` |


## 源码（部分原理）

```csharp
```
// Rotate 内部（简化）
public void Rotate(Vector3 eulers, Space relativeTo)
{
```csharp
```
Quaternion rotation = Quaternion.Euler(eulers);
if (relativeTo == Space.Self)
```csharp
```
this.rotation = this.rotation * rotation; // 先转后叠（增量）
else
```csharp
```
this.rotation = rotation * this.rotation;
}

绕轴旋转的底层调用了 `Quaternion.AxisAngle`，将"绕某轴转N度"转成四元数。

```csharp
```
// RotateAround 内部（简化）
public void RotateAround(Vector3 point, Vector3 axis, float angle)
{
```csharp
```
Vector3 relativePos = position - point;           // 到支点的向量
Quaternion q = Quaternion.AngleAxis(angle, axis);// 绕轴的旋转四元数
relativePos = q * relativePos;                  // 旋转该向量
position = relativePos + point;                  // 新位置 = 旋转后向量 + 支点
transform.Rotate(axis, angle, Space.World);     // 同步旋转自身朝向
}
