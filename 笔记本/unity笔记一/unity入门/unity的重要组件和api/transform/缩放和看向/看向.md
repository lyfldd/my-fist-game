# Transform.LookAt 看向
## 概念

`LookAt` 让对象的正面（Z轴正方向，即 forward）持续指向某个目标点或另一个对象。它常用于让角色/敌人/相机始终面向目标。


## 用法

### LookAt 所有重载版本

```csharp
```
// 重载1：看向世界坐标中的某个点
transform.LookAt(Vector3.zero);

// 重载2：看向另一个 Transform（对象的当前位置）
transform.LookAt(targetTransform);

// 重载3：看向另一个 Transform，并指定世界Up向量（用于控制"歪头"的幅度）
transform.LookAt(Vector3.zero, Vector3.up);        // 以世界Y轴为上向量
transform.LookAt(targetTransform, Vector3.up);

// 重载4：分别指定目标点和上向量
transform.LookAt(new Vector3(5, 0, 5), Vector3.up);

## 原理

`LookAt` 内部做了两件事：

1. 计算从当前对象到目标的方向向量：`direction = target - position`
2. 用这个方向构造一个四元数旋转，直接赋值给 `transform.rotation`

```csharp
```
// LookAt 内部等价实现（简化）
public void LookAt(Vector3 worldPosition, Vector3 worldUp)
{
```csharp
```
Vector3 direction = (worldPosition - position).normalized;
Quaternion targetRotation = Quaternion.LookRotation(direction, worldUp);
rotation = targetRotation;
}

`Quaternion.LookRotation` 是 LookAt 的底层，它用方向向量和上向量构造一个四元数。


## 坑点

- `LookAt` 会**立即设置**对象的 rotation 到目标方向，没有平滑过渡。如果需要平滑旋转，用 `Quaternion.Lerp` 或 `transform.rotation = Quaternion.Slerp(...)`。
- **不要在 FixedUpdate 中用 LookAt**——FixedUpdate 每帧可能运行多次，会导致抖动。
- 当目标与对象位置完全重合时，`LookAt` 结果不确定（方向向量为零向量），建议加距离判断。
- `LookAt(target, Vector3.up)` 的第二个参数只影响对象的"歪头"程度，不能完全避免倾斜。


## 应用场景

| 需求 | 推荐方式 |
|------|----------|
| 敌人一直面向玩家 | `transform.LookAt(player.position);` |
| 相机看向目标 | `mainCamera.transform.LookAt(target.position);` |
| 子弹/炮弹飞行方向 | `transform.LookAt(targetPoint);` |
| 平滑转向目标（不抖动） | `transform.rotation = Quaternion.Slerp(from, Quaternion.LookRotation(dir), t);` |
