拓展方法就是  为现有的  非静态变量  类型添加  新方法
1.提升程序的拓展性
2.不需要再对对象重写新方法
3.不需要继承来添加新方法
4.为别人封装的类写额外的方法

特点是
1.要写在静态类里面
2.一定是一个静态函数
3.第一个参数是拓展目标
4.第一个参数用this修饰

基本语法是
访问修饰符 static 返回值 函数名字（this 拓展类名 参数名，参数类型 参数名............）
例如 

我们为int拓展了一个成员方法
value代表该方法的实例化对象

 public static void speakvalue(this int value)
{
Console.WriteLine("lyf" + value);
}

拓展方法的使用
int i = 10;
i.speakvalue（）;
输出结果就是lyf10
就相当于我们把这个i=10传进去，然后我们执行这个拓展里面的逻辑

本质就是为了已经存在的结构体或者是类里面增加一个方法

拓展的方法名字和类里面的方法的名字重名了怎么办呢

这个时候我们先用原来的方法
