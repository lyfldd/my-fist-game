那要是想要同时返回两个结果呢我们返回复杂的数据结构就可以了
可以是数字也可以是枚举
static int[] e(int a,int b,int c)
{
    int sum = a + b;
    int jian = a - b;
    int[] arr = { sum,jian}; ;
    return arr;
}
