Queue（队列）是 C# 中先进先出（FIFO, First In First Out）的集合，严格遵循先入队的数据先出队规则，常用于任务排队、消息处理、广度优先搜索（BFS）等场景

C# 提供两种常用队列：
1.非泛型 Queue（System.Collections，不推荐，存在装箱拆箱、类型不安全问题）
2.泛型 Queue<T>（System.Collections.Generic，推荐，类型安全、性能更高）

使用前要引用命名空间
// 泛型 Queue<T>（推荐）
using System.Collections.Generic;

// 非泛型 Queue（不推荐）
using System.Collections;
