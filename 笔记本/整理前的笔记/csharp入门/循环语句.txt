while循环
while（bool类型值）
{
满足条件之后就会执行代码
当代码执行完之后然后就会回到while开头
再次进行条件判断
要是满足就继续执行
要是不满足就跳出
}
这个死循环不太好，尽量不要用
可以加break和contine
while
{
	代码一
	break；
	代码二（这里就不执行）
}

int i = 0;
while (true)
{
    ++i;
    Console.WriteLine(i);
    if (i == 10)  
    {
        break; 
    }
}
Console.WriteLine(i);
break 写在 if 内部，但 if 本身不是可被 break 跳出的结构
break 会向上找最近的 “可跳出结构”，也就是外层的 while 循环。

while
{
	Console.WriteLine(continue前);
	continue；
	Console.WriteLine(continue后);这里不会打印因为到了continue
				会继续调回循环那里
}
