复合运算符+=
和加法一样用自己拼接右侧的内容
但是例如str+=1+2+3+4，会先把右侧的相加为10然后拼接给str，因为从左往右看第一个是数字往后相加
默认是数字
但是例如str+=“”+1+2+3但是这个就是拼接起来因为从左往右第一个是字符串会自动调用
把后面的字符变成字符串类型
但是又有这个str+=1+2+“”+3+4,1+2那里前面没有字符串类型的所以要想加，3+4前面有字符串
不用相加要拼接

条件运算符
比较两个变量和常量的大小
>是否大于
<是否小于
==是否等于
!=是否不等于
>=是否大于等于
<=是否小于等于
比较之后返回的值是一个bool值
语法：bool result = a>b	先算右边然后再赋值给左边

变量和变量的比较
bool result = a>b

变量和数值的比较
bool result = a>10

数值和数值的比较
bool result = 5>10

计算结果比较 说明条件运算符比算数运算符的级别低
bool result = a+3>a-2

但是不能进行范围的比较
例如判断1<a<6不能这样写

不同变量类型的比较；不同的数值类型可以进行条件运算符的比较
例如int float double等等可以相互比较
特殊类型如char string bool只能进行同类型的的== !=比较
但是char可以和数值类型的比较，还可以和字符类新的比较就是用ask码来比较

逻辑运算符
逻辑与&&	有假则假，同真为真，对两个bool值进行运算  优先级低于条件运算符
bool result =bool变量 && bool变量
也可以多次运用bool result =bool变量 && bool变量bool变量 && bool变量
从左到右第一个执行了是错的那就后面的不执行

逻辑或|| 有真则真，同假则假 对两个bool值进行运算
bool result =bool变量 || bool变量
从左到右第一个执行了是真的那就后面的不执行

逻辑非！ 对bool值进行取反 真变假 假变真	逻辑非的优先级较高仅仅低于括号
result =！bool变量

总结一下混合使用的优先级问题
逻辑非最高，逻辑与高于逻辑或，逻辑算术运算符低于算术运算符 条件运算符

逻辑运算符的短路规则
&&从左到右第一个执行了是错的那就后面的不执行
|| 从左到右第一个执行了是真的那就后面的不执行

位运算符
位与&   连接两个数值进行位运算，将数值转化为二进制，对位运算有0则0
int a=1 //001
int b=5 //101
int c=a&b 那c为001就是1
要是有多个位与那就从左到右运算

位或|   连接两个数值进行位运算，将数值转化为二进制，对位运算有1则1
要是有多个位或那就从左到右运算

异或^   连接两个数值进行位运算，将数值转化为二进制，对位运算相同为0，不同为1

位取反~  将一个数值转化为二进制，0变1,1变0

左移<<让一个数的二进制左移几位就在右侧加几个0左移几位就是乘以 2 的 n 次方
右移>>让一个数的二进制右移几位就在左侧加几个0右移几位就是除以 2 的 n 次方

三目运算符
基本语法三个空位，两个符号
空位？空位：空位；第一个空位是bool的值，第二个空位是返回第一个空位的bool值为真的值
第三个空位是返回第一个空位的bool值为假的值
例如 一个变量=bool值？a：b;要是bool为真那就返回a要是假的就返回b
第二三个空位什么类型都可以但是要两个类型都一样
