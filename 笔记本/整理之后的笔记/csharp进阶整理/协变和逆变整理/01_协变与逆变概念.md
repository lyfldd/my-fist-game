# 01_协变与逆变概念

## 一、问题背景

**协变和逆变是用来解决泛型不支持父类和子类自动转换的问题**

普通泛型不支持继承关系：
```csharp
List<Boss> bosses = new List<Boss>();
List<Enemy> enemies = bosses;  // ❌ 编译错误！
```

---

## 二、核心口诀

| 类型 | 关键字 | 适用场景 | 转换方向 |
|---|---|---|---|
| 协变 | out | 返回值 / 只读接口 | 子类泛型 → 父类泛型 |
| 逆变 | in | 方法参数 / 输入接口 | 父类泛型 → 子类泛型 |

**口诀**：
```
协变 out → 返回值用 → 子类泛型 → 父类泛型
逆变 in → 参数用 → 父类泛型 → 子类泛型
```

---

## 三、协变（Covariant）

### 关键字：out

### 作用
允许**子类泛型 → 父类泛型**转换，支持"只读"场景（返回值 / 输出）

### 使用场景
- 通用刷怪机制
- 遍历所有敌人（IEnumerable<T>）
- AOE 技能

### 代码示例：IEnumerable 协变
```csharp
class Enemy
{
    public int Hp;
    public string Name;
    public Enemy(int hp, string name)
    {
        Hp = hp;
        Name = name;
    }
    public virtual void TakeDamage(int damage)
    {
        Hp -= damage;
    }
}

class Boss : Enemy
{
    public Boss(int bossHp, string name) : base(bossHp, name) { }
    public override void TakeDamage(int damage)
    {
        Hp -= damage;
    }
}

// 攻击所有敌人（用父类接口）
static void DamageAll(IEnumerable<Enemy> enemies)
{
    foreach (var enemy in enemies)
    {
        enemy.TakeDamage(10);
    }
}

// 使用（协变生效：IEnumerable<Boss> → IEnumerable<Enemy>）
List<Boss> enemies = new List<Boss>();
enemies.Add(new Boss(100, "boss1"));
enemies.Add(new Boss(200, "boss2"));

DamageAll(enemies);  // ✅ 无需转换，直接使用
```

### 代码示例：委托协变
```csharp
// 定义怪物登场
public virtual void SayHello()
{
    Console.WriteLine($"敌人 {Name} 出现了！血量：{Hp}");
}

// Boss 实现
public override void SayHello()
{
    Console.WriteLine($" Boss {Name} 登场！血量：{Hp}");
}

// 协变委托：生成怪物（out T 表示返回值支持协变）
public delegate T SpawnHandler<out T>();

static void DoSpawn(SpawnHandler<Enemy> spawnHandler)
{
    Enemy enemy = spawnHandler();
    enemy.SayHello();
}

static void Main()
{
    // 定义生成 Boss 的委托（SpawnHandler<Boss>）
    SpawnHandler<Boss> spawnBoss = () => new Boss(5000, "远古巨龙");

    // 协变生效：SpawnHandler<Boss> 直接传给 SpawnHandler<Enemy>
    DoSpawn(spawnBoss);  // ✅ 无需转换，直接使用
}
```

## 四、逆变（Contravariant）

### 关键字：in

### 作用
允许**父类泛型 → 子类泛型**转换，支持"只写"场景（方法参数 / 输入）

### 使用场景
- 通用受伤处理
- 事件回调
- AI 行为系统

### 代码示例
```csharp
class Enemy
{
    public string Name;
    public int Hp;
    public Enemy(string name, int hp) { Name = name; Hp = hp; }
    public virtual void TakeDamage(int damage) => Hp -= damage;
}

class Boss : Enemy
{
    public Boss(string name, int hp) : base(name, hp) { }
}

// 逆变委托：处理受伤（in T 表示参数支持逆变）
public delegate void DamageHandler<in T>(T enemy, int damage);

static void Main()
{
    // 1. 定义处理所有 Enemy 受伤的委托（父类泛型）
    DamageHandler<Enemy> hurtEnemy = (enemy, damage) =>
    {
        enemy.TakeDamage(damage);
        Console.WriteLine($"{enemy.Name} 受到 {damage} 伤害，剩余血量：{enemy.Hp}");
    };

    // 2. 逆变生效：父类委托直接赋值给子类委托
    DamageHandler<Boss> hurtBoss = hurtEnemy;

    // 3. 直接用通用委托处理 Boss 受伤
    Boss dragon = new Boss("远古巨龙", 5000);
    hurtBoss(dragon, 1000);  // ✅ 无需为 Boss 单独写受伤逻辑
}
```

### 完整示例：受伤系统重构
```csharp
// 删掉每个子类都要写的受伤方法后，用逆变委托统一处理
public delegate void GetDamage<in T>(T t, int damage);

// 写一个全部受伤的通用处理
GetDamage<Enemy> enemygetdamage = (Enemy enemy, int damage) =>
{
    enemy.Hp -= damage;
    Console.WriteLine($"{enemy.Name} 受到 {damage} 点伤害！剩余血量：{enemy.Hp}");
};

// 使用
SpawnHandler<Boss> spawnBoss = () => new Boss(5000, "lyf");
Boss boss = DoSpawn(spawnBoss) as Boss;
enemygetdamage(boss, 1000);  // ✅ 用父类委托处理子类对象
```

## 五、协变逆变的本质理解

### 形象比喻

| 概念 | 比喻 |
|---|---|
| 协变 out | 生产者：子类工厂可以当父类工厂用 |
| 逆变 in | 消费者：父类处理器可以当子类处理器用 |

### 内存安全理解

```
协变（out）：只往外拿东西，不往里放
→ 安全：拿出来的都是子类对象，可以当父类用

逆变（in）：只往里塞东西，不往外拿
→ 安全：塞进去的都是父类引用，可以兼容子类
```

---

## 六、修正错误汇总表

| 错误认知 | 正确理解 |
|---|---|
| 泛型支持子类向父类自动转换 | ❌ 普通泛型不支持，必须加 out/in |
| out 关键字可以用于参数 | ❌ out 只能用于返回值（生产端） |
| in 关键字可以用于返回值 | ❌ in 只能用于参数（消费端） |
| 协变和逆变可以同时用 | ❌ 一个泛型参数要么是 out（协变）要么是 in（逆变），不能同时 |

## 七、遗漏知识点补充

### 常见的协变接口（out 在 .NET 中已默认支持）
```csharp
IEnumerable<T>      // 协变，默认支持
IEnumerator<T>       // 协变
IQueryable<T>        // 协变
IComparer<T>        // 逆变
Predicate<T>         // 逆变
Converter<TInput, TOutput>  // TOutput 协变，TInput 逆变
```

### 游戏开发实战场景
| 场景 | 用什么 | 示例 |
|---|---|---|
| 统一管理所有敌人 | IEnumerable<T>（协变） | `void ShowAll(IEnumerable<Enemy>)` 接收任意子类型敌人列表 |
| 统一的受伤处理 | 逆变 in | `void ApplyDamage(DamageHandler<Enemy>)` 可处理任意子类型 |
| 同时需要读写 | 不能用协变逆变 | 用普通的泛型约束 `where T : Enemy` |
