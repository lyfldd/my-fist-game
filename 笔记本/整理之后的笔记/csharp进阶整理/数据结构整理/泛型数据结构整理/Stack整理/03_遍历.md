Stack<T> —— 遍历
=============================================

一、遍历方式

// 方式 1：foreach（从栈顶到栈底，不删除）
foreach (string item in stack)
{
    Console.WriteLine(item);
}

// 方式 2：循环弹栈（每次取出并删除，直到栈空）
while (stack.Count > 0)
{
    var item = stack.Pop();
    Console.WriteLine("弹出：" + item);
}


二、应用场景

- 撤销操作（Ctrl+Z）
- 关闭所有 UI 面板
- 回溯寻路路径
- 清空路径点

循环弹栈会把栈清空，遍历完就不能再用了。
如果想保留栈，只看不删，用 foreach。
