Stack<T> —— 修改元素
=============================================

Stack<T> 没有直接修改指定位置元素的方法，有两种变通思路：


一、值类型：复制到数组改完再重建

Stack<int> stack = new Stack<int>();
stack.Push(1);
stack.Push(2);
stack.Push(3);

// 第一步：复制到数组（顺序：栈顶→栈底，即 [3,2,1]）
int[] array = stack.ToArray();

// 第二步：修改数组中的元素
if (array.Length > 1)
    array[1] = 20;   // 修改第二个元素

// 第三步：清空原栈
stack.Clear();

// 第四步：逆序重新压栈（保持原来的栈顺序）
for (int i = array.Length - 1; i >= 0; i--)
    stack.Push(array[i]);

// 结果：3 → 20 → 1


二、引用类型：直接改属性

class Person
{
    public string Name { get; set; }
    public int Age { get; set; }
}

Stack<Person> personStack = new Stack<Person>();
personStack.Push(new Person("Alice", 30));
personStack.Push(new Person("Bob", 25));

// 弹出栈顶，修改属性，再压回去
if (personStack.Count > 0)
{
    Person top = personStack.Pop();
    top.Age = 26;
    personStack.Push(top);
}
