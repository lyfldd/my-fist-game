Stack<T> —— 其他方法
=============================================

一、方法汇总表

  方法              作用                          时间复杂度
  ───────────────────────────────────────────────────────
  Contains(T)        是否包含某值                   O(n)
  Clear()            清空                          O(n)
  ToArray()          转为数组（栈顶→栈底）          O(n)
  CopyTo(T[], int)   复制到已有数组                 O(n)


二、Contains —— 查找

Stack<int> stack = new Stack<int>();
stack.Push(1);
stack.Push(2);
stack.Push(3);

bool has2 = stack.Contains(2);   // true
bool has9 = stack.Contains(9);   // false


三、Clear —— 清空

stack.Clear();   // Count = 0，容量不变


四、ToArray —— 转为数组

// 返回数组，顺序是从栈顶到栈底（与压栈顺序相反！）
int[] arr = stack.ToArray();
// arr = [3, 2, 1]


五、CopyTo —— 复制到已有数组

string[] arr = new string[stack.Count];
stack.CopyTo(arr, 0);   // 从 arr[0] 开始存，顺序是栈顶到栈底
