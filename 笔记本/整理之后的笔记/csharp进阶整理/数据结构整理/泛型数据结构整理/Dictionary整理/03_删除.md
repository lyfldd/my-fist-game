Dictionary<TKey, TValue> —— 删除元素
=============================================

一、方法汇总表

  方法              作用                    时间复杂度
  ─────────────────────────────────────────────────
  Remove(key)       删除指定键值对          O(1)
  Clear()           清空所有键值对          O(n)


二、Remove —— 删除指定键

dict.Remove("name");   // 删除 name 键值对

// 返回值表示是否删除成功
bool existed = dict.Remove("name");  // true（存在并删除）
bool existed2 = dict.Remove("不存在的键");  // false（本来就没有）


三、Clear —— 清空所有

dict.Clear();   // Count 变为 0，所有键值对被移除
