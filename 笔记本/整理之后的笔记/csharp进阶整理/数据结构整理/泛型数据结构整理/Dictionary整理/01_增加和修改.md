Dictionary<TKey, TValue> —— 增加和修改元素
=============================================

一、增加元素 —— Add

// Add(key, value)：添加键值对
dict.Add("score", 100);

// 注意：key 已存在会抛 ArgumentException
dict.Add("score", 200);  // ❌ 抛异常（key 已存在）


二、索引器添加或修改

// 索引器：key 不存在则添加，存在则覆盖
dict["name"] = "张三";   // 添加
dict["name"] = "李四";   // 覆盖，dict["name"] 变为"李四"


三、方法汇总表

  操作              方法                   说明
  ───────────────────────────────────────────────────────
  添加（安全）       Add(key, value)       key 已存在抛异常
  添加或覆盖         dict[key] = value     自动覆盖，无视 key 是否存在
  修改值             dict[key] = 新值       key 必须存在，否则变成添加
