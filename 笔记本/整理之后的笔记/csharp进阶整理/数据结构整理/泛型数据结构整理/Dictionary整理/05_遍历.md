Dictionary<TKey, TValue> —— 遍历
=============================================

一、三种遍历方式

// 方式 1：遍历所有键值对（最常用）
foreach (var kvp in dict)
{
    Console.WriteLine($"键：{kvp.Key}, 值：{kvp.Value}");
}

// 方式 2：遍历所有键
foreach (var key in dict.Keys)
{
    Console.WriteLine($"键：{key}, 值：{dict[key]}");
}

// 方式 3：遍历所有值
foreach (var value in dict.Values)
{
    Console.WriteLine(value);
}


二、元组解构遍历（C# 3.0+）

// 用元组语法糖，kvp 自动解构成 key 和 value
foreach (var (key, value) in dict)
{
    Console.WriteLine($"{key} = {value}");
}


三、遍历时的注意事项

⚠️ 不要在 foreach 中增删字典
// ❌ 错误：遍历中增删会抛 InvalidOperationException
foreach (var kvp in dict)
{
    if (kvp.Value < 0)
        dict.Remove(kvp.Key);
}

// ✅ 正确：先把要删的键收集起来再删
var keysToRemove = dict.Where(kvp => kvp.Value < 0).Select(kvp => kvp.Key).ToList();
foreach (var key in keysToRemove)
{
    dict.Remove(key);
}
