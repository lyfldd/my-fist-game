Dictionary<TKey, TValue> —— 查找
=============================================

一、方法汇总表

  方法                    作用                              时间复杂度
  ───────────────────────────────────────────────────────────────────
  ContainsKey(key)         是否包含指定键（推荐）              O(1)
  ContainsValue(value)     是否包含指定值（需遍历，慢）         O(n)
  Contains(KeyValuePair)   是否包含指定键值对                  O(1)


二、ContainsKey（推荐）

// 判断键是否存在，时间复杂度 O(1)
if (dict.ContainsKey("name"))
{
    // 存在才取，安全
    var name = dict["name"];
}


// TryGetValue 的替代方案
if (dict.TryGetValue("name", out var n))
{
    // 与 ContainsKey + 索引器效果相同，但只查一次哈希表，更高效
}


三、ContainsValue（注意性能）

// 判断值是否存在，需遍历所有元素，时间复杂度 O(n)
bool has100 = dict.ContainsValue(100);

// ⚠️ 注意：值可能重复，找到第一个就返回 true
// 值类型的 ContainsValue 每次都全遍历，数据量大时开销明显


四、Contains —— 键值对整体查找

// 判断这个具体的键值对是否存在
bool exists = dict.Contains(new KeyValuePair<string, int>("name", 100));

// 用途：比如删除一个确定的键值对
dict.Remove(new KeyValuePair<string, int>("name", 100));
