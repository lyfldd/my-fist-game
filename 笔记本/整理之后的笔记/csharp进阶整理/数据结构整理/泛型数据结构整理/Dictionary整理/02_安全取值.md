Dictionary<TKey, TValue> —— 安全取值
=============================================

一、两种取值方式对比

  方式              键不存在时行为        推荐程度
  ─────────────────────────────────────────────────
  索引器 dict[key]  抛异常               ⚠️ 慎用
  TryGetValue       返回 false，不抛异常  ✅ 推荐

⚠️ 注意：索引器取值时，key 不存在会抛 KeyNotFoundException！


二、TryGetValue（推荐写法）

// 安全取值，不会抛异常
if (dict.TryGetValue("name", out var name))
{
    Console.WriteLine(name);   // 存在时执行
}
else
{
    Console.WriteLine("键不存在");  // 不存在时执行
}

// out 变量可读出来直接用
dict.TryGetValue("score", out var score);


三、取值示例

Dictionary<string, int> dict = new Dictionary<string, int>
{
    { "hp", 100 },
    { "mp", 50 }
};

// ✅ 推荐：TryGetValue
if (dict.TryGetValue("hp", out var hp))
{
    Console.WriteLine($"HP: {hp}");
}

// ❌ 危险：索引器（key不存在会崩）
Console.WriteLine(dict["hp2"]);  // 抛 KeyNotFoundException
