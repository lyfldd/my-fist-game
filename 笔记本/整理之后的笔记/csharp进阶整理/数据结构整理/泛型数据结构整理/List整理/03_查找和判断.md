List<T> —— 查找和判断
=============================================

一、方法汇总表

  方法                               作用                              返回值
  ───────────────────────────────────────────────────────────────────────────
  Contains(T item)                   是否包含指定值（值比较）            bool
  Exists(Predicate<T>)                是否存在满足条件的元素             bool
  TrueForAll(Predicate<T>)            所有元素是否都满足条件             bool
  Find(Predicate<T>)                  找第一个满足条件的元素             T / default(T)
  FindLast(Predicate<T>)              找最后一个满足条件的元素           T / default(T)
  FindAll(Predicate<T>)              找所有满足条件的元素               List<T>
  FindIndex(Predicate<T>)             找第一个满足条件的元素索引         int（找不到返回 -1）
  FindLastIndex(Predicate<T>)         找最后一个满足条件的元素索引        int（找不到返回 -1）


二、Contains —— 是否包含指定值（值比较）

List<int> list = new List<int> { 1, 2, 3, 4, 5 };
bool has3 = list.Contains(3);   // true
bool has10 = list.Contains(10); // false

// 注意：值类型用值比较，引用类型用引用比较（是否同一对象）


三、Exists —— 是否存在满足条件的元素

List<int> list = new List<int> { 1, 2, 3, 4, 5 };
bool hasEven = list.Exists(x => x % 2 == 0);  // true（有偶数）
bool hasNegative = list.Exists(x => x < 0);   // false


四、TrueForAll —— 所有元素是否都满足条件

List<int> list = new List<int> { 2, 4, 6, 8 };
bool allEven = list.TrueForAll(x => x % 2 == 0);  // true（全是偶数）

List<int> list2 = new List<int> { 2, 3, 6, 8 };
bool allEven2 = list2.TrueForAll(x => x % 2 == 0);  // false（有奇数）


五、Find —— 找第一个满足条件的元素

List<int> list = new List<int> { 1, 2, 3, 4, 5, 3 };

int first = list.Find(x => x > 2);    // 3（第一个大于 2 的）
int firstNeg = list.Find(x => x < 0); // 0（找不到返回 default(int)）

// FindLast —— 找最后一个满足条件的元素
int last = list.FindLast(x => x > 2);  // 3（最后一个大于 2 的）
// 如果没有找到，返回 default(T)（int 为 0，引用类型为 null）


六、FindAll —— 找所有满足条件的元素（最常用）

List<int> list = new List<int> { 1, 2, 3, 4, 5, 3 };

// 找所有大于 2 的元素，返回新 List<T>
List<int> result = list.FindAll(x => x > 2);
// result = [3, 4, 5, 3]


七、FindIndex / FindLastIndex —— 找元素索引

List<int> list = new List<int> { 1, 2, 3, 4, 5, 3 };

int idx1 = list.FindIndex(x => x == 3);       // 2（第一个 3 的索引）
int idx2 = list.FindLastIndex(x => x == 3);   // 5（最后一个 3 的索引）
int idx3 = list.FindIndex(x => x > 10);       // -1（找不到）

// 带范围版本：从索引 3 开始找第一个大于 2 的
int idx4 = list.FindIndex(3, x => x > 2);     // 4

// 带范围版本：从索引 0 开始往后找 3 个元素内，第一个大于 2 的
int idx5 = list.FindIndex(0, 3, x => x > 2); // 2


八、综合对比

  场景                     推荐方法           返回值
  ─────────────────────────────────────────────────────
  只判断是否存在              Exists            bool
  判断是否所有都满足          TrueForAll        bool
  找第一个（只要一个）        Find              T
  找最后一个                 FindLast          T
  找所有匹配的               FindAll           List<T>
  找索引（只要索引不要值）    FindIndex         int（-1 表示没找到）
  精确值查找                 Contains           bool
