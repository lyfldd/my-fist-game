List<T> —— 复制和转换
=============================================

一、方法汇总表

  方法                                       作用                         时间复杂度
  ─────────────────────────────────────────────────────────────────────────────────
  ToArray()                                   转为 T[] 数组                O(n)
  GetRange(int index, int count)              获取子集（返回新 List）      O(n)
  CopyTo(T[] array)                          复制到数组                   O(n)
  CopyTo(int index, T[] array, int arrayIndex, int count)  部分复制       O(n)


二、ToArray —— 转为 T[] 数组

List<int> list = new List<int> { 1, 2, 3, 4, 5 };

int[] arr = list.ToArray();
// arr = [1, 2, 3, 4, 5]


三、GetRange —— 获取子集

List<int> list = new List<int> { 1, 2, 3, 4, 5, 6, 7 };

// 从索引 2 开始，取 3 个元素（返回新 List，不影响原列表）
List<int> sub = list.GetRange(2, 3);
// sub = [3, 4, 5]
// 原列表不变：[1, 2, 3, 4, 5, 6, 7]


四、CopyTo —— 复制到数组

List<string> list = new List<string> { "A", "B", "C" };
string[] arr = new string[5];

// 全部复制到 arr，从 arr[0] 开始
list.CopyTo(arr);
// arr = [A, B, C, null, null]


五、CopyTo —— 部分复制

List<string> list = new List<string> { "A", "B", "C", "D" };
string[] arr = new string[10];

// 从 list[1] 开始，复制 3 个元素，到 arr[2] 位置
list.CopyTo(1, arr, 2, 3);
// arr = [null, null, B, C, D, null, null, null, null, null]


六、应用场景

场景 1：备份列表
List<int> backup = original.ToArray().ToList();
// 或直接用 GetRange(0, original.Count) 返回新 List

场景 2：分页
List<T> page1 = allItems.GetRange(0, pageSize);
List<T> page2 = allItems.GetRange(pageSize, pageSize);

场景 3：传递给需要数组的旧 API
SomeLegacyMethod(list.ToArray());
