List<T> —— 排序和反转
=============================================

一、方法汇总表

  方法                                      作用                  时间复杂度
  ───────────────────────────────────────────────────────────────────────────
  Sort()                                    默认升序排序            O(n log n)
  Sort(Comparison<T>)                       自定义比较（降序等）     O(n log n)
  Sort(int index, int count, IComparer<T>)  部分排序                O(n log n)
  Reverse()                                 反转顺序                O(n)
  Reverse(int index, int count)             部分反转                O(n)


二、Sort —— 升序排序（默认）

List<int> list = new List<int> { 3, 1, 4, 1, 5, 9, 2, 6 };
list.Sort();
// [1, 1, 2, 3, 4, 5, 6, 9]


三、Sort —— 自定义排序

// 降序排序（从小到大 → 从大到小）
list.Sort((a, b) => b.CompareTo(a));
// [9, 6, 5, 4, 3, 2, 1, 1]

// 按绝对值排序
list.Sort((a, b) => Math.Abs(a).CompareTo(Math.Abs(b)));

// 按字符串长度排序
List<string> words = new List<string> { "apple", "hi", "banana" };
words.Sort((a, b) => a.Length.CompareTo(b.Length));
// [hi, apple, banana]


四、Sort —— 部分排序

List<int> list = new List<int> { 5, 1, 4, 3, 9, 2, 8 };
// 从索引 1 开始，排 4 个元素
list.Sort(1, 4, null);
// [5, 1, 2, 3, 4, 9, 8]
// 索引 0 的 5 和索引 5~6 的 9、8 不动


五、Reverse —— 反转顺序

List<int> list = new List<int> { 1, 2, 3, 4, 5 };
list.Reverse();
// [5, 4, 3, 2, 1]


六、Reverse —— 部分反转

List<int> list = new List<int> { 1, 2, 3, 4, 5, 6 };
// 从索引 2 开始，反转 3 个元素
list.Reverse(2, 3);
// [1, 2, 6, 5, 4, 3]


七、游戏开发中的排序应用

// 按攻击力排序（角色列表）
players.Sort((a, b) => b.Attack.CompareTo(a.Attack));
// 攻击力高的排前面（降序）


// 物品按价值排序
items.Sort((a, b) => b.Value.CompareTo(a.Value));


// 按名称升序（字符串比较）
items.Sort((a, b) => string.Compare(a.Name, b.Name));
