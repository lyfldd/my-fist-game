List<T> —— 访问和修改
=============================================

一、索引器访问和修改

List<int> list = new List<int> { 10, 20, 30, 40 };

// 读取
int first = list[0];      // 10
int last = list[list.Count - 1];  // 40

// 修改
list[0] = 100;            // [100, 20, 30, 40]
list[list.Count - 1] = 400; // [100, 20, 30, 400]

// 注意：索引超出范围会抛 ArgumentOutOfRangeException


二、获取长度

List<int> list = new List<int> { 10, 20, 30 };

int count = list.Count;    // 3

⚠️ 注意：是 Count，不是 Length！
List<T> 是集合，用 Count。
数组是 Length。
混用是新手常见错误。


三、设置长度（不存在！）

// ❌ 错误：List<T> 没有直接设置数量的方法
list.Length = 10;  // 不存在！

// ✅ 如果要扩展，用 Add 填默认值
while (list.Count < 10)
{
    list.Add(0);
}


四、直接修改元素值（常见用法）

List<Player> players = new List<Player>
{
    new Player("剑士", 100),
    new Player("法师", 80)
};

// 通过索引修改
players[0].Hp = 50;   // ✅ 引用类型，直接改属性
players[0] = new Player("剑士", 50);  // ✅ 整体替换

说明：
- List<T> 存的是引用类型对象时，直接改对象属性，所有引用都会同步
- List<T> 存的是值类型时，只能整体替换，不能通过引用改
