LinkedList<T> —— 遍历
=============================================

一、foreach 遍历值（最简洁）

foreach (int item in list)
{
    Console.WriteLine(item);
}


二、用节点手动遍历（可以拿到前后节点信息）

// 从前往后
var node = list.First;
while (node != null)
{
    Console.WriteLine(node.Value);
    node = node.Next;
}

// 从后往前（双向链表特有优势）
var node2 = list.Last;
while (node2 != null)
{
    Console.WriteLine(node2.Value);
    node2 = node2.Previous;
}


三、使用场景建议

  场景                   推荐方式
  ─────────────────────────────────
  只读取值               foreach（简洁）
  需要前后节点信息        节点遍历（Find 返回的节点可访问 Previous/Next）
  需要增删操作时          先拿到节点再操作（节点删改才是 O(1)）
