LinkedList<T> —— 查找元素
=============================================

一、方法汇总表

  方法                    作用                              返回值                       时间复杂度
  ─────────────────────────────────────────────────────────────────────────────────────────
  Contains(T value)        判断是否包含某值                  bool                         O(n)
  Find(T value)            从前向后找，返回第一个匹配节点     LinkedListNode<T> 或 null     O(n)
  FindLast(T value)        从后向前找，返回最后一个匹配节点   LinkedListNode<T> 或 null     O(n)


二、Contains —— 判断是否包含

LinkedList<int> list = new LinkedList<int>(new int[] { 1, 2, 3, 4, 2 });

bool has2 = list.Contains(2);   // true
bool has9 = list.Contains(9);   // false


三、Find —— 找第一个匹配节点

// 返回的是节点对象，不只是值
var firstNode = list.Find(2);   // 找到第一个 2 的节点
if (firstNode != null)
{
    Console.WriteLine(firstNode.Value);  // 2
    Console.WriteLine(firstNode.Previous?.Value);  // 1（上一个节点）
    Console.WriteLine(firstNode.Next?.Value);      // 3（下一个节点）
}


四、FindLast —— 找最后一个匹配节点

// 从后向前遍历，返回最后一个匹配的节点
var lastNode = list.FindLast(2);  // 最后一个 2 的节点


五、找不到时的返回值

⚠️ 找不到时返回 null，不是抛异常！
访问返回值前必须先判空。

var node = list.Find(99);
if (node != null)
{
    Console.WriteLine(node.Value);
}
