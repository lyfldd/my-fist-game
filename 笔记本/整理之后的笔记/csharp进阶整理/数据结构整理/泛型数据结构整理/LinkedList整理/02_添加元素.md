LinkedList<T> —— 添加元素
=============================================

一、方法汇总表

  方法                                      作用                    时间复杂度
  ───────────────────────────────────────────────────────────────────────────
  AddFirst(T value)                         头部加值（自动建节点）     O(1)
  AddFirst(LinkedListNode<T> node)           把已有节点移到头部         O(1)
  AddLast(T value)                          尾部加值（自动建节点）     O(1)
  AddLast(LinkedListNode<T> node)            把已有节点移到尾部         O(1)
  AddBefore(node, T value)                  在 node 前面插入值        O(1)
  AddBefore(node, LinkedListNode<T>)         把已有节点插到 node 前面   O(1)
  AddAfter(node, T value)                   在 node 后面插入值        O(1)
  AddAfter(node, LinkedListNode<T>)          把已有节点插到 node 后面   O(1)


二、AddFirst —— 添加到头部

LinkedList<string> list = new LinkedList<string>();
list.AddFirst("张三");
list.AddFirst("李四");
// 结果：李四 → 张三


三、AddLast —— 添加到尾部

list.AddLast("小明");
list.AddLast("小红");
// 结果：头 → 尾（追加到现有节点后面）


四、AddBefore / AddAfter —— 在指定节点前后插入

// 在"小红"前面插入"小刚"
var node = list.Find("小红");
list.AddBefore(node, "小刚");
// 结果：小明 → 小刚 → 小红

// 在"小明"后面插入"小亮"
var node2 = list.Find("小明");
list.AddAfter(node2, "小亮");
// 结果：小明 → 小亮 → 小刚 → 小红


五、传入已有节点的规则

⚠️ 已有节点不能属于另一个 LinkedList，否则抛 InvalidOperationException

// ✅ 正确：新建节点传入
var newNode = new LinkedListNode<string>("王五");
list.AddFirst(newNode);

// ❌ 错误：节点不能跨链表使用
var otherNode = anotherList.Find("xxx");
list.AddFirst(otherNode);  // 抛异常
