LinkedList<T> —— 核心属性
=============================================

一、属性一览

  属性          类型                      说明
  ─────────────────────────────────────────────────────────
  Count        int                       元素数量
  First        LinkedListNode<T>          头节点（链表为空时为 null）
  Last         LinkedListNode<T>          尾节点（链表为空时为 null）


二、用法示例

LinkedList<int> list = new LinkedList<int>(new int[] { 1, 2, 3 });

int count = list.Count;          // 3

var firstNode = list.First;      // 头节点（值为 1）
var lastNode = list.Last;        // 尾节点（值为 3）

int firstValue = list.First.Value;    // 1
int lastValue = list.Last.Value;     // 3


三、节点对象的属性

LinkedListNode<T> 对象有以下属性：

  节点属性       说明
  ──────────────────────────
  Value          存储的值
  Previous       前一个节点（头节点为 null）
  Next           后一个节点（尾节点为 null）
  List           该节点所属的链表（可用于判断节点是否有效）
