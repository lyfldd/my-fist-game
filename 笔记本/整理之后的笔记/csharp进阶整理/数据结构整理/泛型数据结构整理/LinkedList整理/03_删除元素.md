LinkedList<T> —— 删除元素
=============================================

一、方法汇总表

  方法                                作用                    时间复杂度  注意事项
  ─────────────────────────────────────────────────────────────────────────────────
  Remove(T value)                     删除第一个匹配项         O(n)       找不到返回 false，不抛异常
  Remove(LinkedListNode<T> node)      删除指定节点             O(1)       节点为 null 或不属于本链表抛异常
  RemoveFirst()                       删除头节点               O(1)       链表为空时抛异常
  RemoveLast()                        删除尾节点               O(1)       链表为空时抛异常
  Clear()                             清空所有节点             O(n)


二、Remove —— 按值删除

LinkedList<int> list = new LinkedList<int>(new int[] { 10, 20, 10, 30 });
list.Remove(10);   // 删除第一个 10，结果：20 → 10 → 30


三、Remove —— 按节点删除（最推荐）

// 按节点删除是 O(1)，比按值删快
var node = list.Find(20);
list.Remove(node);  // 直接删除节点 20


四、安全删除头/尾

if (list.Count > 0) list.RemoveFirst();
if (list.Count > 0) list.RemoveLast();


五、删除所有相同值（常用技巧）

// LinkedList 没有 RemoveAll，需要手动循环
var node = list.Find(10);
while (node != null)
{
    var next = node.Next;  // 先保存下一个节点，防止删除后丢失引用
    list.Remove(node);
    node = next;
}


六、⚠️ 重要限制

- 不支持 RemoveAt(int index)（无索引器）
- 按位置删除必须先遍历到节点，再 Remove(node)
- 遍历中删除时，注意先保存 next 再删
