# 顺序存储和链式存储

## 一、基本概念

### 1.1 数据结构
程序中组织和存储数据的方式，定义数据之间的**逻辑关系**和**物理关系**。

### 1.2 线性表
n（n≥0）个相同特性的数据元素按固定顺序排列的**有限序列**。

- 顺序存储：数组、`List<T>`
- 链式存储：单向链表、双向链表、循环链表

---

## 二、顺序存储

用一段**地址连续**的存储单元依次存储线性表的元素，像排队站好，地址相邻。

```csharp
// C# 中的顺序存储典型实现
int[] arr = new int[5];           // 固定长度数组
List<int> list = new List<int>(); // 可动态扩容（推荐）
```

| 特点 | 说明 |
|---|---|
| 内存连续 | 元素在内存中相邻排列 |
| 随机访问 | 通过下标直接访问，时间复杂度 O(1) |
| 长度可变 | `List<T>` 可动态扩容 |
| 存储在堆 | `new` 出来的数组存在堆上 |

### 顺序存储内存示意
```
栈帧（方法区）                堆
┌──────────┐          ┌─────────────────┐
│ arr 引用  │────────→│ [0] [1] [2] [3] │  ← 连续地址
└──────────┘          └─────────────────┘
                       0x1000 ~ 0x101F
```

---

## 三、链式存储

用一组**任意（不要求连续）**的存储单元存储线性表元素，通过节点中的**指针**维护元素之间的关系。

### 3.1 单向链表（Singly Linked List）

每个节点包含：**数据域 + 指向下一个节点的指针（Next）**。

```csharp
class LinkedNode<T>
{
    public T Value;
    public LinkedNode<T> Next; // 指向下一个节点

    public LinkedNode(T value)
    {
        Value = value;
    }
}

// 创建节点
LinkedNode<int> n1 = new LinkedNode<int>(1);
LinkedNode<int> n2 = new LinkedNode<int>(2);
LinkedNode<int> n3 = new LinkedNode<int>(3);

// 连起来：1 → 2 → 3
n1.Next = n2;
n2.Next = n3;
n3.Next = null; // 最后一个节点指向 null

// 遍历
LinkedNode<int> current = n1;
while (current != null)
{
    Console.WriteLine(current.Value);
    current = current.Next;
}
```

> 和数组最大区别：数组连续内存下标访问快，插入删除慢；链表不连续，不能直接下标访问，但插入删除很快。

### 单向链表内存示意
```
内存（非连续）:
┌────┬────┐    ┌────┬────┐    ┌────┬────┐
│ 1  │next│──→ │ 2  │next│──→ │ 3  │null│
└────┴────┘    └────┴────┘    └────┴────┘
```

### 单向链表 — 头插（O(1)）
```csharp
// 在头部插入节点，速度快
LinkedNode<int> newNode = new LinkedNode<int>(0);
newNode.Next = head;
head = newNode; // 更新头节点
```

### 单向链表 — 删除节点
```csharp
// 删除指定节点（如删除第二个节点）
n1.Next = n3; // 让节点1直接指向节点3，节点2被跳过
```

### 单向链表 — 完整实现
```csharp
class LinkedNode<T>
{
    public T Value;
    public LinkedNode<T> Next;

    public LinkedNode(T value)
    {
        Value = value;
    }
}

class LinkedList<T>
{
    private LinkedNode<T> head;

    public void Add(T value)
    {
        LinkedNode<T> newNode = new LinkedNode<T>(value);
        if (head == null)
        {
            head = newNode;
        }
        else
        {
            LinkedNode<T> current = head;
            while (current.Next != null)
            {
                current = current.Next;
            }
            current.Next = newNode;
        }
    }

    // 在指定位置插入
    public void Insert(int index, T value)
    {
        LinkedNode<T> newNode = new LinkedNode<T>(value);
        if (index == 0)
        {
            newNode.Next = head;
            head = newNode;
            return;
        }
        LinkedNode<T> current = head;
        for (int i = 0; i < index - 1; i++)
        {
            if (current == null)
                throw new IndexOutOfRangeException("索引超出范围");
            current = current.Next;
        }
        newNode.Next = current.Next;
        current.Next = newNode;
    }

    // 删除指定位置的节点
    public void RemoveAt(int index)
    {
        if (index == 0)
        {
            head = head.Next;
            return;
        }
        LinkedNode<T> current = head;
        for (int i = 0; i < index - 1; i++)
        {
            if (current == null)
                throw new IndexOutOfRangeException("索引超出范围");
            current = current.Next;
        }
        current.Next = current.Next.Next;
    }
}
```

### 3.2 双向链表（Doubly Linked List）【补充】

每个节点包含：**数据域 + 指向前一个节点的指针（Previous）+ 指向下一个节点的指针（Next）**，可以双向遍历。

> 相当于火车两头都有钩子，既能往前走也能往回走。

```csharp
class DLinkedNode<T>
{
    public T Value;
    public DLinkedNode<T> Previous; // 指向前一个节点
    public DLinkedNode<T> Next;     // 指向下一个节点

    public DLinkedNode(T value)
    {
        Value = value;
    }
}

class DLinkedList<T>
{
    private DLinkedNode<T> head;
    private DLinkedNode<T> tail;

    public void AddLast(T value)
    {
        DLinkedNode<T> newNode = new DLinkedNode<T>(value);
        if (tail == null)
        {
            head = tail = newNode;
        }
        else
        {
            tail.Next = newNode;
            newNode.Previous = tail;
            tail = newNode;
        }
    }

    public void AddFirst(T value)
    {
        DLinkedNode<T> newNode = new DLinkedNode<T>(value);
        if (head == null)
        {
            head = tail = newNode;
        }
        else
        {
            newNode.Next = head;
            head.Previous = newNode;
            head = newNode;
        }
    }

    public void Remove(DLinkedNode<T> node)
    {
        if (node.Previous != null)
            node.Previous.Next = node.Next;
        else
            head = node.Next;

        if (node.Next != null)
            node.Next.Previous = node.Previous;
        else
            tail = node.Previous;
    }

    // 正向遍历
    public void TraverseForward()
    {
        DLinkedNode<T> current = head;
        while (current != null)
        {
            Console.WriteLine(current.Value);
            current = current.Next;
        }
    }

    // 反向遍历
    public void TraverseBackward()
    {
        DLinkedNode<T> current = tail;
        while (current != null)
        {
            Console.WriteLine(current.Value);
            current = current.Previous;
        }
    }
}
```

### 双向链表内存示意
```
←──prev        prev        prev────────→
┌────┬────┬────┐    ┌────┬────┬────┐    ┌────┬────┬────┐
│null│ 1  │next│←──→│prev│ 2  │next│←──→│prev│ 3  │null│
└────┴────┴────┘    └────┴────┴────┘    └────┴────┴────┘
     头节点                                      尾节点
```

### 双向链表 vs 单向链表
| 对比 | 单向链表 | 双向链表 |
|---|---|---|
| 节点结构 | Value + Next | Value + Previous + Next |
| 遍历方向 | 只能向前 | 可双向遍历 |
| 删除节点 | 需记录前一个节点 | O(1) 直接删除 |
| 内存开销 | 较小（1个指针） | 较大（2个指针） |
| C# 对应 | 自定义实现 | `LinkedList<T>` |

> C# 中 `LinkedList<T>` 就是双向链表的官方实现，提供 `AddFirst`、`AddLast`、`AddAfter`、`Remove` 等方法。

---

### 3.3 循环链表（Circular Linked List）

循环链表是链表的变体，**最后一个节点的 Next 指向头节点（或前一个节点）**，首尾相连形成环。

| 类型 | 特点 |
|---|---|
| 单向循环链表 | 尾节点 Next → 头节点，无法反向遍历 |
| 双向循环链表 | 头节点 Previous → 尾节点，尾节点 Next → 头节点，可双向遍历 |

> 应用场景：约瑟夫环问题、环形缓冲区、操作系统中的任务调度等。

---

## 四、顺序存储 vs 链式存储对比

| 对比维度 | 顺序存储（`List<T>`） | 链式存储（`LinkedList<T>`） |
|---|---|---|
| 存储方式 | 物理地址连续 | 物理地址不连续，靠指针连接 |
| 查询 | **O(1)** 下标直接访问 | O(n) 需从头遍历 |
| 头部插入/删除 | O(n) 需移动元素 | **O(1)** 直接操作 |
| 尾部插入/删除 | **O(1)** | **O(1)** |
| 已知节点插入/删除 | O(n)（先找位置） | **O(1)** |
| 内存开销 | 低（无指针开销） | 高（每个节点多1~2个指针） |
| 内存分布 | 堆（连续） | 堆（节点）+ 栈（引用） |
| 长度 | 可变（`List<T>`） | 可动态增长 |
| 典型场景 | 读多写少 | 写多读少 |

> **一句话总结**：查多用 `List<T>`，改多用 `LinkedList<T>`。

---

## 五、C# 中的使用建议

```csharp
// 查多写少 → 用 List<T>
List<int> list = new List<int>();
list.Add(10);
int item = list[3]; // O(1) 随机访问

// 频繁在中间增删 → 用 LinkedList<T>
LinkedList<int> linked = new LinkedList<int>();
linked.AddFirst(10);
linked.AddLast(20);
linked.RemoveFirst(); // O(1)
```
