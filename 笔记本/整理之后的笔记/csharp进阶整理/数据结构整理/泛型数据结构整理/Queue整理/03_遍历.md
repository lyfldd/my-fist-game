Queue<T> —— 遍历
=============================================

一、遍历方式

// 方式 1：foreach（从队头到队尾，不删除元素）
foreach (string item in queue)
{
    Console.WriteLine(item);
}

// 方式 2：循环出队（每次取出并删除，直到队列为空）
while (queue.Count > 0)
{
    string item = queue.Dequeue();
    Console.WriteLine("处理：" + item);
}

// 方式 3：TryDequeue 安全循环（推荐）
while (queue.TryDequeue(out string item))
{
    Console.WriteLine("处理：" + item);
}


二、说明

foreach 内部调用的是 GetEnumerator()，遍历过程不改变队列内容。
循环出队会把队列清空，处理完就没了。
