Queue<T> —— 其他方法
=============================================

一、方法汇总表

  方法                作用                      时间复杂度
  ─────────────────────────────────────────────────────
  Contains(T)          是否包含某值               O(n)
  Clear()              清空队列                   O(n)
  ToArray()            转为数组（队头→队尾）       O(n)
  CopyTo(T[], int)     复制到已有数组              O(n)
  TrimExcess()         释放多余内存               O(n)


二、Contains —— 查找

bool has = queue.Contains("B");   // true


三、Clear —— 清空

queue.Clear();   // Count 变成 0，但容量不变


四、ToArray —— 转为数组

// 按队列顺序（队头→队尾）转为数组
int[] arr = queue.ToArray();


五、CopyTo —— 复制到已有数组

int[] arr = new int[queue.Count];
queue.CopyTo(arr, 0);   // 从 arr[0] 开始，按队头到队尾顺序复制


六、TrimExcess —— 释放多余空间

// 将内部数组缩减到实际元素数量，释放多余内存
// 适合队列长期稳定不再增长时调用
queue.TrimExcess();
