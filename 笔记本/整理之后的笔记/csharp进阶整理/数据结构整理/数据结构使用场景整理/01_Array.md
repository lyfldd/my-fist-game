# Array（数组）使用场景

---

## 一、核心本质

固定长度、连续内存、下标 O(1) 随机访问，值类型/引用类型通用。

---

## 二、核心特性

- 长度一旦创建就不可修改，扩容需要手动创建新数组拷贝
- 内存连续，缓存命中率极高，遍历速度是所有容器里最快的
- 支持排序、二分查找等算法，是所有集合的底层基础

---

## 三、使用场景

### 场景 1：固定数量的已知数据

```csharp
// 固定 4 个属性、6 个技能格子等
int[] attributes = new int[4];      // 力量/敏捷/智力/体力
string[] skillSlots = new string[6]; // 技能栏固定 6 个
```

### 场景 2：高性能遍历/算法

```csharp
// 排序、二分查找、矩阵运算
Array.Sort(arr);
int index = Array.BinarySearch(arr, target);
```

### 场景 3：与其他集合互转的桥梁

```csharp
// List/Stack/Queue 底层都基于数组，数组是最基础的存储单元
int[] arr = list.ToArray();
List<int> list2 = new List<int>(arr);
```

---

## 四、不适用场景

- 数据量不确定、需要频繁增删 → 用 List<T> 代替
- 频繁在中间插入/删除元素（O(n) 效率极低）

---

## 五、时间复杂度速查

| 操作 | 时间复杂度 |
|---|---|
| 按下标访问 | O(1) |
| 遍历全部 | O(n) |
| 查找元素 | O(n)（无序）/ O(log n)（有序二分） |
| 插入/删除（中间） | O(n)（需移动元素） |
| 扩容（手动创建新数组） | O(n) |
