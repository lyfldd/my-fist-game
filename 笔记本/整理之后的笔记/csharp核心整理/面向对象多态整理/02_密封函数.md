密封函数
=============================================

一、概念
用 sealed 修饰的 override 方法叫密封函数。
密封函数不能再被子类的子类重写。

二、语法
sealed override 返回类型 方法名() { }

【注意】sealed 必须和 override 一起使用，不能单独用。

三、作用
当某个方法在子类中已经足够完善，不需要再让子类的子类修改时，
用 sealed 封印它，防止被进一步重写。

四、示例
class GameObject
{
    public virtual void Attack() { Console.WriteLine("基础攻击"); }
}

class Player : GameObject
{
    public sealed override void Attack()
    {
        Console.WriteLine("玩家攻击");
    }
}

class MainPlayer : Player
{
    // 编译错误！Attack 方法已经被 sealed，不能再重写
    // public override void Attack() { }
}

五、sealed 封印的不只是方法，还有类
sealed class FinalBoss
{
    // ...
}

// FinalBoss 的任何方法都不能被子类重写
