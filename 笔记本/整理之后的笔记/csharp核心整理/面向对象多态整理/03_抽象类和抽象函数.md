抽象类和抽象函数
=============================================

一、抽象类（abstract class）
用 abstract 修饰的类叫抽象类。
特点：不能实例化（不能 new），只能被继承。

二、抽象方法（abstract method）
用 abstract 修饰的没有方法体的方法。
特点：
1. 必须在抽象类中声明
2. 没有方法体（只有声明）
3. 强制继承它的子类必须实现该方法
4. 默认隐含 virtual（不需要再写 virtual）

三、示例
abstract class Character
{
    public string Name { get; set; }
    public int Health { get; set; }

    // 抽象方法（强制子类实现）
    public abstract void PerformAction();

    // 普通方法（子类直接继承）
    public void TakeDamage(int damage)
    {
        Health -= damage;
        if (Health <= 0) Console.WriteLine($"{Name} 被击败了");
    }
}

class Warrior : Character
{
    public int Strength { get; set; }

    public override void PerformAction()
    {
        Console.WriteLine($"{Name} 使用力量攻击，威力 {Strength}");
    }
}

class Mage : Character
{
    public int MagicPower { get; set; }

    public override void PerformAction()
    {
        Console.WriteLine($"{Name} 施展魔法，威力 {MagicPower}");
    }
}

// 多态体现
Character[] characters = {
    new Warrior { Name = "亚瑟", Strength = 100 },
    new Mage { Name = "甘道夫", MagicPower = 200 }
};

foreach (Character c in characters)
{
    c.PerformAction();  // 不同子类输出不同内容
}

四、规则
1. 抽象类可以有抽象方法，也可以有普通方法
2. 抽象方法不能有方法体
3. 抽象类可以有构造函数（供子类 base() 调用）
4. 抽象方法访问修饰符不能是 private
5. 子类实现抽象方法时，访问修饰符必须与抽象方法一致
