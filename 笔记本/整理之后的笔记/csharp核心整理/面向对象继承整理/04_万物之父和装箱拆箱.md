万物之父 object 和装箱拆箱
=============================================

一、object（万物之父）
object 是 C# 中所有类型的基类（System.Object），是引用类型。
可以用 object 变量存储任何类型的值。

object o1 = 10;       // 存值类型
object o2 = "hello";  // 存字符串
object o3 = new int[3]; // 存数组（引用类型）

object o = new Player(); // 用 object 存储子类对象（里氏替换）

二、object 的类型转换
值类型 → object（隐式装箱）：
    int i = 10;
    object o = i;

object → 值类型（必须强转）：
    int j = (int)o;

引用类型 → object（隐式）：
    Player p = new Player();
    object o = p;

object → 引用类型（用 as 或强转）：
    Player p2 = o as Player;    // 安全，失败返回 null
    Player p3 = (Player)o;       // 强转，失败抛异常

【纠正】原文"引用类型就用 as"不准确：
  - as 用于引用类型转引用类型
  - 值类型转值类型用强转
  - 引用类型转值类型也用强转（先拆箱）
  - string 是特殊引用类型，可以用 as 也可以用强转

三、装箱和拆箱
装箱（Boxing）：将值类型转换为 object（值类型→堆）
  int i = 10;
  object o = i;  // 装箱，值从栈复制到堆

拆箱（Unboxing）：将 object 转换为值类型（堆→栈）
  int j = (int)o;  // 拆箱，从堆复制到栈

四、装箱的性能问题
装箱会产生新对象（堆内存分配），增加性能消耗。
频繁大量装箱会触发 GC，影响性能。

五、应用：params object[] 实现任意类型参数
static void PrintAll(params object[] args)
{
    foreach (object item in args)
    {
        Console.Write(item + " ");
    }
}

PrintAll(1, "hello", 3.14, true);
// 输出：1 hello 3.14 True
