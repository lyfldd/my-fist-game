继承的基本规则
=============================================

一、概念
当一个类（A）继承另一个类（B）时，A 类会拥有 B 类的所有成员（字段、方法等）。
被继承的类叫"父类"或"基类"，继承的类叫"子类"或"派生类"。

二、基本语法
class 父类名 { }
class 子类名 : 父类名 { }

示例：
class Teacher
{
    public string name;
    public int number;

    public void SpeakName()
    {
        Console.WriteLine(name);
    }
}

class TeachingTeacher : Teacher  // TeachingTeacher 继承 Teacher
{
    public string subject;  // 子类独有的字段

    public void SpeakSubject()
    {
        Console.WriteLine(subject);
    }
}

// 使用
TeachingTeacher t = new TeachingTeacher();
t.name = "张三";
t.number = 1;
t.SpeakName();     // 输出：张三（继承自父类）
t.SpeakSubject();  // 输出：数学（子类自己的方法）

三、两大特性
1. 单根性：子类只能有一个父类（C# 不支持多继承，类只能继承一个类）
2. 传递性：子类可以间接继承父类的父类

四、访问修饰符对继承的影响
  private：子类和外部都不能访问
  protected：子类可以访问，外部不行
  public：任何地方都可以访问
  internal：同一程序集内可访问

五、同名成员处理（new 隐藏）
当子类和父类有同名成员时，子类的成员会隐藏父类的成员。
必须使用 new 关键字明确声明，否则编译器会报警告。

class BaseClass
{
    public int value = 10;
    public void Show() { Console.WriteLine("Base"); }
}

class DerivedClass : BaseClass
{
    public new int value = 20;       // 用 new 隐藏父类成员
    public new void Show() { Console.WriteLine("Derived"); }
}

DerivedClass d = new DerivedClass();
Console.WriteLine(d.value);  // 20（子类值）
Console.WriteLine(d.Show()); // Derived

【注意】隐藏不是多态！这是静态绑定（编译时决定），不是运行时多态。
