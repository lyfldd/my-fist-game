内部类和分部类
=============================================

一、内部类（Nested Class）
在一个类内部声明另一个类。

作用：将相关的类组织在一起，使代码结构更清晰。
内部类可以访问外部类的所有成员（包括 private）。

class Person
{
    public int age;
    public string name;
    public Body body;

    public class Body  // 内部类
    {
        public Arm leftArm;
        public Arm rightArm;

        public class Arm  // 嵌套的内部类
        {
            public int length;
        }
    }
}

// 使用：外部类.内部类
Person p = new Person();
Person.Body b = new Person.Body();

【访问规则】
1. 内部类默认访问修饰符是 private
2. 想要在外部使用内部类，内部类必须声明为 public
3. 内部类可以访问外部类的所有成员（包括 private）
4. 外部类访问内部类成员需要通过内部类实例

二、分部类（Partial Class）
将一个类拆分成多个文件来写，关键字是 partial。

作用：分离关注点，增加程序的可扩展性。
常见用途：WinForms 设计器自动生成的代码（partial class Form1）

partial class Student
{
    public string name;
    public bool sex;
}

partial class Student
{
    public int age;
}

partial class Student
{
    private int number;
}

// 三个文件加起来才是一个完整的 Student 类

【规则】
1. 所有 partial 类的访问修饰符必须一致
2. partial 类不能有相同的成员（字段、方法等不能重复）
3. partial 类写在不同文件，最终编译时合并

三、分部方法（Partial Method）
将方法的声明和实现分离，只能在 partial 类中声明。

【规则】
1. 不能加访问修饰符，默认是 private
2. 返回值只能是 void
3. 可以有参数，但不能用 out 关键字（ref 可以）
4. 如果只有声明没有实现，编译器会忽略该方法

partial class Student
{
    partial void OnNameChanged(string oldName, string newName); // 声明
}

partial class Student
{
    partial void OnNameChanged(string oldName, string newName)
    {
        Console.WriteLine($"姓名从 {oldName} 改为 {newName}");
    }
}
