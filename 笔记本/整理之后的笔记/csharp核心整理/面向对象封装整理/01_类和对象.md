类和对象
=============================================

一、概念
类（Class）：具有相同的特征（属性）和相同的行为（方法）的对象的抽象模板。
对象（Object）：类的具体实例。

类 → 人类
对象 → 张三、李四、王五

二、类的声明
class 类名
{
    // 特征 —— 成员变量
    // 行为 —— 成员方法
    // 保护特征 —— 成员属性
    // 构造函数
    // 索引器
    // 运算符重载
    // 静态成员
}

类声明在 namespace（命名空间）中。
类本身默认访问修饰符是 internal。

三、实例化对象
类名 变量名;                 // 声明引用，值为 null
类名 变量名 = null;          // 同上，显式赋 null
类名 变量名 = new 类名();     // 实例化，在堆上分配内存

四、引用类型的特点
当两个类变量指向同一个堆地址时，修改堆上的内容，两个变量都会看到变化。
但如果其中一个变量重新指向了新地址，则两个变量的联系断开。

    Person a = new Person();
    Person b = a;       // a 和 b 指向同一个堆对象
    b.Name = "Tom";     // 通过 b 修改，a.Name 也变成 "Tom"
    b = new Person();   // b 指向新对象，a 不受影响

【注意】非静态的成员方法不要加 static 关键字！
