静态类和静态构造函数
=============================================

一、静态类（static class）
用 static 修饰的类，只能包含静态成员，不能被实例化。

static class MathHelper
{
    public static int Max(int a, int b)
    {
        return a > b ? a : b;
    }

    public static double Sqrt(double x)
    {
        return Math.Sqrt(x);
    }
}

// 使用（直接类名调用）
int max = MathHelper.Max(3, 5);

【特点】
1. 不能实例化（不能 new）
2. 只能有静态成员
3. 不能有实例构造函数（但可以有静态构造函数）
4. 默认隐式 sealed（不能被继承）

二、静态构造函数
在构造函数前加 static，用于初始化静态成员。
一个类只能有一个静态构造函数。

【规则】
1. 不能加访问修饰符（自动 private）
2. 不能有参数
3. 只自动执行一次（在第一次访问该类时调用）
4. 静态类和普通类都可以有静态构造函数

class TestStatic
{
    public static int index1 = 0;
    public static int index2 = 0;

    static TestStatic()  // 静态构造函数
    {
        Console.WriteLine("静态构造函数执行");
    }

    public static void TestFun() { }
}

// 调用
Console.WriteLine(TestStatic.index1);  // 先输出"静态构造函数执行"，再输出 0

【执行顺序验证】
Console.WriteLine("Hello");
Console.WriteLine(TestStatic.index1);

输出：
Hello              ← main 函数第一条语句
静态构造函数执行   ← 第一次访问 TestStatic 时触发
0                  ← 输出静态字段的值

三、普通类的静态构造函数
普通类（非静态类）也可以有静态构造函数，用途相同：
class Player
{
    public static int playerCount = 0;

    static Player()  // 只在第一次访问 Player 类时执行一次
    {
        Console.WriteLine("玩家系统初始化");
    }
}
