运算符重载
=============================================

一、概念
运算符重载让自定义的类或结构体可以使用运算符进行运算。
关键字：operator

二、基本语法（在类或结构体内部声明）
public static 返回类型 operator 运算符(参数列表)

【规则】
1. 必须是 public static 方法
2. 至少有一个参数是当前类（或结构体）类型
3. 不能使用 ref 和 out 参数
4. 不能重载的运算符：成员访问（.）、条件运算符（?:）、赋值运算符（=）
5. 条件运算符（==、!=、<、>、<=、>=）需要成对重载
6. 重载应保持运算符的语义一致性

三、可重载的运算符
算术运算符：+、-、*、/、%
位运算符：&、|、^、<<、>>
比较运算符：==、!=、<、>、<=、>=
递增/递减：++、--

四、示例：重载 + 实现两个坐标相加
class Point
{
    public int X { get; set; }
    public int Y { get; set; }

    public Point(int x, int y)
    {
        X = x;
        Y = y;
    }

    // 重载 + 运算符
    public static Point operator +(Point a, Point b)
    {
        return new Point(a.X + b.X, a.Y + b.Y);
    }

    // 重载 == 和 !=（必须成对）
    public static bool operator ==(Point a, Point b)
    {
        return a.X == b.X && a.Y == b.Y;
    }

    public static bool operator !=(Point a, Point b)
    {
        return !(a == b);
    }
}

// 使用
Point p1 = new Point(1, 2);
Point p2 = new Point(3, 4);
Point p3 = p1 + p2;  // 等价于 p1.operator+(p2)
Console.WriteLine(p3.X);  // 4

六、一元运算符重载
class Counter
{
    public int Value { get; set; }

    public static Counter operator ++(Counter c)  // 前置和后置 ++ 都用这个
    {
        return new Counter { Value = c.Value + 1 };
    }
}
