扩展方法
=============================================

一、概念
扩展方法：为现有的（静态/非静态）类型添加新方法，无需继承或修改原类型。
关键字：static + this（第一个参数用 this 修饰）

二、特点
1. 写在静态类里面
2. 一定是静态方法
3. 第一个参数是被扩展的类型，用 this 修饰
4. 提升程序的可扩展性（为别人封装的类添加方法）
5. 不会真正改变原类型的代码

三、基本语法
访问修饰符 static 返回值 方法名(this 被扩展类型 参数名, 其他参数...)
{
    // 方法体
}

四、示例
// 为 int 扩展一个"说话"方法
public static class IntExtensions
{
    public static void SpeakValue(this int value)
    {
        Console.WriteLine("当前数值是：" + value);
    }

    // 为 string 扩展一个判断是否为空的方法
    public static bool IsNullOrEmpty(this string str)
    {
        return string.IsNullOrEmpty(str);
    }
}

// 使用
int i = 10;
i.SpeakValue();         // 输出：当前数值是：10

string name = "张三";
Console.WriteLine(name.IsNullOrEmpty());  // False

五、冲突处理
当扩展方法名与原类型已有方法重名时：
→ 优先使用原类型的方法，扩展方法不会被调用。

六、注意事项
1. 不能用扩展方法修改原类型的 private 成员（只能读取）
2. 不能扩展 static 类（因为 static 类无法实例化，无法用 this）
3. sealed 类也可以扩展
