成员变量和访问修饰符
=============================================

一、成员变量
成员变量是描述对象"特征"的数据。

class Person
{
    public string name;      // 姓名
    public int age;          // 年龄
    public SexType sex;       // 性别（枚举类型）
    public Person girlfriend; // 引用自身（与结构体不同，类可以引用自己）
    public Person[] friends;  // 数组引用自身（不能在此处初始化）
    public Pet pet;          // 也可以引用结构体
}

【注意】类内部引用自己时不加 static，不是"相当于函数"。

二、枚举类型声明（常与成员变量配合）
enum SexType
{
    Man,
    Woman   // 注意：枚举值应使用帕斯卡命名
}

三、访问修饰符

  修饰符        作用域
  public      任何地方都可以访问
  private     只有本类内部可以访问（不写默认是 private）
  protected   本类及子类可以访问
  internal    同一程序集（Assembly）内可见

四、成员变量的默认值
  值类型（int、float、bool）→ 默认值 0 / 0.0 / false
  引用类型（class）         → 默认值 null
  枚举类型（enum）          → 默认值 0（第一个枚举值）

  示例：
  Person p = new Person();
  Console.WriteLine(p.age);       // 输出 0
  Console.WriteLine(p.name);       // 输出 null

【补充】readonly 修饰符：只能在构造函数或初始化器中赋值，之后不可修改。
