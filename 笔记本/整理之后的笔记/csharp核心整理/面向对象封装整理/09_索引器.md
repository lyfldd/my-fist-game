索引器
=============================================

一、概念
索引器让对象可以像数组一样通过下标访问其内部元素。
本质：是一种带参数的属性。

二、基本语法
访问修饰符 返回类型 this[参数类型 参数名, ...]
{
    get { return 对应元素; }
    set { 设置对应元素 = value; }
}

三、示例：让 Person 对象通过下标访问 friends
class Person
{
    private string name;
    private int age;
    public Person[] friends;

    public Person this[int index]
    {
        get { return friends[index]; }
        set { friends[index] = value; }
    }
}

// 使用
Person p = new Person();
p.friends = new Person[3];
p[0] = new Person { name = "张三" };
p[1] = new Person { name = "李四" };
Console.WriteLine(p[0].name);  // 张三

四、索引器重载
索引器可以有不同参数类型和参数个数（重载）

class Person
{
    private string name;
    private int age;

    // 通过整数下标访问
    public string this[int index]
    {
        get
        {
            switch (index)
            {
                case 0: return name;
                case 1: return age.ToString();
                default: return "";
            }
        }
    }

    // 通过字符串访问（按名称获取属性）
    public string this[string propertyName]
    {
        get
        {
            switch (propertyName)
            {
                case "name": return name;
                case "age":  return age.ToString();
                default:     return "";
            }
        }
    }
}

// 使用
Person p = new Person { name = "张三", age = 20 };
Console.WriteLine(p["name"]);  // 张三
Console.WriteLine(p[0]);        // 张三
