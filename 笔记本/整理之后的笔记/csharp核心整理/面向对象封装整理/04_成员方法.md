成员方法
=============================================

成员方法用来描述对象的行为（相当于普通函数，但属于某个类）。

一、基本语法
访问修饰符 返回值 方法名(参数列表)
{
    // 方法体
}

二、示例
class Person
{
    public string name;
    public int age;

    public void Speak(string s)
    {
        Console.WriteLine("{0}说：{1}", name, s);
    }

    public bool IsAdult()
    {
        return age >= 18;
    }
}

// 使用
Person p = new Person();
p.name = "李四";
p.age = 19;
p.Speak("我爱你");
Console.WriteLine(p.IsAdult());  // True

三、重要规则
1. 非静态方法不加 static
2. 实例成员方法可以访问类的所有成员（静态/非静态都可以）
3. 方法名使用帕斯卡命名法（PascalCase）
4. 局部变量作用域在方法内部
