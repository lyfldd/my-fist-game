二、算术运算符
────────────────────────────────────────

+   加
-   减
*   乘
/   除
%   取余数（取模）

运算规则：
1. 先计算右侧，再赋值给左侧
   int i = 1;
   i = i + 2;   // i 的值变成 3

2. 右侧表达式中同名变量不会在表达式内迭代（用的是进入表达式前的值）
   int i = 5;
   i = i + 1 + i;   // 右侧的两个 i 都是 5，结果是 5+1+5=11

3. 定义时就可以用表达式初始化
   int i = 1 + 2 + 3;   // i = 6

⚠️ 整数除法的坑：
   float f = 1 / 2;    // 结果是 0！因为 1 和 2 都是整数，先整除得 0 再赋给 float
   float f = 1f / 2;   // 结果是 0.5，因为有一个 f 变成了浮点数
   float f = 1 / 2f;   // 同上，结果是 0.5

────────────────────────────────────────
三、复合赋值运算符
────────────────────────────────────────

+=   -=   *=   /=   %=
自己等于自己与右侧进行运算的结果

int i = 1;
i += 3;    // 等同于 i = i + 3;  结果 i = 4
i -= 1;    // 等同于 i = i - 1;  结果 i = 3
i *= 2;    // 等同于 i = i * 2;  结果 i = 6
i /= 3;    // 等同于 i = i / 3;  结果 i = 2
i %= 3;    // 等同于 i = i % 3;  结果 i = 2

四、自增减运算符
────────────────────────────────────────

int a = 1;
a++;   // 后缀：先使用 a 的当前值，再让 a 自增 1
++a;   // 前缀：先让 a 自增 1，再使用 a 的新值

【重点区别示例】
int a = 5;
int b = a++;   // b = 5（先用 a 的值赋给 b，再自增）a 变成 6
int c = ++a;   // c = 7（a 先自增变成 7，再赋给 c）

// a-- 和 --a 同理（自减）

五、算术运算符优先级
────────────────────────────────────────

优先级从高到低：
  括号 ()
  一元运算（++  --  -取负号  !）

────────────────────────────────────────
六、字符串拼接
────────────────────────────────────────

① 用 + 号拼接
   string str = "123";
   str = str + "456";     // str = "123456"
   str = str + 1;         // 数字会自动转为字符串，str = "1231"

② 用 += 复合拼接
   str += "abc";    // 等同于 str = str + "abc"

   ⚠️ 混合运算时注意从左到右的规则：
   str += 1 + 2 + 3;      // 右侧 1+2+3 先数字相加得 6，再拼接 → str + "6"
   str += "" + 1 + 2 + 3; // 右侧第一个是空字符串，后面都变成字符串拼接 → "123"
   str += 1 + 2 + "" + 3; // 1+2 先数字相加得 3，遇到 "" 后变拼接，再拼 3 → "33"

③ string.Format（格式化拼接）
   string.Format("模板字符串", 值1, 值2, ...)
   用占位符 {0} {1} {2}... 替代要填入的值
   
   string result = string.Format("我是{0}，今年{1}岁，想{2}", "张三", 18, "做游戏");
   // result = "我是张三，今年18岁，想做游戏"

④ Console.WriteLine 里直接格式化（效果等同于 string.Format）
   Console.WriteLine("我是{0}，今年{1}岁", "张三", 18);
   ⚠️ 占位符数量可以少于后面的参数（多余的不报错），但不能多于参数（会报错）

⑤ 字符串插值 $（推荐，C# 6.0 之后最常用）
   string name = "张三";
   int age = 18;
   Console.WriteLine($"我是{name}，今年{age}岁");
   // 输出：我是张三，今年18岁

七、关系运算符（比较运算符）
────────────────────────────────────────

// 名称注意：原笔记叫"条件运算符"，正确叫法是"关系运算符"或"比较运算符"
// 三目运算符才叫"条件运算符"

>    大于
<    小于
==   等于
!=   不等于
>=   大于等于
<=   小于等于

// 比较结果返回 bool 值（true 或 false）
bool result = a > b;
bool result = a > 10;
bool result = 5 > 10;   // result = false
bool result = a + 3 > a - 2;   // 先算算术运算，再比较（算术运算符优先级高于关系运算符）

⚠️ 注意：不能写范围比较
   // 错误写法：1 < a < 6（不合法）
   // 正确写法：a > 1 && a < 6（要用逻辑运算符连接）

不同类型的比较规则：
  - 数值类型（int、float、double 等）可以互相用所有关系运算符比较
  - char 可以和数值类型比较（char 会转成 ASCII 码参与比较）
    例如：'A' == 65 → true （'A' 的 ASCII 码就是 65）
  - string 和 bool 只能用 == 和 != 比较，不能用 > < 等

────────────────────────────────────────
八、逻辑运算符
────────────────────────────────────────

&&   逻辑与：两个都为 true，结果才为 true；有一个 false 则为 false（有假则假）
     优先级低于关系运算符
||   逻辑或：有一个为 true 结果就是 true；两个都 false 才为 false（有真则真）
!    逻辑非：取反，true 变 false，false 变 true
     优先级最高（仅次于括号）

优先级：!（最高）> && > ||

bool result = a > 1 && a < 6;    // a 在 (1,6) 范围内
bool result = a < 0 || a > 100;  // a 小于0 或 大于100
bool result = !isAlive;          // isAlive 取反

短路规则（重要！）：
  &&：从左到右，碰到 false 就停止，后面的表达式不再执行
  ||：从左到右，碰到 true 就停止，后面的表达式不再执行
  // 这有时会影响带副作用的表达式，例如函数调用

九、位运算符（进阶内容，了解即可）
────────────────────────────────────────

位运算是对二进制位直接进行操作的运算符

&    位与：对应位都是 1 结果才是 1，有 0 则 0
     int a = 1;  // 二进制：001
     int b = 5;  // 二进制：101
     int c = a & b;  // 结果：001 = 1

|    位或：对应位有 1 则 1，都是 0 才是 0
     int c = a | b;  // 001 | 101 = 101 = 5

^    异或：对应位相同为 0，不同为 1
     int c = a ^ b;  // 001 ^ 101 = 100 = 4

~    位取反：0 变 1，1 变 0（按位取反）
     ~1 结果是 -2（因为有符号整数，需要了解补码知识）

<<   左移：将二进制向左移 n 位，右侧补 0
     int c = 1 << 3;  // 0001 左移3位 = 1000 = 8
     // 规律：左移 n 位 = 乘以 2 的 n 次方

>>   右移：将二进制向右移 n 位
     int c = 8 >> 1;  // 1000 右移1位 = 0100 = 4
     // 规律：右移 n 位 = 除以 2 的 n 次方（取整）
     ⚠️ 注意：有符号整数（如 int）右移时，左侧补的是符号位（正数补 0，负数补 1）
             无符号整数（如 uint）右移时，左侧一律补 0

多个位运算：从左往右依次执行

十、三目运算符（条件运算符）
────────────────────────────────────────

语法：
  条件表达式 ? 值A : 值B

规则：
  - 条件表达式为 true，返回值A
  - 条件表达式为 false，返回值B
  - 值A 和值B 的类型必须相同（或可隐式转换）

示例：
  int a = 10, b = 20;
  int max = a > b ? a : b;   // max = 20（a > b 为 false，取 b）

  bool isAdult = age >= 18 ? true : false;
  // 更简洁写法：bool isAdult = age >= 18;

十一、运算符优先级总结（从高到低）
────────────────────────────────────────

  1. 括号 ()
  2. 一元运算：++  --  -（取负）  !  ~
  3. 乘 *  除 /  取余 %
  4. 加 +  减 -
  5. 移位 <<  >>
  6. 关系运算 >  <  >=  <=
  7. 相等判断 ==  !=
  8. 位与 &
  9. 位异或 ^
  10. 位或 |
  11. 逻辑与 &&
  12. 逻辑或 ||
  13. 三目运算符 ? :
  14. 赋值 =  +=  -=  *=  /= 等
