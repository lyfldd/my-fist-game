五、14 种变量类型
────────────────────────────────────────

大类一：有符号整型（可以存正数、负数）
  sbyte   范围：-128 ~ 127
  short   范围：-32768 ~ 32767
  int     范围：-2147483648 ~ 2147483647   ← 最常用
  long    范围：-9223372036854775808 ~ 9223372036854775807

大类二：无符号整型（只能存正数和 0）
  byte    范围：0 ~ 255
  ushort  范围：0 ~ 65535
  uint    范围：0 ~ 4294967295（约 42 亿）
  ulong   范围：0 ~ 18446744073709551615（约 1.8 万亿亿）

大类三：浮点数（小数）
  float    精度：7~8 位有效数字，字面量后需加 f，例如：float f = 3.14f;
  double   精度：15~17 位有效数字，默认小数类型，例如：double d = 3.14;
  decimal  精度：28~29 位有效数字，字面量后需加 m，精度最高，常用于金融计算
             例如：decimal price = 9.99m;

  ⚠️ 注意：浮点数存储的是近似值，不是精确值。
     整数除以整数结果仍是整数，例如 1/2 = 0（不是 0.5！）
     要得到小数结果，至少一个操作数要是浮点数：1f/2 = 0.5f

大类四：特殊类型
  bool    true / false，表示真或假
  char    存储单个字符（注意：不是字符串！），用单引号
            char c = 'A';   // 字母
            char c = '好';  // 汉字也可以，但只能一个字
  string  存储字符串（任意长度），用双引号
            string s = "Hello World";

八、类型转换
────────────────────────────────────────

① 隐式转换（自动，小类型 → 大类型，不会丢失数据）
   sbyte → short → int → long → float → double
   byte  → short/ushort → int/uint → long/ulong → float → double
   char  → int（char 有对应的 ASCII/Unicode 数值）

② 显式转换（强制转换，括号语法）
   变量类型 变量名 = (目标类型)源变量;
   int i = (int)3.99;   // 结果是 3，直接截断小数（不是四舍五入！）
   int j = (int)'A';    // 结果是 65（char 转 int 得到 ASCII 码）
   ⚠️ 注意：强转可能导致范围溢出或精度丢失

③ Parse 方法（字符串 → 指定类型）
   变量类型.Parse("字符串");
   int i = int.Parse("123");       // ✅ 正确
   float f = float.Parse("3.14");  // ✅ 正确
   // ⚠️ 字符串内容必须能合法转换，否则会报异常
   // 游戏开发中，读取存档/配置文件里的字符串数据时常用

④ Convert 方法（更强大，更常用）
   语法：Convert.To目标类型(值);
   int a = Convert.ToInt32("123");       // 字符串转 int
   float b = Convert.ToSingle("3.14");   // 字符串转 float
   // ⚠️ Convert.ToInt32 遇到小数时会四舍五入（与强转不同！）
   //    例如：Convert.ToInt32(3.5) = 4，(int)3.5 = 3

   常用 Convert 方法对照表：
   Convert.ToSByte()    转 sbyte
   Convert.ToInt16()    转 short
   Convert.ToInt32()    转 int     ← 最常用
   Convert.ToInt64()    转 long
   Convert.ToByte()     转 byte
   Convert.ToUInt16()   转 ushort
   Convert.ToUInt32()   转 uint
   Convert.ToUInt64()   转 ulong
   Convert.ToSingle()   转 float
   Convert.ToDouble()   转 double
   Convert.ToDecimal()  转 decimal
   Convert.ToString()   转 string
   Convert.ToBoolean()  转 bool（非零为 true，0 为 false）

⑤ ToString 方法（任意类型 → string）
   int num = 42;
   string s = num.ToString();   // s = "42"
   // 所有类型都可以调用 .ToString()

十、异常捕获 try-catch-finally
────────────────────────────────────────

// 场景：用户输入内容时，可能输入非法值（比如输入汉字，却要转成数字）
// 这时就需要用 try-catch 来"兜底"，防止程序崩溃

基本语法：
try
{
    // 把可能出错的代码放到这里
}
catch
{
    // 出错了就执行这里的代码
}
finally  // 可以省略
{
    // 不管有没有出错，最后都会执行这里
}

// 完整示例：
try
{
    string str = Console.ReadLine();
    int i = Convert.ToInt32(str);   // ← 如果 str 不是数字，这里会抛出异常
    Console.WriteLine(i);
}
catch
{
    Console.WriteLine("请输入数字！");
}

// 补充：可以在 catch 后面捕获具体的异常类型
try
{
    int i = Convert.ToInt32(Console.ReadLine());
    Console.WriteLine(i);
}
catch (FormatException)
{
    Console.WriteLine("格式错误，请输入整数！");
}
catch (OverflowException)
{
    Console.WriteLine("数字超出范围！");
}
catch (Exception e)   // 兜底，捕获所有异常
{
    Console.WriteLine("发生了错误：" + e.Message);
}
