────────────────────────────────────────
一、while 循环
────────────────────────────────────────

// 先判断条件，条件为 true 才进入循环体
// 如果条件一开始就是 false，一次都不会执行

while (bool类型值)
{
    // 满足条件就执行，执行完再回到 while 判断条件
    // 条件不满足就跳出循环
}

示例：
int i = 0;
while (i < 5)
{
    Console.WriteLine(i);
    i++;
}
// 输出：0 1 2 3 4

────────────────────────────────────────
二、break 和 continue
────────────────────────────────────────

break：立即跳出当前循环，循环后面的代码继续执行
continue：跳过本次循环剩余代码，直接进入下一次循环判断

【break 示例】
int i = 0;
while (true)   // 死循环
{
    ++i;
    Console.WriteLine(i);
    if (i == 10)
    {
        break;   // i 到 10 时跳出循环
    }
}
Console.WriteLine("循环结束，i = " + i);
// 注意：break 在 if 里面，但 if 不是可跳出结构
// break 会向上找最近的循环（while），跳出 while 循环

【continue 示例】
int i = 0;
while (i < 5)
{
    i++;
    if (i == 3)
    {
        continue;   // 跳过 i=3 时的打印，直接回到循环判断
    }
    Console.WriteLine(i);
}
// 输出：1 2 4 5（3 被跳过了）

────────────────────────────────────────
三、do-while 循环
────────────────────────────────────────

// 与 while 的区别：先执行一次循环体，再判断条件
// 不管条件是否满足，至少执行一次

do
{
    // 先执行这里
} while (bool类型值);   // ⚠️ 注意这里有分号

示例：
int i = 0;
do
{
    Console.WriteLine(i);
    i++;
} while (i < 5);
// 输出：0 1 2 3 4

// 典型使用场景：要求用户必须输入一次内容
string input;
do
{
    Console.WriteLine("请输入内容（不能为空）：");
    input = Console.ReadLine();
} while (input == "");

────────────────────────────────────────
四、for 循环
────────────────────────────────────────

for (初始表达式; 条件表达式; 增量表达式)
{
    // 执行循环体
}

执行顺序：
  1. 先执行 初始表达式（只执行一次）
  2. 判断 条件表达式，为 true 就执行循环体，为 false 就退出
  3. 循环体执行完后，执行 增量表达式
  4. 回到步骤2

参数说明：
  初始表达式：一般是声明计数变量，例如 int i = 0
  条件表达式：决定是否继续循环的 bool 表达式
  增量表达式：一般是计数变量的自增减，例如 i++ 或 i--

示例：
for (int i = 0; i < 5; i++)
{
    Console.WriteLine(i);
}
// 输出：0 1 2 3 4

// 倒序循环
for (int i = 5; i > 0; i--)
{
    Console.WriteLine(i);
}
// 输出：5 4 3 2 1

// 三个部分都可以省略（省略条件就是死循环）
for (;;)
{
    // 死循环，需要 break 来退出
}

────────────────────────────────────────
五、foreach 循环（重要，原笔记遗漏）
────────────────────────────────────────

// 用于遍历集合或数组，比 for 更简洁
// 不需要写索引，直接拿到每个元素

foreach (元素类型 元素变量名 in 集合或数组)
{
    // 使用 元素变量名
}

示例（遍历数组）：
int[] numbers = { 1, 2, 3, 4, 5 };
foreach (int num in numbers)
{
    Console.WriteLine(num);
}
// 输出：1 2 3 4 5

示例（遍历字符串中的每个字符）：
string word = "Hello";
foreach (char c in word)
{
    Console.Write(c + " ");
}
// 输出：H e l l o

⚠️ 注意：foreach 遍历时不能修改集合的内容，只能读取

────────────────────────────────────────
六、嵌套循环（补充）
────────────────────────────────────────

// 循环里面可以嵌套循环
// 常用于处理二维数据（如矩阵、九九乘法表等）

// 九九乘法表示例
for (int i = 1; i <= 9; i++)
{
    for (int j = 1; j <= i; j++)
    {
        Console.Write($"{j}×{i}={i * j}\t");
    }
    Console.WriteLine();
}

// 嵌套循环中的 break 只跳出当前所在的那一层循环

────────────────────────────────────────
七、四种循环的对比与选用建议
────────────────────────────────────────

循环        何时使用
while       不确定循环次数，条件满足才执行（可能一次都不执行）
do-while    不确定循环次数，但至少要执行一次
for         知道明确的循环次数，或需要用索引访问元素
foreach     遍历数组、列表等集合时最方便（不需要索引）
