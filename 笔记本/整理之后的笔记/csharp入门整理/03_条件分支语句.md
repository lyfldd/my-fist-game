────────────────────────────────────────
一、if 语句
────────────────────────────────────────

if (bool类型值)
{
    // 条件为 true 时执行这里
}

示例：
if (hp <= 0)
{
    Console.WriteLine("你已经死亡");
}

────────────────────────────────────────
二、if-else 语句
────────────────────────────────────────

if (bool类型值)
{
    // 条件为 true 时执行
}
else
{
    // 条件为 false 时执行
}

示例：
if (score >= 60)
{
    Console.WriteLine("及格");
}
else
{
    Console.WriteLine("不及格");
}

────────────────────────────────────────
三、if-else if-else 语句
────────────────────────────────────────

if (条件1)
{
    // 满足条件1 执行
}
else if (条件2)
{
    // 条件1不满足，满足条件2 执行
}
else if (条件3)
{
    // 可以有任意多个 else if
}
else
{
    // 以上条件都不满足时执行（可省略）
}

示例：
if (score >= 90)
{
    Console.WriteLine("优秀");
}
else if (score >= 75)
{
    Console.WriteLine("良好");
}
else if (score >= 60)
{
    Console.WriteLine("及格");
}
else
{
    Console.WriteLine("不及格");
}

────────────────────────────────────────
四、嵌套 if（补充）
────────────────────────────────────────

// if 可以嵌套在另一个 if 的代码块内
if (isAlive)
{
    if (hp <= 20)
    {
        Console.WriteLine("血量危急！");
    }
}

────────────────────────────────────────
五、switch 语句
────────────────────────────────────────

switch (变量或表达式)
{
    case 常量1:
        // 变量等于常量1时执行
        break;      // ⚠️ 必须有！非空 case 不写 break 会编译报错
    case 常量2:
        // 变量等于常量2时执行
        break;
    default:        // 可省略
        // 所有 case 都不匹配时执行
        break;
}

⚠️ 重要规则（C# 与其他语言的区别）：
  - C# 中，非空的 case 代码块末尾必须有 break（或 return / goto case）
  - 不允许像 C/C++ 那样通过"没有 break"来自动贯穿到下一个 case
  - 只有"空 case"才可以不写 break（利用贯穿）

switch 支持的类型（比 if 适合做等值匹配）：
  整型（int、long 等）、char、string、bool 以及枚举类型

示例（字符串类型）：
string cmd = Console.ReadLine();
switch (cmd)
{
    case "攻击":
        Console.WriteLine("发动攻击！");
        break;
    case "防御":
        Console.WriteLine("进入防御状态！");
        break;
    default:
        Console.WriteLine("未知指令");
        break;
}

────────────────────────────────────────
六、switch 贯穿（空 case 合并）
────────────────────────────────────────

// 【正确的贯穿写法】：空 case（没有任何代码）可以贯穿到下一个 case
int a = 1;
switch (a)
{
    case 1:         // case 1 没有代码，会贯穿到 case 2
    case 2:
        Console.WriteLine("a 是 1 或 2");
        break;
    default:
        break;
}

// 【非空 case 想贯穿】：需要用 goto case
switch (a)
{
    case 1:
        Console.WriteLine("满足case1");
        goto case 2;    // 主动跳转到 case 2
    case 2:
        Console.WriteLine("满足case2或由case1跳来");
        break;
}

────────────────────────────────────────
七、if vs switch 的选择建议
────────────────────────────────────────

使用 if：
  - 条件是范围判断（>  <  >=  <=）
  - 条件是复杂逻辑表达式（&&  ||）
  - 条件数量较少（1~3 个）

使用 switch：
  - 对同一个变量进行多个精确等值匹配
  - 条件数量较多（代码更整洁）
  - 匹配字符串或枚举值
