修正的错误（共 11 处）
────────────────────────────────────────

① a.length → a.Length（C# 区分大小写，数组用 .Length）
   文件：01、03

② "不能再函数语句块里面声明" → "不能在函数语句块里面声明"（错别字）
   文件：01

③ "应用类型" → "引用类型"（错别字）
   文件：05

④ "值类型存储在堆上，需要手动申请和释放"的描述错误
   正确：C# 中堆上内存由 GC（垃圾回收器）自动管理，不需要手动释放
   文件：05

⑤ ref 描述"可以把值类型转换为引用类型"说法不准确
   正确：ref 是按引用传递，让函数对外部变量的修改生效
   文件：06

⑥ "代码快" → "代码块"（错别字）
   文件：02

⑦ "在同一一个语句快" → "在同一个语句块"（错别字）
   文件：03

⑧ "数量想同" → "数量相同"（错别字）
   文件：03

⑨ "更具" → "根据"（错别字）
   文件：03

⑩ params 参数"也可以在前面加ref"说法错误
   正确：C# 不支持 ref params 组合
   文件：03

⑪ 结构体笔记中多余的 using System.Security.Cryptography 引用（误粘贴）
   文件：07

⑫ "达到定制的目的" → "达到终止的目的"（可能是语音输入误差）
