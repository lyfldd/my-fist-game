────────────────────────────────────────
一、冒泡排序
────────────────────────────────────────

核心思想：
  每轮遍历时，相邻两个元素比较大小，大的向右"冒泡"
  每轮结束后，最大值被"沉"到最右端
  n-1 轮后，排序完成

// 图解（升序，以 [3,1,4,1,5] 为例）
// 第 1 轮：3和1比→交换→1和4比→不换→4和1比→交换→4和5比→不换 → [1,3,1,4,5]，5确定
// 第 2 轮：1和3比→不换→3和1比→交换→3和4比→不换 → [1,1,3,4,5]，4确定
// 以此类推...

// 完整代码（升序）
static void BubbleSort(int[] a)
{
    for (int j = 0; j < a.Length - 1; j++)         // 轮数：n-1 轮
    {
        for (int k = 0; k < a.Length - 1 - j; k++) // 每轮比较次数（已排好的不用再比）
        {
            if (a[k] > a[k + 1])    // 前面的比后面的大，就交换
            {
                int temp = a[k];
                a[k] = a[k + 1];
                a[k + 1] = temp;
            }
        }
    }
}

// 降序：只需把 > 改成 <
// if (a[k] < a[k + 1]) → 交换

// 完整示例（含随机数生成）
Random random = new Random();
int[] a = new int[10];
for (int i = 0; i < 10; i++)
{
    a[i] = random.Next(0, 101);   // 生成 0~100 的随机数
}

// 打印排序前
Console.Write("排序前：");
foreach (int num in a) Console.Write(num + " ");
Console.WriteLine();

// 冒泡排序
for (int j = 0; j < a.Length - 1; j++)
{
    for (int k = 0; k < a.Length - 1 - j; k++)
    {
        if (a[k] > a[k + 1])
        {
            int temp = a[k];
            a[k] = a[k + 1];
            a[k + 1] = temp;
        }
    }
}

// 打印排序后
Console.Write("排序后：");
foreach (int num in a) Console.Write(num + " ");
Console.WriteLine();

二、选择排序
────────────────────────────────────────

核心思想：
  每轮从"未排序部分"中找到最小值的下标
  将最小值和未排序部分的第一个元素交换
  每轮确定一个元素的最终位置，n-1 轮后完成

// 图解（升序，以 [3,1,4,1,5] 为例）
// 第 1 轮：从全部中找最小值 1（下标1），与 a[0] 交换 → [1,3,4,1,5]
// 第 2 轮：从 a[1]~ 找最小值 1（下标3），与 a[1] 交换 → [1,1,4,3,5]
// 以此类推...

// 完整代码（升序）
static void SelectionSort(int[] a)
{
    for (int m = 0; m < a.Length - 1; m++)    // 轮数：n-1 轮
    {
        int minIndex = m;                      // 假设当前位置是最小值
        for (int n = m + 1; n < a.Length; n++) // 从后面找更小的
        {
            if (a[minIndex] > a[n])
            {
                minIndex = n;                  // 记录更小值的下标
            }
        }
        // 把找到的最小值放到 a[m] 的位置
        int temp = a[m];
        a[m] = a[minIndex];
        a[minIndex] = temp;
    }
}

// 降序：只需把 > 改成 <

────────────────────────────────────────
三、两种排序的对比
────────────────────────────────────────

特性              冒泡排序              选择排序
----------------------------------------------------------
核心操作          相邻元素两两比较交换   找最小值，交换到前面
交换次数          多（每次比较都可能交换）少（每轮只交换一次）
稳定性            稳定（相同元素相对顺序不变）不稳定
时间复杂度        O(n²)                 O(n²)
适用场景          教学入门，小数据量      教学入门，小数据量

// 两者时间复杂度相同，都是 O(n²)，实际项目中数据量大时应用 Array.Sort()
