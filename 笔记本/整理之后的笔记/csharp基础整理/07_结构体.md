────────────────────────────────────────
一、结构体的基本概念
────────────────────────────────────────

结构体（struct）：
  - 一种自定义的值类型
  - 是数据（字段）和函数（方法）的集合
  - 可以把相关的数据打包在一起，方便管理
  - 类似于"轻量级的类"，适合表示小型的数据结构

声明位置：
  ✅ namespace 语句块中（推荐）
  ✅ class 语句块中
  ✅ struct 语句块中（嵌套结构体）
  ❌ 函数语句块中（不能在函数内声明结构体）

────────────────────────────────────────
二、结构体的基本语法
────────────────────────────────────────

// 结构体由三部分组成：变量（字段）、构造函数（可选）、方法（函数）

struct Student
{
    // ① 访问修饰符
    // public：公共的，可以被外部访问
    // private：私有的，只能在结构体内部使用（默认不写就是 private）

    // ② 变量（字段）
    // 结构体中声明字段时不能直接赋初始值（C# 9 之前的限制）
    // 还可以包含其他结构体类型的字段
    public int age;
    public bool sex;
    public int number;
    public string name;

    // ③ 构造函数（可选，但推荐定义，方便初始化）
    // 规则：没有返回值、函数名和结构体名相同、必须有参数
    //       如果声明了构造函数，必须对所有字段赋值
    public Student(int age, bool sex, int number, string name)
    {
        // 用 this 区分结构体的字段和传入的参数（同名时）
        this.age = age;
        this.sex = sex;
        this.number = number;
        this.name = name;
    }
    // 构造函数也可以重载（不同参数列表的多个版本）

    // ④ 方法（函数）
    // ⚠️ 注意：结构体里的方法不加 static
    public void Speak()
    {
        Console.WriteLine($"我的名字是 {name}，今年 {age} 岁");
    }
}

────────────────────────────────────────
三、结构体的使用
────────────────────────────────────────

// 方式 1：声明后逐个赋值
Student s1;
s1.age = 10;
s1.sex = true;
s1.number = 1;
s1.name = "张三";
s1.Speak();

// 方式 2：使用构造函数（更简洁）
Student s2 = new Student(18, true, 2, "李四");
s2.Speak();

────────────────────────────────────────
四、结构体是值类型（重要！）
────────────────────────────────────────

// 结构体是值类型，赋值时会复制一份，两个变量互相独立
Student a = new Student(10, true, 1, "张三");
Student b = a;       // b 是 a 的独立副本
b.name = "李四";
Console.WriteLine(a.name);   // 输出 "张三"（a 不受影响）
Console.WriteLine(b.name);   // 输出 "李四"

// 这和 class（类）不同，class 是引用类型，赋值后两个变量指向同一个对象

────────────────────────────────────────
五、结构体的限制（补充）
────────────────────────────────────────

1. 不能显式定义无参构造函数（C# 10 之前）
   ❌ public Student() { }   // 编译报错（C# 9 及以前）
   ✅ C# 10+ 允许结构体有无参构造函数

2. 结构体不能继承其他结构体或类
   ❌ struct A : B { }     // 报错
   ✅ 结构体可以实现接口

3. 结构体不能被继承（隐式 sealed）

────────────────────────────────────────
六、结构体 vs 类（简要对比，详见核心笔记）
────────────────────────────────────────

特性              结构体（struct）        类（class）
------------------------------------------------------------
类型              值类型                  引用类型
内存位置          栈                      堆
赋值行为          复制一份（独立）         共享同一对象
继承              不支持                  支持
适合场景          小型数据（坐标、颜色）  复杂对象（玩家、怪物）

// 示例：Vector2（二维坐标）适合用结构体
struct Vector2
{
    public float x;
    public float y;

    public Vector2(float x, float y)
    {
        this.x = x;
        this.y = y;
    }

    public float Distance(Vector2 other)
    {
        float dx = x - other.x;
        float dy = y - other.y;
        return (float)Math.Sqrt(dx * dx + dy * dy);
    }
}
