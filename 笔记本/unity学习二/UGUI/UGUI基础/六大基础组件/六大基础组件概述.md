## 一、六大基础组件概述

### 1. 主要学习内容

- **使用方式**：在 Unity 中直接右键使用 UGUI，无需额外引入内容
- **组件位置**：六大基础组件分别位于 `Canvas` 和 `EventSystem` 对象上
- **组件概述**：了解六大组件各自的功能和作用

### 2. 如何在 Unity 中使用 UGUI

**创建步骤：**

1. 新建 Unity 工程（3D 项目）
2. 在 Assets 下创建 `Scripts` 和 `Lesson1` 文件夹
3. 新建场景并保存

**UI 创建：**

- **方法**：在 Hierarchy 窗口右键选择 UI 选项
- **示例**：创建 Image 控件会自动生成 Canvas 和 EventSystem

### 3. 六大基础组件在哪里

**Canvas 对象组件：**

| 组件 | 说明 |
|------|------|
| Canvas | 画布组件 |
| Canvas Scaler | 分辨率自适应 |

**EventSystem 对象组件：**

| 组件 | 说明 |
|------|------|
| EventSystem | 输入事件系统 |
| Standalone Input Module | 独立输入模块 |

![EventSystem 对象的两个组件](images/page2_img2.png)

> **重要性说明**：创建 UI 控件时这两个对象会自动生成，是 UGUI 系统的核心。

### 4. 六大基础组件概述

| 组件 | 功能 |
|------|------|
| **Canvas** | 画布组件，负责渲染所有 UI 控件 |
| **Canvas Scaler** | 实现分辨率自适应 |
| **Graphic Raycaster** | 处理射线事件交互 |
| **RectTransform** | 控制 UI 位置和锚点对齐 |
| **EventSystem** | 玩家输入事件响应系统 |
| **Standalone Input Module** | 独立输入模块，监听玩家操作 |

### 5. 内容总结

- **使用要点**：直接右键创建，无需额外设置
- **组件位置**：Canvas 和 EventSystem 两个核心对象
- **功能概述**：六大组件各司其职，共同构成 UGUI 系统
