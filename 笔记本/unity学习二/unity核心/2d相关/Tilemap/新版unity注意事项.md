|选项|核心作用|末日沙盒场景用法举例|
|:--|:--|:--|
|**Animated Tile**|带帧动画的瓦片|燃烧的火焰、摇晃的树枝、滴水的管道、发光的霓虹灯，给静态场景加动态效果|
|**Rule Tile**|按邻居自动拼接的瓦片（最常用）|用一套 “边缘 / 角落 / 中心” 的瓦片，自动生成平滑的地面、墙壁、废墟轮廓，不用手动拼边角|
|**Isometric Rule Tile**|等距视角专用的 Rule Tile|如果你想做斜 45° 视角的末日场景，用它处理等距瓦片的拼接逻辑|
|**Hexagonal Rule Tile**|六边形地图专用的 Rule Tile|做六边形格子的策略 / 生存玩法（比如格子化的城市探索）|
|**Rule Override Tile**|覆盖 Rule Tile 规则的变体|比如同一种墙壁，想做破损 / 完好两种状态，用它复用拼接逻辑，只换图片|
|**Advanced Rule Override Tile**|带更多条件的 Rule Override Tile|可以加更复杂的规则，比如 “只有周围有僵尸时，瓦片才显示破损版本”|
|**Custom Rule Tile Script**|自定义规则瓦片模板|你可以写 C# 脚本，实现完全自定义的瓦片逻辑，比如根据玩家距离改变状态、和生存数值联动|

|选项|核心作用|末日沙盒场景用法举例|
|:--|:--|:--|
|**Rectangular**|标准矩形网格（传统 2D 俯视角）|你的俯视角 3D 生存沙盒最常用的类型，比如城市街道、建筑内部的方形网格|
|**Hexagonal Flat-Top**|六边形网格（平边在上）|做六边形格子的探索地图，比如废墟城市的格子化探索，或者回合制遭遇战|
|**Hexagonal Pointed-Top**|六边形网格（顶点在上）|和上面类似，只是六边形的朝向不同，看你的美术资源是哪种朝向|
|**Isometric**|等距斜 45° 网格|做斜视角的末日场景，比如斜向的城市废墟、等距建筑|
