以下是 `Tile Palette` 菜单中所有网格布局类型的完整说明，包含 Inspector 参数、功能说明与末日沙盒项目实战用法。

---

## 1. Rectangular 矩形网格（默认 / 最常用）

**作用**：标准直角网格，也是你俯视角 3D 末日沙盒项目的首选布局。

表格

|参数 / 特性|功能说明|实战用法|
|:--|:--|:--|
|网格形状|规则正方形 / 矩形格子，上下左右四向相邻|城市街道、建筑内部、废墟地图的基础布局|
|Cell Size|单瓦片像素尺寸，可自定义宽高|32x32 / 64x64 像素是末日场景最常用的尺寸，和美术资源对齐|
|Neighbor Logic|四向（上下左右）+ 四斜角（共 8 方向）邻居判定|完美适配 `Rule Tile` 做墙壁、地面的无缝拼接|
|Sorting Order|按 Y 轴从上到下排序，数值越大渲染越靠前|地面层设低数值，墙壁 / 建筑层设高数值，保证正确遮挡|
|适用场景|俯视角 2D/3D、横版、斜视角转 2D 的常规地图|你的末日生存项目直接选这个，无需额外适配|

---

## 2. Hexagonal Flat-Top 六边形网格（平边朝上）

**作用**：六边形网格布局，六边形的平边在上下方，顶点在左右两侧。

表格

|参数 / 特性|功能说明|实战用法|
|:--|:--|:--|
|网格形状|六边形格子，平边在上 / 下，顶点在左 / 右|适合横向延伸的探索地图、战棋类格子地图|
|Neighbor Logic|6 个方向邻居判定：左、右、左上、右上、左下、右下|适配 `Hexagonal Rule Tile` 做六边形地块拼接|
|Cell Size|六边形宽高比固定为 `2 : √3`（约 1.1547 : 1），需按比例设置|美术资源要按此比例制作，避免瓦片错位|
|适用场景|六边形探索 / 战棋系统、格子化废墟城市探索、回合制遭遇战|可用于末日城市的六边形格子探索玩法|


---

## 3. Hexagonal Pointed-Top 六边形网格（顶点朝上）

**作用**：六边形网格布局，六边形的顶点在上下方，平边在左右两侧。

表格

|参数 / 特性|功能说明|实战用法|
|:--|:--|:--|
|网格形状|六边形格子，顶点在上 / 下，平边在左 / 右|适合做纵向延伸的地图、塔防 / 策略类布局|
|Neighbor Logic|6 个方向的邻居判定（左、右、左上、右上、左下、右下）|和 `Hexagonal Rule Tile` 配合，处理六边形地块拼接|
|Cell Size|宽高比固定为 √3:2，需按比例设置|美术资源要按这个比例制作，保证瓦片无缝拼接|
|适用场景|六边形塔防、纵向延伸的废墟走廊、策略格子地图|末日场景中可以用来做建筑内部的六边形格子布局|

---

## 4. Isometric 等距网格（斜 45° 视角）

**作用**：等距视角专用网格，适配斜 45° 的伪 3D 视角。

表格

|参数 / 特性|功能说明|实战用法|
|:--|:--|:--|
|网格形状|菱形格子，模拟斜 45° 视角的透视效果|伪 3D 斜视角末日场景、等距建筑 / 废墟地图|
|Neighbor Logic|等距视角专属的前后层邻居判定，自带层级遮挡逻辑|适配 `Isometric Rule Tile` 做斜向建筑的前后层拼接|
|Cell Size|通常设置为 2:1 的宽高比（如 64x32 像素）|美术资源要按等距透视制作，保证瓦片对齐|
|Sorting Order|自动按格子的 X/Y 坐标计算层级，靠前的瓦片渲染更靠上|无需手动设置，Unity 会自动处理前后遮挡|
|适用场景|斜 45° 伪 3D 末日场景、等距城市废墟、斜视角建筑|可以用来做风格化的斜视角末日城市地图|

---

## 补充：Tile Palette 通用 Inspector 参数

所有网格类型的调色板，都自带以下核心参数：

表格

|参数|功能说明|实战用法|
|:--|:--|:--|
|Grid|网格类型选择（就是上面 4 种）|俯视角项目固定选 `Rectangular`|
|Cell Size|单瓦片的像素尺寸|和你的美术资源像素大小保持一致（如 32x32）|
|Cell Swizzle|坐标转换规则，适配不同视角的渲染顺序|矩形网格默认即可，等距 / 六边形网格会自动适配|
|Sort Order|瓦片在调色板中的排序基准|保持默认即可，不影响场景渲染|
