## 1. None（默认）

**作用**：不开启聚焦功能，相机保持默认行为。

表格

|特性|说明|实战用法|
|:--|:--|:--|
|行为模式|相机自由移动、缩放，不会自动对齐 Tilemap 或 Grid|适合大范围地图编辑、查看整体布局|
|适用场景|不需要精准对齐的场景，比如调整相机位置、预览地图效果|日常开发默认即可|

## 2. Tilemap

**作用**：相机聚焦到当前选中的 `Tilemap` 上，自动对齐该 Tilemap 的边界和中心。

表格

|特性|说明|实战用法|
|:--|:--|:--|
|行为模式|相机自动定位并聚焦到你选中的 Tilemap 组件，缩放会以 Tilemap 为中心|专注编辑某一层瓦片（比如只画地面层或墙壁层）|
|对齐逻辑|相机视口会自动适配 Tilemap 的边界，避免超出画布|快速查看当前层的瓦片，方便排查错位、漏画问题|
|适用场景|单层级地图编辑、精细化绘制某一层的细节|编辑建筑、废墟、装饰等单一层级时推荐使用|

---

## 3. Grid

**作用**：相机聚焦到整个 `Grid` 上，自动对齐整个网格容器的边界和中心。

表格

|特性|说明|实战用法|
|:--|:--|:--|
|行为模式|相机自动定位并聚焦到场景中的 `Grid` 根节点，缩放以整个网格为中心|同时查看所有 Tilemap 层的整体布局|
|对齐逻辑|相机视口会适配整个 Grid 的边界，包含所有子 Tilemap|检查多层地图的整体对齐、层级遮挡是否正常|
|适用场景|多图层地图编辑、整体布局调整、检查不同层级的瓦片对齐|编辑完所有层级后，用这个选项快速预览整体效果|

## 创建新调色板参数详解

表格

|参数|功能说明|末日沙盒项目实战用法|
|:--|:--|:--|
|**Name**|调色板文件的名称|建议命名为 `Wasteland_Ground` / `Wasteland_Walls` 这类清晰的名字，方便后续管理|
|**Grid**|调色板的网格布局类型，和你 Tilemap 的网格类型必须一致|俯视角项目固定选 `Rectangle`；六边形 / 等距地图对应选对应的类型|
|**Cell Size**|单个瓦片的尺寸，控制瓦片和网格的对齐|`Automatic`：自动根据精灵大小设置（推荐，适合新手）<br><br>`Manual`：手动设置 X/Y/Z，和你的美术资源像素尺寸一致（如 32x32、64x64）|
|**Sort Mode**|瓦片的渲染排序模式|`Default`：按 `Sort Axis` 排序（推荐）<br><br>`Custom`：自定义排序规则，一般不用改|
|**Sort Axis**|瓦片排序的坐标轴，决定前后遮挡顺序|默认 `(0,0,1)` 表示按 Z 轴排序，数值越大渲染越靠前，保证地面层在下方、墙壁层在上方|

---

## 关键设置建议（你的末日项目）

1. **基础配置**
    
    - `Name`：命名为 `Survival_TilePalette` 或按功能拆分（如 `GroundPalette` / `WallPalette`）
    - `Grid`：选 `Rectangle`（标准矩形网格，适配你的俯视角场景）
    - `Cell Size`：保持 `Automatic`，Unity 会自动适配你的瓦片精灵大小，避免错位
    - `Sort Mode`：保持 `Default`，`Sort Axis` 用默认值即可
    
2. **进阶配置（可选）**
    
    - 如果你的瓦片是 32x32 像素，可以手动设置 `Cell Size` 为 `(32,32,0)`，确保对齐精度
    - 如果需要多层地图（地面 / 墙壁 / 装饰），可以创建多个调色板，分别管理不同类型的瓦片，避免混乱

## 快速上手流程

1. 填写 `Name`，选择 `Grid` 为 `Rectangle`
2. `Cell Size` 保持 `Automatic`，其他参数默认
3. 点击 `Create`，保存调色板文件到项目的 `Assets/TilePalettes` 文件夹下
4. 把你的瓦片精灵拖进调色板，就可以开始绘制地图了

## 三、Default Brush（默认笔刷）Inspector 面板

控制你绘制瓦片时的行为逻辑，是影响地图绘制效率的关键：

表格

|参数|功能说明|末日项目用法|
|:--|:--|:--|
|`Script`|当前笔刷的脚本，默认是 `GridBrush`|一般不用修改，除非你要自定义笔刷逻辑|
|`Flood Fill Contiguous Only`|填充桶是否只填充连通区域|✅ 勾选：只填充和点击位置连通的同类型瓦片（推荐，避免误操作）<br><br>❌ 取消：会填充整个 Tilemap 里所有同类型瓦片，容易画崩地图|
|`Lock Z Position`|绘制时是否锁定瓦片的 Z 轴位置|✅ 勾选：所有瓦片画在同一个 Z 层，不会出现层级混乱（推荐）<br><br>❌ 取消：可以在不同 Z 层绘制，适合多层地图（如地面 + 墙壁 + 装饰）|
|`Z Position`|锁定的 Z 轴位置数值|一般设为 `0`，如果有多层地图，可以设为 `1/2` 等，控制渲染层级（数值越大越靠前）|
|`Reset`|重置笔刷参数为默认|配置乱了点一下就能恢复|

## 四、完整使用流程（你的末日项目）

1. **创建调色板**：点击 `New Palette` 下拉菜单 → `Create New Palette`，命名并设置好网格类型（矩形）和瓦片尺寸。
2. **导入瓦片素材**：把你的地面、墙壁、废墟瓦片精灵拖进调色板网格里，按类型整理好。
3. **准备场景**：在 Hierarchy 里创建一个 `Grid`，再在里面创建 `Tilemap`（比如 `GroundLayer` / `WallLayer`），选中它。
4. **开始绘制**：
    
    - 用 `Paint` 工具画地面、墙壁
    - 用 `Box Fill` 快速铺满大片区域
    - 用 `Flood Fill` 一键填充连通的房间地面
    - 用 `Erase` 擦除画错的地方
