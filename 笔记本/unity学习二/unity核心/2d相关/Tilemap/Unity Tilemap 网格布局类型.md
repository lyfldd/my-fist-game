## 1. Rectangular 矩形网格（最常用）

**作用**：标准直角网格，也是你的俯视角末日沙盒项目的首选。

表格

|特性|说明|实战用法|
|:--|:--|:--|
|格子形状|正方形 / 矩形，四向（上下左右）相邻|城市街道、建筑内部、废墟地图的基础布局|
|邻居判定|8 方向（四向 + 四斜角）邻居逻辑|完美适配 Rule Tile 做墙壁、地面的无缝拼接|
|适用场景|俯视角 2D/3D、横版、斜视角转 2D 的常规地图|你的末日生存项目直接选这个，无需额外适配|

---

## 2. Hexagonal - Flat-Top 六边形网格（平边朝上）

**作用**：六边形网格，平边在上下方，顶点在左右两侧。

表格

|特性|说明|实战用法|
|:--|:--|:--|
|格子形状|六边形，平边在上 / 下，顶点在左 / 右|适合横向延伸的探索地图、战棋类格子地图|
|邻居判定|6 方向邻居（左、右、左上、右上、左下、右下）|适配 Hexagonal Rule Tile 做六边形地块拼接|
|宽高比|2 : √3（约 1.1547 : 1），需按比例设置瓦片尺寸|美术资源要和网格比例匹配，避免错位|
|适用场景|六边形探索 / 战棋系统、格子化废墟城市探索、回合制遭遇战|可用于末日城市的六边形格子探索玩法|

---

## 3. Hexagonal - Pointed-Top 六边形网格（顶点朝上）

**作用**：六边形网格，顶点在上下方，平边在左右两侧。

表格

|特性|说明|实战用法|
|:--|:--|:--|
|格子形状|六边形，顶点在上 / 下，平边在左 / 右|适合纵向延伸的地图、塔防 / 策略类布局|
|邻居判定|6 方向邻居（上、下、左上、右上、左下、右下）|适配 Hexagonal Rule Tile 做六边形地块拼接|
|宽高比|√3 : 2（约 0.866 : 1），需按比例设置瓦片尺寸|美术资源要和网格比例匹配，避免错位|
|适用场景|六边形塔防、纵向延伸的废墟走廊、策略格子地图|可用于末日建筑内部的六边形格子布局|

---

## 4. Isometric 等距网格（斜 45° 视角）

**作用**：等距视角专用网格，适配斜 45° 的伪 3D 视角。

表格

|特性|说明|实战用法|
|:--|:--|:--|
|格子形状|菱形格子，模拟斜 45° 透视效果|伪 3D 斜视角末日场景、等距建筑 / 废墟地图|
|邻居判定|等距视角专属前后层邻居逻辑，自带层级遮挡|适配 Isometric Rule Tile 做斜向建筑的前后层拼接|
|宽高比|通常 2:1（如 64x32 像素）|美术资源需按等距透视制作，保证对齐|
|适用场景|斜 45° 伪 3D 末日场景、等距城市废墟、斜视角建筑|可用于风格化的斜视角末日城市地图|

---

## 5. Isometric Z as Y 等距网格（Z 轴映射为 Y 轴）

**作用**：特殊的等距网格，将 Z 轴坐标映射为 Y 轴渲染顺序，专门处理带高度的等距场景。

表格

|特性|说明|实战用法|
|:--|:--|:--|
|坐标逻辑|把瓦片的 Z 轴位置作为 Y 轴排序依据，实现高度层级|做有高度差的等距建筑，比如带楼层的废墟、高台|
|渲染顺序|自动按 Z 值排序，靠前的瓦片渲染更靠上|无需手动设置 Sorting Order，自动处理前后遮挡|
|适用场景|带高度的伪 3D 等距场景、多层建筑、立体废墟|可用于制作带高低差的末日建筑场景|

---

## 补充：和 Tile Palette 的对应关系

你新建 Tilemap 时选的网格类型，必须和 Tile Palette 里创建调色板时选的网格类型**完全一致**，否则会出现瓦片错位、渲染异常：

- Rectangular ↔ Rectangular
- Hexagonal Flat-Top ↔ Hexagonal Flat-Top
- Hexagonal Pointed-Top ↔ Hexagonal Pointed-Top
- Isometric / Isometric Z as Y ↔ Isometric
