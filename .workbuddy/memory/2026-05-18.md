## 车辆系统 Editor v7 — 轮子旋转修复

- **问题**：v6 世界空间 `LookRotation(worldForward, Vector3.up)` 在 X=-90° 时 worldForward=世界Y=Vector3.up → forward==up → **LookRotation 退化**，返回垃圾旋转（Y=90°）
- **根因**：`LookRotation` 的 forward 和 up 不能同向，世界空间的 worldForward 恰好等于 worldUp
- **修复**：轮子旋转改回**本地空间**直接构建：
  ```
  localSuspension = Inv(rootRot) * Vector3.up   // 悬挂=本地Z
  localAxle       = rawBounds.rightAxis          // 车轴=本地X
  localRoll       = cross(axle, suspension)      // 滚动=本地-Y
  wheelRot        = LookRotation(localRoll, localSuspension)
  ```
- **结果**：轮子欧拉角应为 `(-90, 0, 0)`（等价 270,0,0），不再是 Y=90°
- **文件**：`CreateVehicleData.cs` v6→v7（头部注释、旋转计算、日志全更新）
- **MEMORY.md**：已更新

## 车辆系统 悬架弹跳修复

- **现象**：车辆落地后弹来弹去（欠阻尼震荡）
- **根因**：VehicleController.ApplyWheelPhysics() 每帧覆盖悬架阻尼为 VehicleData.suspensionDamper（默认 2000，OffRoad 设为 2500），远低于临界阻尼 ~8000
- **修复**（3 文件）：
  - `VehicleController.cs`: 删除 FixedUpdate 中的阻尼覆盖，移到 ApplyConfig()（Start 执行一次），值 ≥ 8000
  - `VehicleData.cs`: 默认 suspensionDamper 2000 → 8000
  - `CreateVehicleData.cs`: OffRoad 预设 suspensionDamper 2500 → 8000

## 车辆系统 SHIFT加速 + 防侧翻 (2026-05-18)

### 需求
- SHIFT 两档速度：不按=原速，按下=maxSpeed×2
- SHIFT 油耗 1.5 倍
- 车辆过弯容易侧翻需要修复

### 改动（3 文件）

**VehicleData.cs**: 新增 4 个配置字段
- `boostSpeedMultiplier = 2f` — SHIFT 速度倍率
- `boostFuelMultiplier = 1.5f` — SHIFT 油耗倍率
- `antiRollStiffness = 5000f` — 反侧倾杆刚度
- `centerOfMassYOffset = -0.3f` — 重心下移量

**VehicleController.cs**: 核心逻辑
- `_isBoosting` + `SetBoost()` — 加速状态
- `EffectiveMaxSpeed` 属性 — 按 boost 动态计算最高速度
- 油耗：`fuelRate *= vehicleData.boostFuelMultiplier`
- `ApplyAntiRoll()` — 反侧倾杆：检测左右轮悬挂压缩差，压缩多侧上推、压缩少侧下拉
- `ApplyRollStabilization()` — 角速度阻尼：本地空间对 X(pitch)×0.95、Z(roll)×0.92，Y(yaw)不处理保持正常转向
- `ApplyConfig()` 中设置 `_rb.centerOfMass` 下移
- 调试 GUI 显示当前/最高速度和 SHIFT 状态

**VehicleInputLock.cs**: `Input.GetKey(KeyCode.LeftShift)` → `_vehicleController.SetBoost()`

### 防侧翻原理（三层防护）
1. **降低重心 (centerOfMassYOffset)** — 物理上越重的车越不容易翻，重心越低稳定性越好
2. **反侧倾杆 (AntiRoll)** — 模拟真车底盘连杆，转弯时外侧轮被压缩、内侧轮抬起，反侧倾杆施加反向力把车身拉平
3. **角速度阻尼 (RollStabilization)** — 如果车身已经产生翻滚角速度，每帧衰减 5-8%，不让它越滚越猛
