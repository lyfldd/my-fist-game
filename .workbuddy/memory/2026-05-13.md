### 地图色块丢失修复（16:04）
- **根因**: SimpleVertexColor.shader 第一个 SubShader 是 URP 专用（引用 `Packages/com.unity.render-pipelines.universal/...`），项目没有 URP 包。Unity shader 编译规则：任一 SubShader 编译失败 → 整个 shader 资产无效 → 后面的 Built-in 备胎 SubShader 也不生效 → 所有 Chunk Mesh 渲染失败（不可见）
- **修复**: 重写 SimpleVertexColor.shader 为纯 Built-in RP：
  - ForwardBase Pass：主方向光 + 阴影 + 自定义 `_AmbientStrength` / `_DiffuseStrength` 参数
  - ForwardAdd Pass：额外点光源，Blend One One
  - ShadowCaster Pass：`V2F_SHADOW_CASTER` 宏标准实现
  - 删除所有 URP HLSL include 路径
- **MeshStage.cs 清理**: 删除 `Shader.Find("Universal Render Pipeline/Lit")` 回退路径（在此项目无意义）
- **MEMORY.md 更新**: 渲染管线说明改为最终确认版，禁止使用 URP Lit 的规则写入
