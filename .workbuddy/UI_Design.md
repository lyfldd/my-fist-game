# 总面板 UI 设计思路

## 一、触发方式

按 **B** 键打开/关闭总面板。打开时游戏暂停（Time.timeScale = 0）。

## 二、整体布局（三区横向排列）

```
┌──────────────┬──────────────────────┬──────────────────┐
│   Zone 1      │      Zone 2          │     Zone 3        │
│  (左 38%)     │     (中 40%)         │    (右 22%)       │
│              │                      │                   │
│ 人物轮廓预览  │  容器网格区域         │  背包网格          │
│ + 装备槽位   │  (上衣口袋/裤子口袋   │  + 人物属性        │
│ (头/上衣/   │   腰带/胸挂)          │  (护甲/保暖)      │
│  裤/腰/背)  │   + C5预留区          │                   │
│              │                      │                   │
└──────────────┴──────────────────────┴──────────────────┘
```

### Zone 1（左）：人物 + 装备
- 上半部分：纯 UI 绘制的人物身体轮廓（非模型/摄像机）
  - 画个简单人体轮廓，按装备位分区域
  - 装备槽在人物轮廓右侧并列排布
- 装备槽位列表（从上到下）：
  - 头部（Head）：头盔
  - 上衣（Tops）：战术夹克/T恤
  - 胸挂（Vest）：战术背心
  - 防弹（BodyArmor）：防弹衣（未来）
  - 裤子（Pants）：战术裤/牛仔裤
  - 腰带（Belt）：战术腰封
  - 背包（Backpack）：军用背包
- 交互方式：
  - **拖拽物品到装备槽** → 穿上装备
  - **双击装备槽** → 卸下装备（丢到地上）

### Zone 2（中）：容器网格
- 显示已装备物品的容器格子
  - 左侧列：上衣口袋（Tops container）、裤子口袋（Pants container）
  - 右侧列：腰带挂点（Belt container）、胸挂口袋（Vest container）
- 容器只在对应装备穿上时显示
  - 未装备上衣 → 上衣区域不显示网格
  - 穿上战术夹克 → 显示 2×4 网格
- 目前全部通过代码手动绘制（无 UI Toolkit/UGUI Prefab）
- 预留 C5 区域（未来功能）

### Zone 3（右）：背包 + 属性
- 背包容器网格（Backpack container）
- 下方显示人物属性：
  - 护甲总值
  - 保暖值
  - 负重（当前/最大）
- 背包为单独的装备槽位，不同于其他容器

## 三、交互逻辑

### 拖拽（Drag & Drop）
- 所有网格内物品可拖拽
- 拖拽到装备槽 → 穿上装备
- 拖拽到容器网格 → 移动物品到该容器
- 拖拽到背包网格 → 放入背包
- 拖出面板/右键点击 → 丢到地上
- 拖拽时物品图标跟随鼠标
- 拖拽过程中禁止滚动面板

### 双击
- 装备槽双击 → 卸下装备（实例化 WorldItem 丢到玩家面前地上）
- 容器/背包中的物品双击 → 尝试使用（消耗品自动使用，装备自动穿戴）

### 滚动
- 打开面板时用鼠标滚轮可滚动
- 内容超出面板高度时出现滚动区域

## 四、技术实现

### 纯代码绘制 UI
- 全部用 Unity uGUI (Text/Image/RectTransform) 通过 C# 代码生成
- 没有 Prefab/UI Document，运行时动态创建
- 每个格子（ItemCell）是一个 RectTransform + Image（背景）+ 可能有的 Text（数量）

### ViewData 模式
- InventoryUI 不直接引用 Inventory 实例
- 通过 InventoryViewData（快照结构体）读取数据
- 每次面板刷新时重新构建 ViewData → 绘制 UI
- 解耦 UI 和业务逻辑

### 关键类
```
InventoryUI.cs          主面板：绘制三区 + 处理输入
DragDropManager.cs      拖拽状态管理
ItemDragHandler.cs      拖拽图标跟随鼠标
QuickItemBar.cs         快捷栏（5格，数字12345）
SurvivalHUD.cs          生存数值HUD（场景中常显）
```

### Canvas 层级
```
- 主Canvas：InventoryUI 总面板
  - DragLayer（最高层）：拖拽中的物品图标
  - InventoryPanel（中层）：三区布局
  - 各容器网格（下层）：格子背景
  - 装备槽（与容器同层）
```

## 五、快速物品栏（QuickItemBar）

- 在场景中常显（非面板内）
- 5 个格子，数字键 1~5 选中
- 每个物品占 1 格
- 选中后按左键使用（吃东西/喝东西/投掷）
- 基础 5 格，未来可扩容
- 支持拖拽物品放入快捷栏

## 六、地图生成 UI（独立）

地图生成有自己的编辑器窗口，不在这里：
```
Tools/WorldGen/Preview
```
不在游戏内总面板中展示。
