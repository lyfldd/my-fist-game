# 战斗系统

> **状态**: 已实现 · **更新日期**: 2026-06-22 · **依赖**: L2 装备系统 + L2 人物系统(SurvivalXPSystem/StaminaSystem) + L1 EventBus

## 一、功能概述

近战+远程双模式战斗系统，支持武器切换、属性/技能联动伤害公式、散射管理、弹药系统、辅助瞄准。通过 `IDamageable` 接口统一伤害管道，僵尸AI状态机驱动攻击行为。

## 三、IDamageable 接口

```csharp
// _Game/Systems/Combat/IDamageable.cs
public interface IDamageable
{
    void TakeDamage(float damage);
    bool IsDead { get; }
}
```

设计理念：任何能受伤的东西实现此接口 —— 玩家 / 僵尸 / 门 / 车窗 / 建筑等。攻击系统只依赖接口，不关心具体实现。

**实现类：**

| 类 | 说明 |
|---|---|
| `DamageablePlayer` | 玩家受伤，转发 `PlayerDamaged` 事件给 SurvivalSystem；支持 `Invincible` 无敌 |
| `DamageableZombie` | 僵尸受伤（扣血+闪红+死亡+发布事件+销毁） |
| `AIBot` (AIBotCombat) | AI机器人友军，近战攻击跳过友军 |
| `PlacedStructure` | 建筑可被攻击破坏 |

## 四、玩家近战攻击 — PlayerCombat

**文件：** `_Game/Systems/Combat/PlayerCombat.cs`

### 输入绑定
- 左键（Mouse0）绑定到 `InputRouter.BindMouse(0, InputPriority.Gameplay, TryMeleeAttack, this)`
- `WeaponSwitcher` 控制 `enabled`：装备枪械时禁用，近战/空手时启用

### 攻击参数（空手 vs 近战武器）

| 参数 | 空手默认 | 近战武器（取自 ItemData） |
|---|---|---|
| range | `MELEE_ATTACK_RANGE` = 2m | `weapon.weaponRange` |
| radius | `MELEE_ATTACK_RADIUS` = 0.5m | `emptyHandRadius × 1.2` |
| damage | `MELEE_ATTACK_DAMAGE` = 10 | `weapon.weaponDamage` |
| cooldown | `MELEE_ATTACK_COOLDOWN` = 0.5s | `weapon.fireRate` |

### 伤害公式

```
finalDamage = baseDamage × (1 + Strength × 0.05 + meleeLevel × 0.08)
```
- `Strength`: 角色力量属性（默认5）
- `meleeLevel`: 近战专精技能等级（`SkillType.近战专精`）
- 常量: `STRENGTH_DAMAGE_MULT = 0.05`, `MELEE_SKILL_DAMAGE_MULT = 0.08`

### 攻击流程

1. **冷却检查** — `_cooldownTimer > 0` 则跳过
2. **体力检查** — `StaminaSystem.CanPerform(STAMINA_DRAIN_MELEE = 15f)` 不足则跳过
3. **球体扫描** — `Physics.OverlapSphere(origin + forward * range * 0.5, radius)` 在角色前方扫掠
4. **友军过滤** — 跳过含 `AIBot` 组件的实体
5. **伤害施加** — 命中第一个非友军 `IDamageable` → `TakeDamage(finalDamage)`
6. **事件发布:**
   - `ThreatReportEvent(sourceId, targetId, finalDamage)` → 威胁系统
   - `SurvivalXpGained(XP_DODGE_SUCCESS=8, "melee_hit")` → 生存经验
7. **音效:**
   - 命中: `SoundEmitter.EmitMeleeHit(hitPoint)` (15m范围)
   - 未命中: `SoundEmitter.EmitMeleeSwing(hitPoint)` (5m范围)

### 近战起点
- `origin = transform.position + Vector3.up * PLAYER_MELEE_ORIGIN_Y (0.8m)` — 胸部高度

## 五、远程射击 — WeaponShooting

**文件：** `_Game/Systems/Weapon/WeaponShooting.cs`

### 输入绑定
- 左键（Mouse0）→ `InputRouter.BindMouse(0, InputPriority.Gameplay, OnFireButton, this)`
- `WeaponSwitcher` 控制 `enabled`：装备枪械时启用，近战时禁用

### 自动/半自动判定
- `fireRate ≤ 0.25`（`semiAutoFireRateThreshold`）→ **全自动**（按住连射）
- `fireRate > 0.25` → **半自动**（每次点击一发，通过 `_wantsSemiFire` 标记实现）

### 弹药系统

- `_currentMag` 跟踪当前弹匣剩余子弹数
- `magazineSize = 0` → 无弹药限制（近战/特殊武器，不触发弹药检查）
- 每次射击 `_currentMag--`
- 弹匣归零 → 自动触发 `StartReload()`

### 换弹流程

```
弹匣归零
  → CountAmmoInInventory(ammoItemName) → 背包中搜索弹药物品
  ├─ 背包无弹药 → WeaponDryFireEvent（咔咔声）→ 不射击
  └─ 背包有弹药 → Inventory.RemoveItemByName(ammoItemName, take)
                → _currentMag += take
                → _isReloading = true, _reloadTimer = reloadTime
                → 计时结束 → FinishReload() → ItemReloadedEvent → UI刷新
```

### 散射系统

| 参数 | 说明 | ItemData 字段 |
|---|---|---|
| baseSpread | 基础散射角（度，锥形半角） | `baseSpread` (默认2°) |
| maxSpread | 最大散射角 | `maxSpread` (默认15°) |
| spreadPerShot | 每发射击散射增量 | `spreadPerShot` (默认1.5°) |
| spreadRecovery | 散射恢复速度（度/秒） | `spreadRecovery` (默认8°/s) |

- **射击时**: `_currentSpread = min(_currentSpread + spreadPerShot, maxSpread)`
- **停火时**: `_currentSpread = MoveTowards(baseSpread, spreadRecovery × deltaTime)`
- **技能减免**: `effectiveSpread = _currentSpread × (1 - gunLevel × 0.06)`
  - `gunLevel`: 枪械专精技能等级（`SkillType.枪械专精`）
  - `GUN_SKILL_SPREAD_REDUCTION = 0.06`

### 射线检测

- **起点** (`GetMuzzlePosition`): 优先 `WeaponHolder.HandWorldPos`，兜底为 `transform.position + forward*0.5m + up*1.45m`
- **方向**: `GetSpreadDirection(aimDir, effectiveSpread * 0.5)` — 在瞄准方向锥形内随机采样
- **射程**: `weapon.weaponRange`
- **命中检测**: 射线命中 `IDamageable` → `TakeDamage(weaponDamage)`；排除 AIBot 友军

### 散射采样算法（Rodrigues 旋转公式）

```
1. 构建瞄准方向的正交基:
     perpAxis = Cross(Vector3.up, aimDir)
     upAxis = Cross(aimDir, perpAxis)

2. 在锥形截面内随机采样（极坐标 → 笛卡尔）:
     angle = Random(0, spreadHalfAngle × Deg2Rad)
     rotation = Random(0, 360°)

3. Rodrigues 旋转:
     offset = (perpAxis * cos(rotation) + upAxis * sin(rotation)) * sin(angle)
     return (aimDir * cos(angle) + offset).normalized
```

### 射击事件

每发子弹发布 `WeaponFiredEvent`，携带完整数据：
- 武器 ItemData、弹药类型、枪声音色（Pistol/Rifle/Shotgun/Heavy）
- 枪口火焰类型（Small/Medium/Large）、枪口世界坐标
- 射击方向、是否命中、命中目标 GameObject

### 存档接口
- `GetCurrentMag(EquipSlot slot)` → 获取指定槽位当前弹夹子弹数
- `SetCurrentMag(EquipSlot slot, int ammo)` → 恢复指定槽位弹夹子弹数

## 六、武器瞄准 — WeaponAiming

**文件：** `_Game/Systems/Weapon/WeaponAiming.cs`

### 瞄准模式
- 右键（Mouse1）切换 `IsAimingDownSights`（**仅装备枪械时可用**）
- 切走枪械时强制退出瞄准
- `IsAimingDownSights` 供散布显示、移动减速等系统读取

### 辅助瞄准（Aim Assist）

```
射线检测僵尸层
  → 命中 IDamageable
  → 目标距离 ≤ AIM_ASSIST_RADIUS (2m)
  → 吸附锁定目标胸部位置 (zombiePos + up * PLAYER_CHEST_HEIGHT=1.2m)
  → IsAssisting = true, AssistTarget = zombieTransform
```

### 普通瞄准
- 无辅助时 → 读取 `MouseGroundProjector.GroundPoint`（鼠标地面投影）
- 兜底: `transform.position + forward * AIM_FALLBACK_DISTANCE (10m)`

### 输出属性

| 属性 | 类型 | 用途 |
|---|---|---|
| `AimTarget` | Vector3 | 瞄准目标世界坐标 |
| `AimDirection` | Vector3 | 枪口→目标的归一化方向 |
| `IsAimingDownSights` | bool | 是否右键瞄准 |
| `IsAssisting` | bool | 辅助瞄准是否激活 |
| `AssistTarget` | Transform | 辅助瞄准锁定目标 |

### 同步
- `AimDirection` 每帧同步到 `WeaponHolder.AimDirection`（控制模型朝向）

## 八、武器切换 — WeaponSwitcher

**文件：** `_Game/Systems/Weapon/WeaponSwitcher.cs`

- **Q键** 循环切换武器槽: `RightHand → LeftHand → KnifeBelt → SidearmBelt`
- 从 `Inventory.equipped` 字典查找武器
- 自动模式切换:
  - `weapon.isFirearm == true` → `WeaponShooting.enabled = true`, `PlayerCombat.enabled = false`
  - `weapon.isFirearm == false` → 反之
- 装备新武器时自动切换到该槽
- 当前武器被卸下时自动切到下一个有武器的槽（否则空手 `EquipSlot.None`）

## 九、散布可视化 — SpreadVisualizer

**文件：** `_Game/Systems/Weapon/SpreadVisualizer.cs`

- 使用两个 `LineRenderer`（Unlit材质，无光照）
- `_lrCone`: 锥形边界 — 4条母线（右/上/左/下）+ 远端截面圆（30段弧）
- `_lrCenter`: 红色轴线，从胸部Y=1.3到远端瞄准点
- 仅在右键瞄准 + 散布 > 0.01° 时显示
- 颜色阈值:
  - 散布 ≤ 3° → 绿色
  - 散布 ≤ 8° → 黄色
  - 散布 > 8° → 红色
- 3D锥形使用真实瞄准方向（含俯角），非展平

## 十、僵尸受伤与死亡 — DamageableZombie

**文件：** `_Game/Systems/Combat/DamageableZombie.cs`

### 受伤流程
1. `TakeDamage(damage)` → `_currentHealth -= damage`
2. `FlashRed()` → 协程将材质颜色变红 `ZOMBIE_FLASH_DURATION (0.15s)` 后恢复
3. `_currentHealth <= 0` → `Die()`

### 死亡流程
1. `IsDead = true`
2. 发布 `EntityDeathEvent(instanceID)` → ThreatSystem 清理威胁记录
3. 调用 `ZombieStateMachine.Die()` → AI 状态机进入 Dead 状态
4. 发布 `ZombieDied(position, lootGroup)` → 僵尸死亡事件
5. 发布 `SurvivalXpGained(XP_KILL_NORMAL_ZOMBIE=25, "zombie_kill")` → 生存经验
6. `Destroy(gameObject, ZOMBIE_DESTROY_DELAY=1f)` → 延迟1秒销毁

### 默认血量
- `maxHealth = ZOMBIE_MAX_HEALTH = 100`（普通僵尸；快速僵尸50，胖僵尸300由 ZombieData 覆盖）

## 十二、武器数据 — ItemData

**文件：** `_Game/Config/ItemData.cs`（ScriptableObject，菜单路径: Create → Game/Item）

### 武器专属字段

| 字段 | 类型 | 说明 | 默认值 |
|---|---|---|---|
| `isFirearm` | bool | true=枪械(远程) / false=近战 | false |
| `weaponDamage` | float | 单发/单次伤害 | 10 |
| `weaponRange` | float | 最大射程/攻击距离(米) | 50 |
| `fireRate` | float | 射击间隔(秒/发)，近战为冷却时间 | 0.5 |
| `baseSpread` | float | 基础散射角(度，锥形半角) | 2 |
| `maxSpread` | float | 最大散射角(度) | 15 |
| `spreadRecovery` | float | 不射击时散射恢复速度(度/秒) | 8 |
| `spreadPerShot` | float | 每发射击散射增量(度) | 1.5 |
| `magazineSize` | int | 弹匣容量（0=无限弹药/近战） | 0 |
| `ammoItemName` | string | 消耗的弹药物品名（如 "Ammo_9mm"），空=无需弹药 | "" |
| `gunshotSoundType` | string | 枪声音色: Pistol / Rifle / Shotgun / Heavy | "" |
| `muzzleFlashType` | string | 枪口火焰: Small / Medium / Large | "" |
| `reloadTime` | float | 换弹耗时(秒) | 1.5 |
| `equipSlot` | EquipSlot | 装备槽位（RightHand/LeftHand/KnifeBelt/SidearmBelt） | None |

### 弹药匹配机制
- `ammoItemName` 与背包中同名物品匹配
- 换弹时从背包按名称扣减（`Inventory.RemoveItemByName(ammoItemName, take)`）
- 空字符串或 null → 无需弹药（近战/电磁炮等），不触发弹药检查和换弹

## 十三、伤害流程总览

```
输入（左键鼠标0）
    │
    ├─ WeaponSwitcher 判断武器类型
    │   ├─ isFirearm → WeaponShooting.enabled=true, PlayerCombat.enabled=false
    │   └─ !isFirearm → PlayerCombat.enabled=true, WeaponShooting.enabled=false
    │
    ├─ [近战路径] PlayerCombat.TryMeleeAttack()
    │   ├─ 冷却检查 (_cooldownTimer)
    │   ├─ 体力检查 (StaminaSystem.CanPerform)
    │   ├─ 获取武器数据（范围/伤害/冷却）
    │   ├─ 伤害公式: baseDamage × (1 + STR×0.05 + MeleeLv×0.08)
    │   ├─ Physics.OverlapSphere 球体扫描
    │   ├─ 跳过 AIBot 友军
    │   ├─ IDamageable.TakeDamage(finalDamage)
    │   ├─ ThreatReportEvent ──→ ThreatSystem
    │   ├─ SurvivalXpGained ──→ SurvivalXPSystem
    │   ├─ EmitMeleeHit/EmitMeleeSwing ──→ 音效
    │   └─ 只命中第一个有效目标后 break
    │
    └─ [远程路径] WeaponShooting.TryFire()
        ├─ 弹匣归零 → 自动换弹 (或 WeaponDryFireEvent)
        ├─ 散射更新: _currentSpread += spreadPerShot
        ├─ 技能减免: effectiveSpread = spread × (1 - GunLv×0.06)
        ├─ Rodrigues 旋转公式生成随机散射方向
        ├─ Physics.Raycast(枪口, 散射方向, weaponRange)
        ├─ 跳过 AIBot 友军
        ├─ IDamageable.TakeDamage(weapon.weaponDamage)
        ├─ ThreatReportEvent ──→ ThreatSystem
        ├─ WeaponFiredEvent ──→ SoundEmitter/MuzzleFlashSystem/NoiseSystem
        └─ Debug.DrawRay 可视化（红=命中，黄=未命中）

右键 Mouse1 → WeaponAiming
    ├─ IsAimingDownSights 切换
    ├─ 射线检测僵尸 → 辅助瞄准 (距离≤2m时吸附)
    ├─ 无辅助 → MouseGroundProjector 地面投影
    └─ AimDirection 同步到 WeaponHolder
```
