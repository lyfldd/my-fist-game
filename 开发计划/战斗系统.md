## 一、功能概述

近战+远程双模式战斗系统，支持武器切换、属性/技能联动伤害公式、散射管理、辅助瞄准、僵尸AI状态机驱动攻击。通过 IDamageable 接口统一伤害处理。

## 四、伤害公式

```
近战最终伤害 = baseDamage × (1 + Strength × 0.05 + 近战专精等级 × 0.08)

远程伤害 = weapon.weaponDamage  (固定值，无属性加成)

有效散射角 = currentSpread × (1 − 枪械专精等级 × 0.06)
```

## 六、战斗流程

### 近战攻击

```
鼠标左键 → PlayerCombat.Update()
  → 冷却检查 + 体力检查(≥15)
  → 获取 ActiveWeapon → 计算范围/伤害
  → finalDamage × (1 + STR×0.05 + 近战专精×0.08)
  → Physics.OverlapSphere(锥形区域)
  → 命中: IDamageable.TakeDamage + SoundEmitter.EmitMeleeHit + SurvivalXpGained
  → 挥空: SoundEmitter.EmitMeleeSwing
```

### 远程射击

```
鼠标/按住 → WeaponShooting.Update()
  → 获取 ActiveWeapon(isFirearm=true)
  → 冷却检查 + 散射管理(add per shot → cap → recover)
  → 有效散射 × (1 − 枪械专精×0.06)
  → Rodrigues旋转公式生成锥形随机方向
  → Physics.Raycast(枪口, 散射方向, range)
  → 命中: IDamageable.TakeDamage + WeaponFiredEvent + EmitGunshot
  → 弹道穿透 DecibelSystem → 僵尸听觉联动
```

### 僵尸攻击玩家

```
ZombieAwarenessSystem(每0.5s)
  → 视觉锥(angle+range) + Linecast视线遮蔽
  → 发现 → 级联唤醒15m内僵尸
  → Idle/Wander → Chase → Attack(贴脸)
  → ZombieState_Attack: 每cooldown秒对playerTarget.TakeDamage(attackDamage)
  → DamageablePlayer: 发布PlayerDamaged → 生存系统处理
```
