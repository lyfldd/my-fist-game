## 一、功能概述

中央时间管理器，驱动游戏世界的时间推进（日/夜循环）、Directional Light 旋转与环境光切换、天数追踪、时间相关事件广播（每小时/每天）。同时作为多个下游系统的时钟源：生存系统Tick间隔、容器刷新冷却、僵尸夜间倍率。

## 四、日夜循环

```
太阳仰角: -90°(午夜) → 0°(日出/日落) → +90°(正午) → -90°
根据小时数计算角度，太阳光绕X轴旋转

太阳强度: 0.3(日出) → 1.2(正午) → 0.3(日落)
阴影强度: 0.3~1.0
环境光: 0.08(深夜) → 1.0(正午)

四种时段颜色插值:
  dayColor → sunsetColor → nightColor → dawnColor → dayColor
  (日出5:00 / 日落17:00 / 夜晚19:00 / 黎明5:00)
```

**月亮**: 与太阳对面，强度0.05~0.35，色调偏蓝
