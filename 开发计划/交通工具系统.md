# 交通工具系统 (Vehicle System)

## 概览

交通工具系统是一个基于 Unity **Rigidbody + WheelCollider** 物理的车辆驾驶模块。玩家可以靠近车辆按 `E` 上车，WASD 控制油门/转向，Space 手刹，Left Shift 加速档，再按 `E` 下车。系统通过事件总线实现各子系统解耦。

### 核心特性
- **真实物理**：WheelCollider ×4，前后轮分工（前轮转向+驱动，后轮仅驱动）
- **燃料系统**：消耗背包中的燃料物品（如 `SyntheticGasoline`），无燃料自动熄火
- **SHIFT 加速**：按下时提高极速上限，消耗更多燃料
- **防侧翻**：反侧倾杆（Anti-Roll Bar）+ 角速度阻尼（Roll Stabilization）
- **摄像机切换**：上车自动拉远摄像机，下车恢复
- **输入锁定**：驾驶时禁用角色移动/战斗/交互，重映射 WASD/E 到车辆控制
- **后备箱**：配置层面支持（`trunkWidth`/`trunkHeight`），逻辑层待实现

## 数据层：VehicleData (ScriptableObject)

`VehicleData` 是一种 ScriptableObject，每种车型对应一个 `.asset` 文件，存放在 `Assets/_Game/Config/VehicleData/`。

### 字段详解

#### 基本信息
| 字段 | 类型 | 说明 |
|------|------|------|
| `vehicleName` | `string` | 车辆显示名称 |
| `category` | `VehicleCategory` | 车种类别枚举 |

#### VehicleCategory 枚举
| 值 | 含义 |
|----|------|
| `Car` | 普通汽车（4座 + 后备箱） |
| `Truck` | 卡车（2座 + 大货箱） |
| `Motorcycle` | 摩托车（1~2座，无后备箱） |
| `Bicycle` | 自行车（1座，纯人力） |

#### 驾驶参数
| 字段 | 默认值 | 说明 |
|------|--------|------|
| `maxSpeed` | 15 m/s (~54 km/h) | 最高速度（单位：米/秒） |
| `motorTorque` | 800 | 电机扭矩，值越大加速越快 |
| `brakingForce` | 3000 | 制动力 |
| `maxSteerAngle` | 35度 | 最大转向角（度） |
| `reverseTorque` | 400 | 倒车扭矩 |

#### 物理参数
| 字段 | 默认值 | 说明 |
|------|--------|------|
| `mass` | 1200 kg | 车身质量（Rigidbody.mass） |
| `suspensionDamper` | 8000 | 车轮悬架阻尼 |

#### 加速/极速（SHIFT Boost）
| 字段 | 默认值 | 说明 |
|------|--------|------|
| `boostSpeedMultiplier` | 2.0 | SHIFT 按下时有效最高速度 = maxSpeed * 此值 |
| `boostFuelMultiplier` | 1.5 | SHIFT 按下时油耗倍率 |

#### 稳定性
| 字段 | 默认值 | 说明 |
|------|--------|------|
| `antiRollStiffness` | 5000 | 反侧倾杆刚度，防止转弯侧翻 |
| `centerOfMassYOffset` | -0.3 | 重心 Y 偏移，负值降低重心增加稳定性 |

#### 油量
| 字段 | 默认值 | 说明 |
|------|--------|------|
| `fuelCapacity` | 40 L | 油箱容量（升） |
| `fuelConsumptionRate` | 0.02 L/s | 每秒油耗（油门踩下时自动消耗） |

#### 后备箱（配置层，逻辑待实现）
| 字段 | 默认值 | 说明 |
|------|--------|------|
| `trunkWidth` | 4 | 后备箱格子列数 |
| `trunkHeight` | 3 | 后备箱格子行数 |

#### 座位
| 字段 | 默认值 | 说明 |
|------|--------|------|
| `driverSeatOffset` | (-0.5, 0.5, 0) | 驾驶员座位相对车身偏移 |
| `exitOffset` | (-2, 0, 0) | 下车点相对车身偏移 |

#### 模型碰撞体
| 字段 | 默认值 | 说明 |
|------|--------|------|
| `colliderCenter` | (0, 0.8, 0) | BoxCollider 中心偏移 |

## 物理层：VehicleController

挂载在车辆根 GameObject 上，`[RequireComponent(typeof(Rigidbody))]`。

### WheelCollider 布局

```
            前轮(转向+驱动)
        [FL] ---------- [FR]
         |                |
         |   车身(Body)    |
         |                |
        [RL] ---------- [RR]
            后轮(仅驱动)
```

- **前轮**（FL, FR）：接收 `steerAngle` + `motorTorque` + `brakeTorque`
- **后轮**（RL, RR）：仅接收 `motorTorque` + `brakeTorque`

### 核心方法

#### 公共接口（由 VehicleInputLock 调用）
| 方法 | 参数 | 说明 |
|------|------|------|
| `SetThrottle(float)` | -1~1 | 油门（正=前进，负=倒车，0=空档） |
| `SetSteer(float)` | -1~1 | 转向（负=左转，正=右转） |
| `SetBrake(float)` | 0~1 | 刹车 |
| `SetBoost(bool)` | - | 切换 SHIFT 加速模式 |
| `SetDriver(GameObject)` | - | 设置/清除驾驶员 |
| `StopImmediate()` | - | 急刹：归零所有输入 + 物理速度 |

#### 运行时属性
| 属性 | 类型 | 说明 |
|------|------|------|
| `Driver` | `GameObject` | 当前驾驶员（null = 空车） |
| `HasDriver` | `bool` | 是否有驾驶员 |
| `CurrentFuel` | `float` | 当前油量（L） |
| `CurrentSpeedKmh` | `float` | 当前速度（km/h），由 velocity.magnitude * 3.6 计算 |
| `IsBoosting` | `bool` | 是否处于加速模式 |

### Physics Update (FixedUpdate)

每一物理帧，当 `HasDriver && _isEngineOn` 时依次执行：

1. **ApplyWheelPhysics()** — 将油门/转向/刹车写入 4 个 WheelCollider
   - 前进：`motorTorque = throttle * _motorTorque`
   - 倒车：`motorTorque = throttle * _reverseTorque`（注意 `_reverseTorque` 独立配置）
   - 刹车优先：刹车 > 0.01 时清零 motorTorque
   - 油耗：油门 > 0 时按 `fuelConsumptionRate`（加速时 x `boostFuelMultiplier`）扣减
   - **自动加油**：油量归零时尝试从驾驶员背包中移除 1 个燃料物品（如 `SyntheticGasoline`），补充 20L 燃料

2. **ClampSpeed()** — 限制速度上限
   - 正常：`_maxSpeed`（来自 VehicleData.maxSpeed）
   - SHIFT 加速：`_maxSpeed * boostSpeedMultiplier`
   - 超出限速时强制归一化速度向量

3. **ApplyAntiRoll()** — 反侧倾杆
   - 原理：检测左右轮悬挂压缩差（WheelHit.travel），向压缩多的一侧施加上推力
   - 刚度由 `VehicleData.antiRollStiffness` 控制（默认 5000）
   - 前后轴分别独立计算

4. **ApplyRollStabilization()** — 角速度阻尼
   - 将世界空间角速度转到本地空间
   - X 轴（前翻/后翻）：x0.95（每帧衰减 5%）
   - Z 轴（侧翻）：x0.92（每帧衰减 8%）
   - Y 轴（转向）：不限制（x1.0）
   - 再转回世界空间写回 rb.angularVelocity

### 配置加载 (ApplyConfig)

- 有 `VehicleData` 时从 ScriptableObject 读取所有参数
- 无 `VehicleData` 时使用 `GameConstants` 默认值（`VEHICLE_DEFAULT_*`）
- 一次性设置 4 个 WheelCollider 的悬架阻尼
- 降低重心：`rb.centerOfMass.y += centerOfMassYOffset`（默认 -0.3m）

### 引擎声音
- 上车时调用 `SoundEmitter.StartVehicleEngine(key, pos)`，通过 `DecibelSystem` 持续发出 30m 半径的机械声
- 每帧油门非零时调用 `SoundEmitter.UpdateVehicleEngine(key, pos)` 更新位置
- 下车/熄火时调用 `SoundEmitter.StopVehicleEngine(key)` 停止
- Key 格式：`vehicle_engine_{GetInstanceID()}`，保证每辆车独立

## 交互层：VehicleInteraction

挂载在车辆根 GameObject 上，`[RequireComponent(typeof(VehicleController))]`，实现 `IInteractable` 接口。

### IInteractable 实现

| 成员 | 值 | 说明 |
|------|-----|------|
| `InteractionPrompt` | `"上车 [E]"` / `"下车 [E]"` | 根据是否有驾驶员动态切换 |
| `InteractionTime` | `0f` | 瞬间完成 |
| `IsInteractable` | `!HasDriver` | 仅无驾驶员时可交互 |
| `OnInteract(GameObject)` | 调用 `Enter(interactor)` | 仅当无驾驶员时 |

### 上车流程 (Enter)
1. 保存驾驶员原始 Rigidbody -> 设为 `isKinematic = true`，速度归零
2. 保存驾驶员原始 Collider -> 禁用
3. 将驾驶员移动到座位位置：`transform.position + rotation * (VehicleData.driverSeatOffset)`
4. 将驾驶员设为车辆子物体：`driver.transform.SetParent(transform)`
5. 调用 `_controller.SetDriver(driver)` -> 启动引擎
6. 发布 `VehicleEnteredEvent(gameObject, driver)`

### 下车流程 (Exit)
1. 恢复驾驶员 Rigidbody：`isKinematic = false`
2. 恢复驾驶员 Collider：`enabled = true`
3. 解除父子关系：`driver.transform.SetParent(null)`
4. 放置到下车位置：`transform.position + rotation * (VehicleData.exitOffset)`，Y 保持地面高度
5. 调用 `_controller.SetDriver(null)` -> 熄火
6. 发布 `VehicleExitedEvent(gameObject, driver)`

### Gizmo 可视化
- 绿色球：驾驶员座位位置
- 黄色球：下车点
- 黄线连接两球

## 输入层：VehicleInputLock

挂载在玩家 GameObject 上（与 `PlayerController` / `BuildModeInputLock` 同物体）。

### 事件驱动
- `OnEnable` 时订阅 `VehicleEnteredEvent` 和 `VehicleExitedEvent`
- 收到 `VehicleEnteredEvent`（且司机是自己）-> `BlockInput()`
- 收到 `VehicleExitedEvent`（且司机是自己）-> `RestoreInput()`

### 被禁用的组件
| 组件 | 效果 |
|------|------|
| `PlayerController` | 禁用 WASD 角色移动 |
| `WeaponAiming` | 禁用鼠标瞄准 |
| `WeaponShooting` | 禁用鼠标左键射击 |
| `PlayerCombat` | 禁用战斗输入 |
| `PlayerInteraction` | 禁用 E 键交互检测 |

### 键盘重映射（Update 中每帧执行）

| 键位 | 重映射为 | 说明 |
|------|----------|------|
| `W` | 油门 = 1.0 | 前进 |
| `S` | 油门 = -1.0 | 倒车 |
| `A` | 转向 = -1.0 | 左转 |
| `D` | 转向 = 1.0 | 右转 |
| `Space` | 刹车 = 1.0 | 手刹 |
| `Left Shift` | Boost = true/false | 加速档 |
| `E` | 下车（InputRouter 优先级 Action+5） | 调用 VehicleInteraction.Exit() |

### 保留的输入
- 摄像机跟随（CameraFollow 通过事件切换目标）
- 鼠标移动

## 摄像机切换：CameraFollow

`CameraFollow` 订阅 `VehicleEnteredEvent` / `VehicleExitedEvent`：

- **上车时**：
  - `target` 切换到车辆 Transform
  - `_currentDistance = defaultDistance + VEHICLE_CAMERA_EXTRA_DISTANCE`（默认 +5m）
  - `_isFollowingVehicle = true`

- **下车时**：
  - `target` 恢复到 `_defaultTarget`（玩家 Transform）
  - `_currentDistance` 恢复默认值
  - `_isFollowingVehicle = false`

> `VEHICLE_CAMERA_EXTRA_DISTANCE` 常量定义在 `GameConstants.cs`，值 = 5f。

### VehicleEnteredEvent
```csharp
public readonly struct VehicleEnteredEvent
{
    public GameObject Vehicle { get; }  // 车辆 GameObject
    public GameObject Driver { get; }   // 驾驶员 GameObject
}
```

### VehicleExitedEvent
```csharp
public readonly struct VehicleExitedEvent
{
    public GameObject Vehicle { get; }  // 车辆 GameObject
    public GameObject Driver { get; }   // 驾驶员 GameObject
}
```

### 事件发布/订阅关系

```
VehicleInteraction (发布者)
    |
    +-- Enter() -> EventBus.Publish(VehicleEnteredEvent)
    |       +-- VehicleInputLock.OnVehicleEntered()    -> 阻塞玩家输入
    |       +-- CameraFollow.OnVehicleEntered()        -> 切换摄像机目标
    |
    +-- Exit()  -> EventBus.Publish(VehicleExitedEvent)
            +-- VehicleInputLock.OnVehicleExited()     -> 恢复玩家输入
            +-- CameraFollow.OnVehicleExited()         -> 恢复摄像机目标
```

## 编辑器工具：CreateVehicleData

路径：`Assets/_Game/Editor/CreateVehicleData.cs`

提供 3 个 Unity 菜单项：

| 菜单项 | 功能 |
|--------|------|
| `Tools/Vehicles/Create OffRoad Vehicle Data` | 创建 OffRoad 越野车的 VehicleData .asset |
| `Tools/Vehicles/Setup Vehicle Prefab (from selected)` | 基于选中的 3D 模型自动搭建车辆 Prefab |
| `Tools/Vehicles/Quick Setup OffRoad (Data + Prefab)` | 一键创建 Data + 搭建 Prefab |

### Setup Vehicle Prefab 自动化流程
1. 计算模型包围盒（遍历所有 MeshFilter + SkinnedMeshRenderer）
2. 自动检测车头朝向（XZ 平面上更长轴 = 车头）
3. 按目标车长 4.5m 自动缩放根物体
4. 计算 4 个 WheelCollider 的世界/本地位置和半径（基于包围盒比例）
5. 创建 Rigidbody（mass 1800, drag 0.3, angularDrag 1）
6. 创建 BoxCollider（包围盒 x0.95）
7. 创建 4 个 WheelCollider 子物体（mass 20, spring 35000, damper 4500, 摩擦力曲线预设）
8. 添加 VehicleController + VehicleInteraction 组件
9. 自动更新 OffRoad.asset 中的 colliderCenter/size/driverSeatOffset/exitOffset
10. 保存 Prefab

## 燃料系统详解

### 燃料物品约定
- 默认燃料物品名：`"SyntheticGasoline"`（在 `VehicleController.fuelItemName` 中配置）
- 1 个燃料物品 = 20L 燃料（硬编码在 `ApplyWheelPhysics()` 中）

### 油耗逻辑
- 油耗仅在油门踩下时发生（`_currentThrottle != 0`）
- 基础油耗速率 = `VehicleData.fuelConsumptionRate`（L/s）
- SHIFT 加速时 = `fuelConsumptionRate x boostFuelMultiplier`（默认 x1.5）
- 消耗：`CurrentFuel -= fuelRate * Time.fixedDeltaTime`

### 自动加油（燃油耗尽时）
1. 检查 `fuelItemName` 是否为空 -- 为空则直接熄火
2. 检查驾驶员背包中是否有该名称物品（`_driverInventory.CountItemByName(fuelItemName)`）
3. 如果 > 0：
   - 从背包移除 1 个燃料物品
   - 补充 20L 燃料
   - 引擎保持开启
4. 如果没有：
   - `_isEngineOn = false`（熄火）
   - 停止引擎声音
   - 车辆无法再操作，需下车并手动加油

2. **多乘客**：副驾和后排座位支持（尤其是 Car/Truck 类别）
3. **车辆损坏**：HP 系统 + 碰撞伤害 + 零件掉落
4. **油耗因子**：载重影响油耗，越野地形增加油耗
5. **车辆种类**：摩托车（1人，无后备箱，高速低防护）、自行车（纯人力，消耗体力）
6. **修理系统**：使用零件修理车辆（轮胎/引擎/车身），调用 `XP_REPAIR_VEHICLE`
7. **Key 点火**：需要钥匙/开锁技能才能发动（防盗 + 潜入玩法）
8. **档位系统**：前进/空档/倒车三档切换（当前是连续油门 -1~1）
