# 🔌 系统接入规范

> **日期**: 2026-06-24 · **状态**: 设计阶段
> **目标**: 新建物品只需填 ItemData 字段 + 配 RecipeData，所有系统自动识别接入。

---

## 一、发现机制 — 15 个系统

### 1.1 字段触发型（10 个）

| 系统 | 触发条件 | 接入动作 |
|------|------|------|
| **武器射击** | `category == Weapon && weaponDamage > 0` | WeaponShooting 读 `damage/fireRate/spread/magazineSize/ammoItemName/reloadTime` |
| **武器近战** | `category == Weapon && !isFirearm` | PlayerCombat 读 `weaponDamage` |
| **装备** | `equipSlot != EquipSlot.None` | Inventory.EquipItem 直接装到对应槽；`storageWidth/Height` 扩展容器网格 |
| **护甲** | `armorValue > 0` | SurvivalSystem.GetEffectiveArmor 自动累加 |
| **保暖** | `warmthValue > 0` | SurvivalSystem 自动累加保暖值 |
| **食物/药品** | `itemEffects.Count > 0` 或 `foodType != null` 或 `healAmount > 0` | SurvivalSystem.ApplyItemEffect 自动施加效果（健康/饥饿/口渴/体温/治疗） |
| **燃料** | `fuelValue > 0` | ProductionDevice/Vehicle 按物品名查找对应燃料 ItemData |
| **耐久** | `hasDurability == true && maxDurability > 0` | DurabilitySystem 追踪 instanceId，消耗/损坏事件自动发布 |
| **电子设备** | `electronicDeviceType != Flashlight`（非默认） | ElectronicDevice 读取 deviceType 应用运行时行为 |
| **武器附件** | `attachmentSlot != None` 或 `hostSlots.Length > 0` | WeaponAttachmentHandler 挂载/拆卸；低耐久衰减效果 |

### 1.2 引用触发型（3 个）

| 系统 | 触发条件 | 接入动作 |
|------|------|------|
| **合成** | `RecipeData.resultItem == this` | RecipeCatalog 按 workstation 索引；CraftingUI 显示可用配方 |
| **建造** | `BuildableData.materials[i].itemData == this` | BuildModeController 检查背包材料数量是否足够 |
| **生产设备** | `ProductionDeviceData.recipes[i].output == this` 或 `.input == this` | ProductionDevice.TryProduce 消耗输入→产出输出；输出槽满触发事件 |

### 1.3 配置型（1 个 · 需手动）

| 系统 | 触发条件 | 接入动作 |
|------|------|------|
| **AIBot 武器** | `AIBotWeaponMapping.asset` 中引用 | AIBotCombat 按右臂/左臂枚举选武器；弹药从背包自动消耗；耐久扣减 |

### 1.4 事件驱动型（3 个 · 自动传播）

| 系统 | 触发条件 | 接入动作 |
|------|------|------|
| **耐久 UI** | `DurabilityChangedEvent` | InventoryUI 装备槽 + QuickItemBar 刷新耐久条 |
| **物品详情 UI** | 右键 CellClickHandler | InventoryUI.ShowItemDetail 显示 ItemData + ItemGraphNode 信息 |
| **物品图谱** | `ItemGraphBuilder.Build()` 全量重建 | 自动计算上下游/深度/断头路/来源/动作/消费热度 |

## 二、新建物品操作清单

### 只需 3 步：

```
1. 建 ItemData.asset → 填字段（category/weaponDamage/equipSlot/ammoItemName...）
2. 建 RecipeData.asset → 设置所需材料+产物+工作台（如有）
3. 重建 ItemGraph → Tools/物品图谱/构建 ItemGraph
```

### 额外步骤（特殊物品）：

| 物品类型 | 额外步骤 |
|------|------|
| **建造物** | 建 `BuildableData.asset` + 配材料 + 配 prefab |
| **生产设备** | 建 `ProductionDeviceData.asset` + 配配方 + 建套件 ItemData + 建建造数据 |
| **AIBot 武器** | 在 `AIBotWeaponMapping.asset` 中注册 |
| **容器搜刮品** | 在 `ContainerLootProfile.asset` 中加权重 |
| **3D 模型武器** | 配 `worldPrefab` 引用（阶段 4 做） |

## 三、引用完整性检验

`ItemGraphBuilder.Build()` 末尾自动运行：

```csharp
// 扫全量引用
foreach (var recipe in RecipeCatalog.all)
{
    if (recipe.resultItem == null)      Log("❌ 配方产物缺失: " + recipe.name);
    foreach (var mat in recipe.materials)
        if (mat.itemData == null)        Log("❌ 配方材料缺失: " + mat.name);
}

foreach (var weapon in ItemData.all.Where(i => i.isFirearm))
    if (weapon.ammoItemName != null && FindItem(weapon.ammoItemName) == null)
        Log("❌ 武器弹药缺失: " + weapon.itemName + " → " + weapon.ammoItemName);

foreach (var buildable in BuildableCatalog.all)
    foreach (var mat in buildable.materials)
        if (mat.itemData == null)        Log("❌ 建造材料缺失: " + buildable.displayName);

foreach (var device in ProductionDevice.all)
    foreach (var recipe in device.recipes)
    {
        if (recipe.output == null)       Log("❌ 设备产出空: " + device.name);
        if (recipe.input == null)        Log("❌ 设备输入空: " + device.name);
    }
```

输出形如：
```
[ItemGraph] 引用完整性报告:
  ✅ 配方: 159/159 有效
  ❌ 武器弹药缺失: AK-47 → 7.62mm步枪弹 (弹药物品不存在)
  ❌ 建造材料缺失: 木墙 → 木板 (物品已删除)
  ✅ 生产设备: 31/31 有效
```

**新建物品后的验证命令**: `Tools → 物品图谱 → 构建并验证 ItemGraph`

## 四、AIBotWeaponMapping（新建 ScriptableObject）

```csharp
[CreateAssetMenu(menuName = "Game/AI Bot Weapon Mapping")]
public class AIBotWeaponMapping : ScriptableObject
{
    public Entry[] entries;

    [Serializable]
    public class Entry
    {
        public RightArmWeapon weapon;
        public ItemData weaponItem;      // 武器 ItemData 引用
        public ItemData ammoItem;        // 对应弹药 ItemData 引用
    }
}
```

**自动接入——AI Bot Combat 改读此表：**
```csharp
// 旧: 硬编码 AMMO_PISTOL = "手枪子弹"
// 新: mapping.GetAmmo(RightArmWeapon.Pistol) → 返回 ItemData

```
