## 一、功能概述

多容器网格背包系统，每个装备槽对应一个 `InventoryContainer`（网格+放置物品）。支持物品添加/移除/堆叠/旋转，拖放系统支持容器间移动和装备槽直接投放。T键旋转（3种上下文），坐标检测式命中测试。

## 二、数据层

### InventoryContainer

```csharp
// Systems/Inventory/InventoryContainer.cs
[Serializable]
public class InventoryContainer
{
    public string containerName;
    public EquipSlot equipSlot;
    public KeyCode toggleKey;
    public int gridWidth, gridHeight;
    public List<PlacedItem> placedItems;
}
```

**关键方法**：
- `AddItem(item, count, overloadWeight)` — 优先堆叠已有同类 → 不够则找空闲格创建新PlacedItem
- `RemoveItem(item, count)` — 从后向前匹配移除（先移除后面的）
- `RemoveItemAt(gridX, gridY, count)` — 精确坐标移除（防止同类物品混淆）
- `IsSpaceFree(x, y, w, h)` — AABB重叠检测
- `IsSpaceFreeFor(...excludeX, excludeY...)` — 排除特定区域（原地旋转用）
- `FindSpace(w, h, out x, out y)` — 逐行扫描找空位

### PlacedItem

```csharp
// Systems/Inventory/PlacedItem.cs
public struct PlacedItem
{
    public ItemData itemData;
    public int count;
    public int gridX, gridY;
    public bool rotated;             // 是否顺时针旋转90°
    public bool isGhost;             // 腰带武器占位格(itemData=null)
    public EquipSlot ghostSourceSlot; // 来源武器槽

    public int GridWidth  => rotated ? itemData.gridHeight : itemData.gridWidth;
    public int GridHeight => rotated ? itemData.gridWidth : itemData.gridHeight;
}
```

## 三、Inventory 组件

```csharp
// Systems/Inventory/Inventory.cs — MonoBehaviour, 挂玩家GameObject
public class Inventory : MonoBehaviour
{
    List<InventoryContainer> containers;          // 6个内置容器
    Dictionary<EquipSlot, ItemData> equipped;     // 当前装备
    float currentWeight;                          // 当前总负重
    float overloadWeight;                         // 超重阈值
}
```

**内置容器**（InitializeContainers, 默认全部1×1）：
| 容器 | 快捷键 | EquipSlot |
|------|:---:|------|
| 上衣口袋 | T | Tops |
| 裤子口袋 | P | Pants |
| 腰带 | — | Belt |
| 胸挂 | V | Vest |
| 背包 | B | Backpack |
| 头盔 | — | Head |
| 防弹衣 | — | BodyArmor |

**AddItem 添加优先级**: 裤子 → 腰带 → 胸挂 → 背包（上衣口袋被排除）

**背包级方法**:
- `AddItem(item, count)` — 按优先级尝试各容器 → 成功发InventoryChanged("added") / 失败发("slot_full")
- `RemoveItem(item, count)` — 遍历所有容器移除 → 发InventoryChanged("removed")
- `DropItem(item, count)` — RemoveItem + 生成WorldItem到地面上
- `HasItem(item, count)` — 检查是否持有足够数量

## 四、拖放系统 (DragDropManager)

```
ItemDragHandler(挂每个网格格子上)
  → IPointerDownHandler → DragDropManager.OnPointerDown
  → IPointerUpHandler   → DragDropManager.OnPointerUp

DragDropManager(单例):
  状态机: Idle → Pressed(阈值8px) → Dragging(跟随鼠标图标) → Released

  FindCellAtPosition(screenPos):
    反向遍历已注册 CellRegion → RectTransformUtility 判定落点

  TryMoveToContainer(cr):
    武器槽 → 自动装备
    同容器旋转 → IsSpaceFreeFor 排除自身
    不同容器 → IsSpaceFree
    放不下 → 回退原位 + Toast提示

  T键旋转 (3种上下文):
    1. 选中未拖时 → RotateInPlace (原地旋转)
    2. 拖拽中 → toggle _dragRotated
    3. 已选中时 → RotateSelectedItem
```

**CellRegion**: 记录每个格子的 RectTransform + 所属容器 + 网格坐标 + 是否武器槽，支持坐标→容器的反向查找。

## 五、事件

| 事件 | 何时发布 |
|------|---------|
| InventoryChanged(action, itemName, count) | 物品增/删/移动/旋转/超重/槽满 |
| InventoryViewChangedEvent(snapshot) | 背包数据快照变更 |
| EquipmentChangedEvent(totalArmor, totalWarmth) | 装备变化 |
| WeaponEquippedEvent / WeaponUnequippedEvent | 武器槽变更 |
