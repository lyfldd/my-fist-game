# 天气系统 (Weather System)

> **文件**: `Assets/_Game/Systems/Weather/WeatherManager.cs`  
> **类型**: `WeatherManager` (MonoBehaviour, ServiceLocator 单例)  
> **命名空间**: `_Game.Systems.Weather`  
> **UI**: `Assets/_Game/UI/WeatherHUD.cs`  
> **事件**: `WeatherChangedEvent`（定义于 `Core/GameEvents.cs`）  
> **ScriptableObject**: `Assets/_Game/Config/WeatherData.cs`（含 `WeatherType` 枚举）

---

## 概述

WeatherManager 使用马尔可夫链状态机驱动天气变化，每2游戏小时轮换一次。4种天气（晴/多云/雨/暴雨）各自携带光照参数和环境温度修正，影响下游生存系统。支持3档难度调节和 DevTools 强制覆盖。

## 二、WeatherData ScriptableObject

每个天气类型对应一个 `WeatherData` SO，在 `WeatherManager._weatherData[]` 数组中配置。

### 字段

| 字段 | 类型 | 说明 |
|------|------|------|
| `weatherType` | `WeatherType` | 枚举索引 |
| `displayName` | `string` | 中文显示名（"晴"/"多云"/"雨"/"暴雨"） |
| `rainIntensity` | `float` (0~1) | 雨强：Sunny=0, Cloudy=0, Rain=0.5, Storm=1.0 |
| `tempModifier` | `float` | 温度修正 |
| `thirstRateMult` | `float` (0.5~2) | 口渴速率倍率：晴天1.0, 雨天0.8, 暴雨0.6 |
| `ambientDarkness` | `float` (0~1) | 环境暗度：0=晴天, 0.3=多云, 0.55=雨, 0.75=暴雨 |
| `sunColor` | `Color` | 太阳光颜色 |
| `sunIntensity` | `float` (0~2) | 太阳光强度：1.2=晴, 0.8=多云, 0.5=雨, 0.25=暴雨 |

### 默认值（`WeatherData.CreateDefault`）

| 类型 | displayName | rainIntensity | tempModifier | thirstRateMult | ambientDarkness | sunIntensity | sunColor |
|------|-------------|:---:|:---:|:---:|:---:|:---:|------|
| Sunny | 晴 | 0 | 0 | 1.0 | 0 | 1.2 | (1, 0.95, 0.85) 暖白 |
| Cloudy | 多云 | 0 | -2 | 1.0 | 0.3 | 0.8 | (0.85, 0.88, 0.95) 灰白 |
| Rain | 雨 | 0.5 | -5 | 0.8 | 0.55 | 0.5 | (0.55, 0.6, 0.7) 灰蓝 |
| Storm | 暴雨 | 1.0 | -8 | 0.6 | 0.75 | 0.25 | (0.35, 0.38, 0.45) 暗灰 |

## 三、核心：马尔可夫链天气状态机

### 轮换触发

- **间隔**: 每 `_checkIntervalHours = 2` 游戏小时检查一次
- **条件**: `currentHour % 2 == 0` 且与 `_lastCheckedHour` 不同
- 每帧 `Update()` 中通过 `ServiceLocator.Get<TimeManager>().CurrentHour` 读取

### 转移权重矩阵

定义于 `GameConstants.cs`，4×4 矩阵 `WEATHER_TRANSITION_WEIGHTS[from, to]`:

```
            →Sunny  Cloudy  Rain  Storm
  Sunny  (0) 70     20      10     0   ← 晴不直接跳暴雨
  Cloudy (1) 40     30      25     5
  Rain   (2) 30     30      20    20
  Storm  (3) 50     30      15     5
```

每次轮换从当前天气行取权重做加权随机。

### 权重后处理（`BuildWeights()`）

1. **难度修正** — Rain/Storm 权重倍率

| 难度 | Rain 倍率 | Storm 倍率 | 冷却时间倍率 |
|------|:---:|:---:|:---:|
| Easy | ×0.7 | ×0.5 | ×1.5 |
| Normal | ×1.0 | ×1.0 | ×1.0 |
| Hard | ×1.3 | ×1.5 | ×0.5 |

2. **硬约束** (满足条件时权重直接归零):

| 约束 | 效果 |
|------|------|
| `_stormCooldownRemaining > 0` | Rain 和 Storm 权重 = 0 |
| `_consecutiveRainCount >= 3` | Rain 和 Storm 权重 = 0 |
| 当前 Sunny | Storm 权重 = 0 |

3. **地形修正** — `WEATHER_BIOME_MODIFIERS`（Phase 2 TODO，代码已预留但未接入）

### 加权随机选择（`WeightedPick()`）

```csharp
total = sum(w[0..3])
roll = Random.Range(0, total)
accum = 0
for i in 0..3:
    accum += w[i]
    if roll <= accum → return (WeatherType)i
return _currentWeather  // 兜底
```

## 四、天气切换效果（`ApplyWeather()`）

### 4.1 状态追踪

| 变量 | 行为 |
|------|------|
| `_stormCooldownRemaining` | 进入 Storm 时设 `2 × 难度倍率`，之后每次非 Storm 轮换递减1 |
| `_consecutiveRainCount` | Rain/Storm 时 +1，Sunny/Cloudy 时归零 |
| `_dailyRainTotal` | 当天累计雨量，跨日（`CurrentDay` 变化）时重置为0 |

### 4.2 光照渐变

切换到新天气时设置目标值，每帧在 `LerpSunLight()` 中平滑过渡：

```csharp
_targetSunIntensity = data.sunIntensity;
_targetSunColor    = data.sunColor;
// per-frame:
sunLight.intensity = Lerp(current, target, 1.5 * deltaTime)
sunLight.color     = Lerp(current, target, 1.5 * deltaTime)
```

> `WEATHER_LIGHT_LERP_SPEED = 1.5`，约1~2秒完成过渡。

### 4.3 环境温度

```csharp
AmbientTemperature = 15°C (BASE) + WEATHER_TEMP_MODS[type]
// Sunny → 20°C, Cloudy → 15°C, Rain → 10°C, Storm → 5°C
```

随后调用 `SurvivalSystem.SetEnvironmentTemperature(AmbientTemperature)` 同步给生存系统。

### 4.4 事件广播

```csharp
EventBus.Publish(new WeatherChangedEvent(prev, newType, rainIntensity, AmbientTemperature))
```

## 十一、依赖关系

```
WeatherManager (ServiceLocator)
├── TimeManager       ← CurrentHour (轮换触发), CurrentDay (跨日重置)
├── SurvivalSystem    ← SetEnvironmentTemperature() 同步温度
├── EventBus          ← 发布 WeatherChangedEvent
├── Light _sunLight   ← 叠加天气颜色/强度到太阳
└── WeatherData[4]    ← SO 配置

被依赖:
├── WeatherHUD        ← 读取显示
├── SurvivalSystem    ← 读取 RainIntensity/thirstRateMult
├── SaveLoadManager   ← GetSaveData()/RestoreFromSave()
```
