| 射程 | 15m | 22.5m (+50%) |
| 命中 | 100% | 100% |

### 右臂挂载（远程武器）

| 可装 | 效果 |
|---|---|
| 手枪 | 伤害×1, 1发/1s, 消耗手枪弹 |
| 步枪/猎枪 | 伤害×1.5, 1发/1.5s, 消耗步枪弹 |
| 霰弹枪 | 锥形散射×3目标, 1发/2s, 消耗霰弹 |
| 电磁步枪 | 穿透一排, 1发/3s, 消耗电池组 |

### 左臂挂载（近战/防御）

| 可装 | 效果 |
|---|---|
| 盾牌 | 机器人受伤-30% |
| 电锯 | 3m内持续切割 15/s |
| 短刀 | 2m内近战 20/1.5s |

### 攻击优先级

点击循环切换: `[激光] → [右臂] → [左臂]` → `[右臂] → [激光] → [左臂]` → `[左臂] → [右臂] → [激光]`

决定同时多目标时先打谁。各武器冷却独立，冷却好了就开火，不互等。

### 警觉距离

默认 15m，滑块范围 3m ~ 30m。僵尸进入此范围触发攻击。

## 四、三个指令模式

### 跟随

```
Tick 0.5s:

1. 检查是否可到达玩家
   PathPartial 持续5s → 传送(冷却30s)
   掉出NavMesh → 立即传送
   卡住3s → 重置路径, 5s → 传送

2. 计算距离
   距离 > 设置值+1m → SetDestination, 5m/s
   距离 < 设置值-1m → Stop()
   在范围内 → 面向玩家前方±45°

3. 途中遇到僵尸
   不恋战，绕过（Agent自动避障）
   到达玩家身边后才开始攻击

4. 到达后
   清除玩家身边警觉范围内的僵尸
```

默认跟随距离: 3m (范围1m~10m)

快捷选择: 紧贴(1m) / 适中(3m) / 松散(8m)

### 驻守

```
进入时: 记录驻守点 = 当前位置

Tick 0.5s:

1. 僵尸进警觉范围 → 激活攻击
   僵尸距驻守点 < 8m → 移动过去打
   僵尸距驻守点 > 8m → 只激光远程打

2. 僵尸死亡/跑出范围 → 返回驻守点

3. 可选: 玩家超出解除距离 → 自动切回跟随
   默认100m (范围50m~300m)
```

### 巡逻

```
围绕玩家模式:
  每2s找点: 玩家位置 + 随机偏移(半径内, 0~360°)
  → SetDestination → 到达停3~5s警戒 → 打僵尸 → 下一个点

指定地点模式:
  同逻辑，中心点改为指定坐标

可选: 玩家超出解除距离 → 自动切回跟随
  默认200m (范围100m~500m)

巡逻半径: 默认30m (范围10m~80m)
```

## 五、防自杀机制

| 场景 | 机制 |
|---|---|
| 跟随无法到达 | PathPartial 5s → 传送, 冷却30s |
| 掉出NavMesh | 立即传送回玩家身边 |
| 巡逻遇尸群 | >5只或总血量>自身×2 → 自动切跟随撤退 |
| HP < 30% | 停火, 全速回玩家身边, HUD红闪"⚠重伤" |
| HP < 10% | 丢弃一切任务, 停火省能, 到达后自动驻守 |
| HP瞬间掉>30% | 触发撤退+警报 |
| 卡住 | 3s无进展→重置路径, 5s→传送 |
| 环境危险 | 检测火/爆炸范围→绕行 |

## 六、行为优先级

```
跟随模式:
  1. HP检查 (低血量撤退最高优先)
  2. 到达玩家身边 (绕过僵尸, 不恋战)
  3. 无法到达 → 传送
  4. 到达后 → 清玩家身边的僵尸

巡逻/驻守:
  1. HP检查
  2. 尸群检查 (>5只撤退)
  3. 执行巡逻/驻守任务
  4. 打僵尸
  5. 玩家超距 → 切跟随
```
