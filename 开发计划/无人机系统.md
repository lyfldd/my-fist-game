## 四、AI无人机（AIBot专属）

### 4.1 核心规则

| 规则 | 说明 |
|------|------|
| 来源 | 造AIBot自带1架，不消耗物品 |
| 数量 | 1架/AIBot |
| 活动范围 | 离AIBot 100m |
| 被击落 | 自动飞回AIBot修复，冷却30秒后再释放 |
| AIBot死亡 | 无人机立即销毁 |
| 噪音 | 50m，投影到地面点（僵尸走向地面投影） |

### 4.2 能量系统

```
独立电池 + 铀（容量各为AIBot的 1/4）
伤害/耗能/冷却 = AIBot的 1/4 等比缩放
停靠在AIBot上 → 自动从AIBot能量池补充
能量耗尽 → 强制返航 → 切断一切攻击/操控逻辑 → 只飞回AIBot
```

四能源模式照搬 AIBot：

| | 节能 | 电力 | 铀 | 爆发 |
|------|------|------|------|------|
| 速度 | 0.75x | 1x | 1.25x | 1.5x |
| 消耗倍率 | 0.5x | 1x | 1x | 1.25x |
| 激光伤害 | ❌禁用 | 基准 | 1.5x | 1.87x |
| 冷却倍率 | 0.5x | 1x | 2x | 2.5x |
| 能量源 | 电池 | 电池 | 铀 | 铀 |

耗能/伤害按 AIBot 等比例缩小。

### 4.3 双重锥形：警戒 + 攻击

```
             📷 无人机 (俯视)
             │
        ╔═════╧═════╗
        ║ 警戒锥 120° ║  ← 探测范围，检测僵尸
        ║  ┌─────┐   ║
        ║  │攻击锥│   ║  ← 60°，实际伤害范围
        ║  │ 60° │   ║
        ║  └─────┘   ║
        ╚════════════╝
             ▼ 地面
    ┌──────────────────┐  ← 警戒投影 = 大圆
    │    ┌────────┐    │
    │    │ 攻击区 │    │  ← 攻击投影 = 小圆
    │    └────────┘    │
    └──────────────────┘
    高度越高投影越大
```

| 锥形 | 角度 | 用途 | 检测方式 |
|------|:--:|------|------|
| 警戒锥 | 120° | 僵尸进入 → 触发自动攻击 | OverlapSphere + 锥角过滤 |
| 攻击锥 | 60° | 激光命中范围，造成伤害 | 锥角过滤 + 伤害结算 |

**逻辑流程**（每 0.5s Tick，与 AIBotCombat 同频）：
1. `Physics.OverlapSphereNonAlloc` 扫描警戒锥地面投影范围内的僵尸
2. 过滤：有 `DamageableZombie` 且未死亡
3. 按距离排序（优先 AIBot 当前目标 → 其次最近僵尸）
4. 锁定目标 → 发射激光 → 命中攻击锥内所有僵尸

### 4.4 仇恨系统接入

项目**没有传统仇恨表**，沿用 AIBot 的距离优先模式。无人机通过**协同集火**与 AIBot 配合：

```
AIBot 正在打某僵尸 + 该僵尸在警戒锥内 → 优先帮打（协同集火）
AIBot 没有战斗目标 → 无人机独立选最近僵尸
```

实现方式：
- AIBotCombat 暴露 `CurrentPrimaryTarget`（最高优先级武器的当前目标）
- AIDrone 持有 AIBot 引用，每 Tick 读取
- 如果主目标在警戒锥内 → 锁定；否则 → 自选最近僵尸

### 4.5 噪音投影

| 参数 | 值 |
|------|:--:|
| 噪音来源 | 无人机地面投影点（非无人机本体） |
| 噪音半径 | 50m |
| 触发时机 | 激光攻击时 |

沿用 `DecibelSystem` 发射 `NoiseEvent`，僵尸通过 `ZombieStateMachine.OnSoundHeard()` 被吸引到地面投影点。

### 4.6 三种巡航模式

#### 巡航通用逻辑

```
圆心 XZ 平面做圆周运动：

angle = time × orbitSpeed (30°/s)
offset = (cos(angle) × radius,  0,  sin(angle) × radius)
targetPos = orbitCenter + offset,  y = cruiseHeight
transform → MoveTowards(targetPos, flySpeed × dt)
```

| 参数 | 说明 |
|------|------|
| `orbitCenter` | 巡航圆心（地面XZ） |
| `orbitRadius` | 巡航半径 |
| `cruiseHeight` | 巡航高度 |
| `orbitSpeed` | 30°/s（12秒一圈） |

#### 三种模式对比

| | 跟随 | 警戒 | 定点 |
|------|:--:|:--:|:--:|
| 圆心 | AIBot 位置 | AIBot 位置 | 玩家指定位置 |
| 半径 | **小圆** 5-8m | **大圆** 20-30m | 10-15m |
| 高度 | 10-15m | 15-25m | 玩家设定 |
| 跟随 AIBot | ✅ 圆心每帧更新 | ✅ 圆心每帧更新 | ❌ 圆心固定 |
| 自动攻击 | ✅ | ✅ | ✅ |
| 用途 | 贴身护航 | 广域侦察/预警 | 区域封锁 |

- **跟随**: 小半径护航，圆心绑定 AIBot，AI 移动无人机跟着移动
- **警戒**: 大半径巡逻，圆心绑定 AIBot，覆盖更广提前发现威胁
- **定点**: 圆心锁定地图某点，不跟 AIBot 走

### 4.7 活动范围约束

```
        ┌─────────────────────────┐
        │   硬边界 100m             │
        │   ┌───────────────────┐ │
        │   │  软边界 80m       │ │
        │   │  ┌──────┐        │ │
        │   │  │AIBot │        │ │
        │   │  │ 🤖   │        │ │
        │   │  └──────┘        │ │
        │   │   🛸 巡航中       │ │
        │   └───────────────────┘ │
        │     AIBot 走远了→ 🚶    │
        └─────────────────────────┘
```

| 边界 | 距离 | 行为 |
|------|:--:|------|
| 正常区 | < 80m | 正常巡航/攻击 |
| 软边界 | 80-100m | 中断巡航，直线飞向 AIBot |
| 硬边界 | ≥ 100m | 强制截断，不可超出 |

**定点模式下的处理**：

```
定点巡航中 ─→ 检测 d = |无人机 - AIBot|
                   │
          d < 80m  │  80m ≤ d < 100m   │  d ≥ 100m
             ↓     │      ↓            │     ↓
          继续巡航  │  中断巡航         │  强制拉回
                   │  飞向 AIBot       │  停在 100m 边界
                   │  (保持巡航高度)   │  等待 AIBot 返回
                   │                   │
                   │  AIBot 回来 < 80m │
                   │      ↓            │
                   │  恢复定点巡航      │
```

### 4.8 寻路逻辑

无人机**空中飞行**，不需要 NavMesh（地面寻路），使用**直线插值移动**：

```
当前位置 ────→ 目标位置 (直线，无地面障碍)
```

| 参数 | 值 | 说明 |
|------|:--:|------|
| 正常飞行速度 | 8 m/s | 巡航 + 跟随 |
| 返航速度 | 12 m/s | 能量耗尽/被击落返航时加速 |
| 最小高度 | 5m | 不能低于此高度 |
| 最大高度 | 50m | |

对比 AIBot：`NavMeshAgent.SetDestination()` 地面路径 + 避障；无人机：`Vector3.MoveTowards()` / `SmoothDamp` 直线飞行。

### 4.9 完整状态机

```
                    ┌─────────────────┐
                    │   停靠 (Docked)  │ ← 在 AIBot 上充电
                    └──────┬──────────┘
                           │ [释放]
                           ▼
                    ┌─────────────────┐
              ┌────→│   飞行 (Flying)  │
              │     └──────┬──────────┘
              │            │
              │     ┌──────┼──────┐
              │     ▼      ▼      ▼
              │  跟随   警戒   定点巡航
              │     │      │      │
              │     └──────┼──────┘
              │            │
              │     [能量耗尽/被击落(Hp=0)/召回]
              │            ▼
              │     ┌─────────────────┐
              └─────│  返航 (Returning)│
                    └────────┬────────┘
                             │ [到达AIBot]
                             ▼
                    ┌─────────────────┐
                    │  停靠 (Docked)   │
                    └─────────────────┘
```

**子状态（Flying 内）**：

```
Flying
  ├── Following   — 小圆巡航，圆心跟随 AIBot
  ├── Guarding    — 大圆巡航，圆心跟随 AIBot
  ├── Defending   — 定点巡航，圆心固定
  ├── Attacking   — 锁定目标，激光输出，攻击完回到巡航
  └── Returning   — 直线飞回 AIBot
```

**状态切换规则**：

| 触发条件 | 目标状态 |
|------|------|
| AIBot 面板点 [释放] | Launching → Following |
| AIBot 面板切换模式 | 对应的巡航子状态 |
| 警戒锥发现僵尸 | Attacking（无僵尸后回巡航） |
| AIBot 面板点 [召回] | Returning |
| 能量耗尽 | Returning（强制，切断攻击） |
| 被击落 (HP=0) | Returning（强制返航修复） |
| 到达 AIBot | Docked → 冷却 30s |
| AIBot 死亡 | 立即 Destroy |

## 五、受击规则汇总

```
飞行中:
  高度 > 近战范围 → 僵尸近战打不到
  高度 ≤ 近战范围 (低空攻击时) → 任何可攻击的东西都能打
  远程/投掷僵尸 → 任何高度都能威胁

地面待命:
  等同于地面物品 → 可被任何攻击破坏

种类区别:
  普通无人机 → 被击落 = 销毁
  AI无人机   → 被击落 = 自动飞回AIBot修复
```
