# 3.4 建造系统 ✅ Phase 1 完成

> **状态**: 已实现 · **日期**: 2026-05-15 · **依赖**: L2 背包系统(Inventory) + L2 交互系统(IInteractable) + L2 人物系统(SkillType)

---

## 一、功能概述

玩家可放置建筑结构到游戏世界中，支持 Ghost 半透明预览、网格吸附、重叠检测、材料扣除、建造耗时、技能等级检查、拆除返还材料。7 个默认建造资产覆盖墙体/地板/门窗/家具/路障。

## 三、架构

```
按 B 进入建造模式:
  BuildModeController → state = Preview
    → EventBus.Publish(BuildModeEnteredEvent)
        ├─ GhostPreview.OnEnter → SpawnPreview(半透明)
        ├─ BuildMenuUI.OnEnter → 显示底部菜单
        ├─ BuildModeInputLock.OnEnter → 禁用移动/武器组件
        └─ PlayerInteraction.OnEnter → isBuildModeActive=true

每帧预览(GhostPreview.Update):
  1. MouseGroundProjector.GroundPoint  →  鼠标地面投影点
  2. SnapToGrid(hitPoint, snapSize)    →  网格吸附
  3. Physics.OverlapBox(pos, size/2)   →  重叠检测
  4. 绿色(有效) / 红色(无效)

左键确认放置(5重检查):
  1. activeBuildable != null?
  2. GhostPreview.IsValidPlacement?       (无重叠)
  3. HasRequiredMaterials()?              (背包材料)
  4. IsRequiredToolEquipped()?            (右手工具)
  5. skillLevelRequired >= 建造拆解等级?

  通过 → BuildRoutine(协程,buildDuration秒)
    → CompleteBuilding():
        DeductMaterials() → Instantiate(builtPrefab)
        → Publish(StructurePlacedEvent + SurvivalXpGained)
        → ExitBuildMode()

E 键拆除(PlacedStructure : IInteractable):
  进度条(buildDuration×0.5) → RefundMaterials(returnRate)
    → Publish(StructureDeconstructedEvent) → Destroy(gameObject)
```

## 四、BuildableData 参数

| 参数 | 类型 | 说明 |
|------|------|------|
| displayName | string | UI 显示名称 |
| category | BuildableCategory | Wall/Floor/Furniture/Barricade |
| previewPrefab | GameObject | 半透明预览模型 |
| builtPrefab | GameObject | 放置后的实体模型 |
| placementSize | Vector3 | OverlapBox 检测尺寸 |
| snapSize | float | 网格吸附步长(0=自由) |
| buildDuration | float | 建造耗时(秒) |
| requiredTool | ItemData | 右手需装备的工具 |
| materials | ItemRequirement[] | 材料清单 |
| skillLevelRequired | int | 所需建造拆解等级 |
| maxHealth | float | 结构血量(0=无敌) |
| blocksNavMesh | bool | 预留：阻挡僵尸AI路径 |
| deconstructReturnRate | float | 拆除材料返还比例(0~1) |

## 五、事件

```csharp
BuildModeEnteredEvent(BuildableData buildable)      // 进入建造模式
BuildModeExitedEvent                                 // 退出建造模式
StructurePlacedEvent(BuildableData, Vector3, GameObject)  // 放置完成
StructureDeconstructedEvent(BuildableData, Vector3, GameObject) // 拆除完成
```
