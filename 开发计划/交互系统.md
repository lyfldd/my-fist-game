## 一、功能概述

统一交互接口 `IInteractable`，所有可交互对象（容器/建筑/车辆/地面物品）实现此接口。`PlayerInteraction` 通过 `Physics.OverlapSphere` 检测最近的可交互对象，E 键触发交互。交互提示 UI 和进度条基础设施已就绪，但运行时链接不完整。

---

## 二、IInteractable 接口

```csharp
// Systems/Interaction/IInteractable.cs
public interface IInteractable
{
    string InteractionPrompt { get; }  // UI 提示文本，如"搜索柜子 [E]"
    float InteractionTime { get; }     // 0=瞬间, >0=需按住时间
    bool IsInteractable { get; }       // 当前是否可交互
    void OnInteract(GameObject interactor);
}
```

**实现者**：
| 类 | 作用 | InteractionTime |
|------|------|:--:|
| WorldContainer | 搜索→打开容器 | searchTime (1-4s) |
| GroundItemInteract | 拾取到背包 | 0 (瞬间) |
| VehicleInteraction | 上下车 | 0 (瞬间) |
| PlacedStructure | 拆除返还材料 | buildDuration×0.5 |
| TestInteractable | 调试测试 | 可配置 |

## 三、PlayerInteraction 检测逻辑

```csharp
// Systems/Interaction/PlayerInteraction.cs
void Update()
{
    if (isBuildModeActive) return;  // 建造模式下跳过

    // Physics.OverlapSphere(玩家位置, INTERACT_DETECT_RADIUS=2.5m)
    // 遍历碰撞体 → 找最近 IInteractable + IsInteractable=true
    DetectTarget();

    if (Input.GetKeyDown(KeyCode.E) && _currentTarget != null)
    {
        _currentTarget.OnInteract(gameObject);
        EventBus.Publish(new InteractionEvent(objectName));
    }
}
```
