> **目标**: 消除硬编码，新建物品纯字段驱动，填完即用。

## 二、详细方案

### 2.1 VehicleController — 消除燃料硬编码

```
旧: public string fuelItemName = "SyntheticGasoline";
    ConsumeFuel() → RemoveItemByName(fuelItemName, 1)

新: 删除 fuelItemName 字段
    ConsumeFuel() → 遍历 containers 找 fuelValue > 0 的物品 → RemoveItem
    无燃料物品 → 不可驾驶
```

### 2.2 ElectronicDevice — 读 ItemData 而非硬编码名字

```
旧: GetDurabilityRatio() → switch(deviceType) { Flashlight→"手电筒"... }
    遍历背包找 itemName == 硬编码字符串

新: GetDurabilityRatio() → 遍历背包找 itemData.electronicDeviceType == this.deviceType
    ReplaceBattery() → 同理遍历找 electronicDeviceType == Battery 的物品
```

### 2.3 AIBotWeaponMapping — 替代硬编码

```
新建: Assets/_Game/Config/AIBotWeaponMapping.asset
  [CreateAssetMenu(menuName="Game/AI Bot Weapon Mapping")]
  class AIBotWeaponMapping : ScriptableObject {
      Entry[] entries;
      class Entry { RightArmWeapon weapon; ItemData weaponItem; ItemData ammoItem; }
  }

改 AIBotCombat:
  删: const AMMO_PISTOL etc
  删: GetWeaponItemName 中的 switch-case
  删: GetRightArmFromItemName 反向映射
  加: _mapping 字段（ScriptableObject 引用）
  ConsumeAmmoInternal → 读 _mapping.ammoItem.itemName
  EquipWeapon → 读 _mapping.weaponItem
```

### 2.4 SurvivalSystem — 激活死字段

```
OnItemUsed 加 healAmount 分支:
  if (evt.ItemData.healAmount > 0)
      ModifyHealth(+evt.ItemData.healAmount × 医疗技能加成, "药品");

foodType 触发识别:
  if (evt.ItemData.foodType != null) → 也进入处理（即使 itemEffects 为空）
```

### 2.5 DurabilitySystem — 加安全兜底

```
ConsumeDurability:
  if (!p.Value.itemData.hasDurability || p.Value.itemData.maxDurability <= 0f)
      return;  // ← 新增 maxDurability 检查
```

### 2.6 Inventory — 防容器塌缩

```
EquipItem:
  int newW = item.storageWidth;
  int newH = item.storageHeight;
  if (newW <= 0) newW = GameConstants.DEFAULT_STORAGE_SIZE; // 1
  if (newH <= 0) newH = GameConstants.DEFAULT_STORAGE_SIZE;
```

### 2.7 ItemGraphBuilder — 引用完整性

```
Build() 末尾:
  foreach RecipeData → 检查 resultItem 非 null + materials[i].itemData 非 null
  foreach ItemData(isFirearm) → 检查 ammoItemName 指向存在的物品
  foreach BuildableData → 检查 materials[i].itemData 非 null
  foreach ProductionDeviceData → 检查 recipes 的 input/output 非 null

  输出 Console: "[ItemGraph] ✅ 159/159 ❌ 3 断裂: AK47→7.62mm(不存在)"
```
