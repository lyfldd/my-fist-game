> **日期**: 2026-05-28 · **状态**: ✅ 已完成 · **依赖**: 物品系统(2.1) + 合成系统(4.2) + 建造系统(3.4)

## 一、背景与问题诊断

当前194配方+25工业设备分布在9个工作台上，存在以下结构性问题：

### 1.1 工作台职责混乱

| 问题 | 现状 | 应该 |
|------|------|------|
| **高级工作台抢枪械** | 手枪/步枪/霰弹枪在L5 | 枪械应在车床(L6)，L5不该有枪 |
| **机械加工台抢零件** | 高级零件批量/齿轮批量在L7 | 零件批量是车床工业量产的活 |
| **车床空有躯壳** | 套件/建筑/生产设备都有，但配方=0 | 应该是精密金属加工的核心工作台 |
| **乞丐配方混入高级台** | 篝火套件/铁甲/铁盾在L5 | 应下放到L2-L4 |

### 1.2 物品无上下游追溯

每个物品只知道"用什么材料造"，不知道"造出来后被谁消费"。导致：

- **断头路**：铝锭做完钛合金就没用了；齿轮/轴承造完几个套件后成废品
- **消费不均衡**：钢锭下游30+，铝锭下游只有2个
- **无法判断一个物品是否重要**：没有完整的输入→输出链

### 1.3 终局工业配方偏少

- 精密装配台：仅4个量产配方
- 元素合成炉：2个未完成配方
- 终局工业建筑的功能感不足

## 二、产业链全景图

### 2.1 金属链（主线，智力1-10）

```
段一(1-3): 冶炼
  矿石 → 金属锭（熔炉手工 / 工业炉量产）
  
段二(3-4): 基础零件  
  铁锭 → 通用零件/弹簧/齿轮（中级工作台手工 / 冲压机量产）
  
段三(5-7): 精密金属 + 枪械  ← 车床时代
  钢锭 → 高级零件/轴承/钢管/线圈（车床手工）
  高级零件 → 手枪/步枪/霰弹枪/猎枪/消音器/改造套件
  车床工业自动：齿轮批量/轴承批量/弹簧组件批量/钢管批量

段四(7-9): 重工业材料
  铝锭+铁锭 → 钛合金 → 重狙/榴弹/工业套件
  橡胶+化学试剂 → 碳纤维 → 消音器/无人机

段五(9-10): 终局消费
  钛合金+芯片组+伺服电机 → 自动炮塔/电磁围栏/核电站
  浓缩铀+钛合金 → 小型反应堆 → 聚变核心
```

### 2.2 硅/电子链（支线，智力5-10）

```
沙子→玻璃板→电路板→芯片组→AI核心→量子处理器
电池→电池组→电容组→伺服电机
浓缩铀→小型反应堆→等离子电池(新)

### 2.3 化学链（支线，智力3-8）

### 2.4 生物/医疗链（支线，智力2-10）

```
草药→消毒水→急救包→手术包→疫苗血清→基因样本→逆转疫苗

### 2.5 能源链（终局，智力8-10）

```
铀矿→浓缩铀→小型反应堆→聚变核心(小)→聚变核心(大)
                              →超导线圈→等离子电池
```

### 2.6 食品链（短链，智力0-3）

```
生肉→熟肉→熏肉/肉干→罐头
```

## 三、断头路清单

### 🔴 严重 — 无可持续消费

| 物品 | 当前下游 | 问题 |
|------|---------|------|
| 铝锭 | 钛合金、闪光弹 | 2个下游，做完钛合金就废 |
| 齿轮 | 粉碎机/冲压机/装配台/机械加工台/移动要塞 套件 | 造完一次性消费 |

## 四、修复方案

### 4.1 配方下放（清理高级工作台）

| 配方 | 从 | 到 | 理由 |
|------|:--:|:--:|------|
| 篝火套件 | L5 | L2 | 造个篝火不需要高级工作台 |
| 简易工作台套件 | L5 | L2 | 同上 |
| 熔炉套件 | L5 | L2/L4 | 建造2难度，放简易台 |
| 铁甲 | L5 | L4 | 铁锭+皮革+线，皮甲升级版 |
| 铁盾 | L5 | L4 | 铁锭+木板，木盾升级版 |
| 钢管 | L5 | L4 | 加工难度=齿轮，齿轮就在L4 |
| 钢筋 | L5 | L4 | 同上 |
| 穿甲弹 | L5 | L7 | 枪械6的弹药，机械加工台更合适 |
| 手枪/步枪/霰弹枪 | L5 | L6(车床) | 枪械属于车床 |

### 4.2 车床激活（新建L6工作台等级）

在 WorkstationTier 枚举插入 `Lathe = 6`，后续全后移。

**车床手工配方（精密金属+枪械）：**

| 分类 | 配方 |
|------|------|
| 精密零件 | 高级零件、轴承、钢管、钢筋、线圈 |
| 枪械 | 手枪(枪2)、霰弹枪(枪3)、步枪(枪4)、猎枪(枪5) |
| 武器附件 | 消音器(枪5)、半自动改造(枪6)、全自动改造(枪7) |
| 工业套件 | 车床套件(自举)、冲压机套件、装配台套件 |

**车床工业量产（生产设备自动）：**
- 齿轮批量、轴承批量、弹簧组件批量、钢管批量

### 4.3 断头路修复

| 物品 | 修复方式 |
|------|---------|
| 铝锭 | +铝合金板材（新半成品，用于高级护甲/载具零件） |
| 齿轮 | +车辆维修消耗（每次维修消耗2-3个齿轮） |
| 轴承 | +工业设备运转消耗（定期损坏需更换） |
| 碳纤维 | +碳纤护甲板、碳纤盾牌、复合弓 |
| 弹簧组件 | +减震握把、战术枪托（新武器附件） |
| 钢筋 | +建筑维修消耗 |

### 4.4 终局配方扩充

**精密装配台（工业量产）新配方：**
- 芯片组×2 量产（电路板×6→芯片组×2）
- 光学透镜×2 量产（玻璃板×6→光学透镜×2）
- 传感器模块（新电子半成品）

**元素合成炉新配方：**
- 修复聚变核心(小/大)
- 等离子电池
- 超导线圈
- 高级合金复合材料
- 量子处理器

### 4.5 工作台 vs 工业分工原则

| | 工作台（手工） | 工业设备（量产） |
|---|---|---|
| 定位 | 第一次做、偶尔做、技能要求高 | 批量产出、持续供给 |
| 智力要求 | 高（你要会做） | 低（机器替你做） |
| 消耗 | 玩家体力+时间 | 电力+燃料+原料 |
| 例子 | 手工搓一块芯片组 | 精密装配台每小时产芯片组×5 |

## 五、工作台等级重组

```
L0 徒手          生存：石斧、绷带、火把
L1 篝火          烹饪：烤肉、炖菜、烧水
L2 简易工作台    基础制造：木板、绳子、布卷、基础工具、入门工作台套件
L3 熔炉          冶炼：矿石→锭、玻璃、水泥、火药、精炼煤炭
L4 中级工作台    基础加工：通用零件、弹簧、齿轮、冷兵器、皮甲/铁甲、钢管/钢筋
L5 高级工作台    精密零件+电路入门+弹药+工业套件
L6 车床          精密金属+枪械：高级零件、手枪到猎枪、武器附件
L7 化学台        化工/药剂/燃料/投掷物
L8 机械加工台    重工业：钛合金、碳纤维、重狙、工业设备套件、核电站套件
L9 电子装配台    电子/芯片/智能：芯片组、AI核心、电磁步枪、智能设备
L10 元素合成炉   终极能源：聚变核心、等离子电池、超导线圈
```

## 八、今晚讨论总结（2026-05-28）

1. **产业链梳理**：完成6条链的全景图（金属/电子/化学/生物医疗/能源/食品）
2. **上下游追溯**：逐一检查每个物品的输入→输出，标记断头路
3. **配方下放决策**：高级工作台7个乞丐配方下放+L5枪械迁移到车床
4. **车床激活**：从空壳升级为L6独立工作台等级
5. **分工原则**：工作台=手工（高技能+低量），工业设备=量产（低技能+高量+耗电）
6. **修复方向**：断头路分A类（加持续消耗）+B类（加产品路径）
