## 一、功能概述

四属性(STR/AGI/CON/END) + 十技能双轨成长系统。XP 池 → 手动兑换技能点 → 递增成本升级。体力系统独立运作，受耐力属性驱动。属性/技能通过公式影响战斗伤害、射击精度、建造门槛、负重等。

## 三、四属性 (AttributeType)

```csharp
// Config/CharacterData.cs
public enum AttributeType
{
    Strength,     // 力量: 近战伤害+5%/点, 负重+5kg/点
    Agility,      // 敏捷: 移动速度, 闪避, 潜行
    Constitution, // 体质: 最大生命, 自然恢复
    Endurance     // 耐力: 体力上限+10/点, 体力恢复+2/秒/点
}
```

默认值全部为 **5**，范围 1-10。升级成本公式：`5 + currentLevel × 0.5`。

## 四、十技能 (SkillType) 与升级成本

```csharp
// Config/CharacterData.cs
public enum SkillType
{
    // 战斗组 (基础10, 增量1.7)
    近战专精,   // Melee: 伤害×1.08/级
    枪械专精,   // Guns: 散布×0.94/级
    防御专精,   // Defense: 伤害减免(配合护甲)

    // 生存组
    资源采集,   // Gathering: 基础10, 增量0.6 (最便宜)
    医疗生存,   // Medical: 基础25, 增量1.9
    野外求生,   // Wilderness: 基础15, 增量1.7

    // 工程组
    工匠制作,   // Crafting: 基础35, 增量1.9
    建造拆解,   // Building: 基础35, 增量1.9
    汽车改造,   // Vehicle: 基础40, 增量1.9

    // 心智
    智力        // Intelligence: 基础50, 增量3.3 (最贵)
}
```

默认等级全部为 0，近战专精从 1 开始。最大等级 10。

### 升级成本公式

```
技能: cost = round(baseCost + currentLevel × increment)
属性: cost = round(5 + currentLevel × 0.5)
```

例：近战专精 Lv0→1 需10点, Lv9→10 需25点。智力 Lv0→1 需50点, Lv9→10 需80点。

## 五、SurvivalXPSystem (核心单例)

```csharp
// Systems/Character/SurvivalXPSystem.cs
public class SurvivalXPSystem : MonoBehaviour
{
    public static SurvivalXPSystem Instance;

    int _totalXP;            // 原始经验池
    int _availablePoints;    // 可分配技能点
    int _availableAttrPoints; // 可分配属性点

    Dictionary<AttributeType, RuntimeAttribute> _attributes;   // 4项
    Dictionary<SkillType, RuntimeSkill> _skills;               // 10项
}
```

### XP 获取流程

```
任意系统发布 SurvivalXpGained(amount, source)
  → SurvivalXPSystem.OnXPGained() 订阅接收
  → _totalXP += amount
  → ConvertXPToPoints(): _totalXP / 100 → 累加到 _availablePoints
  → 发布 CharacterStatsChanged("xp_converted", ...)
```

**兑换率**: 100 XP = 1 技能点 (`XP_PER_SKILL_POINT`)

### 升级流程

```
SpendPoint(SkillType skill):
  1. 检查等级 < 10
  2. 查 SkillCostTable.GetCost(skill, currentLevel) → cost
  3. _availablePoints >= cost ?
     是 → 扣除点数 → level++ → 发布 CharacterStatsChanged("skill_up", ...)
     否 → 拒绝

SpendAttributePoint(AttributeType attr):
  同逻辑，查属性成本表
```

## 六、StaminaSystem (体力系统)

```csharp
// Systems/Character/StaminaSystem.cs
public class StaminaSystem : MonoBehaviour
{
    float _currentStamina;
    float _maxStamina;       // = 100 + endurance × 10
    float _currentDrain;     // 当前消耗速率

    // 状态
    bool _isSprinting;
    bool _isOverloaded;
}
```

### 体力公式

```
最大体力 = 100 + 耐力 × 10         (默认150)
恢复(站立) = 15 + 耐力 × 2  /秒   (默认25/秒)
恢复(行走) = 10 + 耐力 × 2  /秒   (默认20/秒)
消耗(奔跑) = 8/秒
消耗(近战) = 15/次 (瞬间)
消耗(劳作) = 10/次 (瞬间)
超重惩罚 = 所有消耗 × 1.5
```

### 关键机制

- 仅在消耗为 0 时恢复（`_currentDrain == 0`）
- 体力归零 → 移动速度 ×0.6（`STAMINA_EXHAUSTED_SPEED_MULT`）
- `CanPerform(float cost)` 门控动作执行
- 外部调用 `SetRunning(bool)` / `SetWalking(bool)` / `SetOverloaded(bool)` 切换状态
