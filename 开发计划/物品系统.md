## 一、功能概述

`ItemData` ScriptableObject 定义所有物品的属性和行为。一个 SO 涵盖：背包属性/装备属性/武器属性/生存效果/世界表现。编辑器一键批量创建默认物品。

---

## 二、ItemData 全字段

```csharp
// Config/ItemData.cs
[CreateAssetMenu(menuName = "Game/Item")]
public class ItemData : ScriptableObject
{
    // 基础信息
    string itemName;         // 显示名称
    Sprite icon;             // 图标
    string description;      // 描述

    // 背包空间
    int gridWidth = 1;       // 占用格子宽
    int gridHeight = 1;      // 占用格子高

    // 游戏属性
    ItemCategory category;   // 分类
    int maxStack = 1;        // 最大堆叠数
    float weight = 1.0;      // 重量(kg)

    // 装备属性
    EquipSlot equipSlot = None;    // 装备位
    int storageWidth = 0;          // 装备后容器扩容宽
    int storageHeight = 0;         // 装备后容器扩容高
    float armorValue = 0;          // 护甲值

    // 世界表现
    GameObject worldPrefab;  // 掉落地面预制体

    // 使用
    float useTime = 1.0;     // 使用耗时

    // 生存效果
    List<ItemEffect> itemEffects;  // 使用效果列表
    float warmthValue = 0;         // 保暖值
    bool isWaterproof = false;

    // 武器属性
    bool isFirearm = false;        // true=枪械 false=近战
    float weaponDamage = 10;
    float weaponRange = 50;
    float fireRate = 0.5;
    float baseSpread = 2;
    float maxSpread = 15;
    float spreadRecovery = 8;
    float spreadPerShot = 1.5;
    int magazineSize = 0;          // 0=无限制/近战
}
```

## 四、ItemEffect 效果系统

```csharp
// Config/AttributeEnums.cs
public enum ItemEffectType
{
    RestoreHealth, RestoreHunger, RestoreThirst,
    RestoreTemperature,
    CureBleeding, CureInfected, FixFracture,
    TemporaryWarmth
}

[Serializable]
public class ItemEffect
{
    public ItemEffectType effectType;
    public float value;
    public float duration;   // 持续效果(秒)
    public bool isInstant;   // true=立即生效
}
```

**效果应用流程**: `QuickItemBar(右击物品)` → `ItemUsedEvent` → `ItemUsageSystem(扣物品)` → `SurvivalSystem.ApplyItemEffect()`

## 六、当前代码与计划差异

| 项目 | 当前代码 | 设计计划 |
|------|:----:|:----:|
| ItemCategory | 5大类(Weapon/Food/Material/Medical/Junk) | 8大类(RawMaterial/SemiFinished/Consumable/Equipment/Ammo/Buildable/Workstation/Functional) |
| ItemQuality | 无 | Scavenged/Handmade/Professional |
| ItemUsageTag | 无 | 烹饪/冶炼/建造/战斗/医疗/能源 |
| ItemRequirement | 无(结构体不存在) | 配方材料需求 |
| hasDurability/maxDurability | 无 | 装备耐久 |
| isWorkstation/workstationTier | 无 | 工作台物品标记 |

这些都是 4.2 合成系统实现时需要改造的项。
