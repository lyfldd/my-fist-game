#### 2.7 职业系统 ⬜ Phase 1 设计完成

> 状态: 设计定稿，待实现。依赖 2.6 人物系统重构（已完成）。
> 日期: 2026-05-20

**设计概述**: 职业是角色创建的入口选择，决定初始属性修正、技能加成、开局装备和随身物品。8个预设职业覆盖不同玩法路线 + 1个彩蛋角色（制作人）。

**执行顺序（关键）**: 必须先穿装备扩容容器，再放物品。
```
1. InitializeContainers()      → 容器默认尺寸(1x1~2x2)
2. ApplyProfession()           → 属性修正 + 技能加成
3. EquipStartingGear()         → 穿装备 → 容器扩容
4. AddStartingItems()          → 填物品 → 放入已扩容容器
```

**CharacterTemplate 新增字段**:
```csharp
public ItemData[] startingEquipment;    // 开局装备（EquipItem 自动扩容）
public ItemRequirement[] startingItems; // 开局随身物品（扩容后放入）
```
