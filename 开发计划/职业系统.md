# 2.7 职业系统 ✅ Phase 1 完成

> **状态**: 已实现 · **日期**: 2026-05-11 · **依赖**: L2 人物系统(SurvivalXPSystem) + L1 ItemData

## 一、功能概述

CharacterTemplate ScriptableObject 定义 9 种职业的属性加成/技能提升/初始装备/初始物品。ProfessionApplier 在角色初始化时分两步应用：先穿装备扩容网格 → 再放入物品。编辑器一键生成所有职业 .asset 文件。

---

## 二、文件清单

| 文件 | 说明 |
|------|------|
| `Config/CharacterTemplate.cs` | 职业模板 SO 定义 (AttributeMod + SkillBoost + startingEquipment + startingItems) |
| `Config/CharacterData.cs` | 含 `AttributeMod` 类 + `profession` 引用字段 |
| `Systems/Character/ProfessionApplier.cs` | 两阶段初始装备应用器 |
| `Systems/Character/SurvivalXPSystem.cs` | ApplyProfession() 加载属性/技能修饰，触发 ProfessionApplier |
| `Editor/CreateProfessionTemplates.cs` | 编辑器菜单：一键生成 9 职业 .asset 文件 |
| `Config/Character/DefaultCharacter.asset` | 默认角色配置，profession 指向"流浪者" |
| `Config/Character/Templates/*.asset` | 9 个职业模板资产 |

## 三、CharacterTemplate 结构

```csharp
// Config/CharacterTemplate.cs
[CreateAssetMenu(menuName = "Game/Character Template")]
public class CharacterTemplate : ScriptableObject
{
    public string templateName;
    [TextArea] public string description;
    public List<AttributeMod> attributeMods;      // 属性加成
    public List<SkillBoost> skillBoosts;          // 技能提升
    public ItemData[] startingEquipment;           // 初始装备 (先穿)
    public ItemRequirement[] startingItems;        // 初始物品 (后放)
}

[Serializable]
public class SkillBoost
{
    public SkillType skill;
    public int bonus;
}

// AttributeMod 定义在 CharacterData.cs
[Serializable]
public class AttributeMod
{
    public AttributeType attributeType;
    public int mod;
}
```

- `startingEquipment` → 装备到 EquipSlot，扩容对应容器网格
- `startingItems` → 放入已扩容的容器空间
- 顺序严格：**先装备后物品**，否则空间不足

## 五、角色创建流程

```
1. PlayerCharacter.Start()
     → 加载 DefaultCharacter.asset (profession = "流浪者")
     → 确保 SurvivalXPSystem 单例存在

2. SurvivalXPSystem.InitFromCharacterData()
     → 从 CharacterData 读基准属性 (全部=5) + 基准技能 (近战=1, 其余=0)
     → 如果 characterData.profession != null:
        a. ApplyProfession(template)
             → 遍历 attributeMods → _attributes[type].BaseValue += mod (夹在 1-10)
             → 遍历 skillBoosts → _skills[type].BaseLevel += bonus
        b. 获取/添加 ProfessionApplier → ApplyStartingGear(template)

3. ProfessionApplier.ApplyStartingGear(template)
     阶段1: 遍历 startingEquipment → Inventory.EquipItem(item)
             → 装备到对应槽位 → 容器网格扩容
     阶段2: 遍历 startingItems → Inventory.AddItem(itemData, count)
             → 物品放入已扩容的容器

4. SurvivalXPSystem.Start()
     → 订阅 SurvivalXpGained 事件
```

## 六、编辑器工具

```csharp
// Editor/CreateProfessionTemplates.cs
[MenuItem("Game/创建默认职业模板")]
static void CreateDefaultProfessionTemplates()
{
    // 为每个职业调用专用方法:
    CreateFirefighter() / CreatePoliceOfficer() / CreateDoctor()
    CreateEngineer() / CreateSoldier() / CreateLumberjack()
    CreateVagrant() / CreateMechanic() / CreateCreator()

    // 保存到 Assets/_Game/Config/Character/Templates/{name}.asset
}
```

通过 `AssetDatabase.LoadAssetAtPath<ItemData>(path)` 加载物品引用，硬编码路径指向 `Config/Items/` 目录。

## 七、设计特点

- **纯 SO 架构**: 职业数据完全以 .asset 文件存在，无硬编码数据
- **两阶段应用**: 先穿装备扩容 → 后放物品，顺序不可逆
- **模板与运行时分离**: CharacterTemplate 是静态配置，SurvivalXPSystem 持有可变 RuntimeSkill/RuntimeAttribute
- **可扩展**: 新增职业只需创建新的 CharacterTemplate .asset，无需改代码
