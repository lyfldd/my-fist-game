## 1. 系统概览

人物系统由三个运行时组件和一个配置层组成：

| 层 | 组件 | 职责 |
|---|------|------|
| 配置 | `CharacterData` (SO) | 定义角色初始属性/技能模板 |
| 配置 | `CharacterTemplate` (SO) | 定义职业（属性修正、技能加成、初始装备、AI机器人开局） |
| 运行时 | `PlayerCharacter` | 挂载于玩家 GameObject，提供便利属性访问 + 负重惩罚联动 |
| 运行时 | `StaminaSystem` | 体力消耗/恢复模型（跑步、近战、劳作），关联耐力属性 |
| 运行时 | `SurvivalXPSystem` | 全局单例经验池，XP→技能点换算，属性/技能升级消费 |
| 运行时 | `ProfessionApplier` | 职业开局物品投放（装备 → 背包物品 → AI机器人） |
| 数据 | `RuntimeSkill` / `RuntimeAttribute` | 可序列化的运行时数据容器，Inspector 可见 |

## 2. 属性系统 (Attributes)

### 2.1 四种身体属性

定义于 `_Game/Config/CharacterData.cs` — `AttributeType` 枚举：

| 属性 | 英文 | 简称 | 作用 |
|------|------|------|------|
| 力量 | Strength | STR | 近战伤害、负重上限 |
| 敏捷 | Agility | AGI | 移动速度、闪避率、潜行效果 |
| 体质 | Constitution | CON | 生命值上限、自然恢复速度 |
| 耐力 | Endurance | END | 体力上限、体力恢复速度 |

- 取值范围：`min=1, max=10`，默认值 `5`
- 升级消耗公式（`SkillCostTable.GetAttributeCost`）：`Round(5 + currentLevel * 0.5)`  
  即 Lv0→Lv1=5点，Lv9→Lv10=10点。

### 2.2 属性定义与修正

```csharp
// 属性定义（配置层）
[Serializable] public class AttributeDef {
    AttributeType type;
    int defaultValue = 5;
    int min = 1;
    int max = 10;
}

// 属性修正（职业/性别用）
[Serializable] public class AttributeMod {
    AttributeType attributeType;
    int mod; // 正数为加，负数为减
}
```

`CharacterData` 内持有 `List<AttributeDef> attributes` 定义初始值，`CharacterTemplate` 内持有 `List<AttributeMod> attributeMods` 提供职业修正。运行时数据使用 `RuntimeAttribute`（`value/min/max` 三字段，标记 `[Serializable]`）。

## 3. 技能系统 (Skills)

### 3.1 十大技能 — 四大分类

定义于 `_Game/Config/CharacterData.cs` — `SkillType` 枚举：

| 分类 | 技能 | 基础消耗 | 递增系数 | 效果描述 |
|------|------|----------|----------|----------|
| 战斗 | 近战专精 | 10 | 1.7 | 近战伤害加成 + 近战闪避 |
| 战斗 | 枪械专精 | 10 | 1.7 | 瞄准精度（散射锥收窄） |
| 战斗 | 防御专精 | 10 | 1.7 | 减伤率（联动装备护甲） |
| 生存 | 资源采集 | 10 | 0.6 | 伐木/挖矿/搜刮速度 + 稀有爆率 |
| 生存 | 医疗生存 | 25 | 1.9 | 医疗品效果加成 + debuff处理 + 草药制作 |
| 生存 | 野外求生 | 15 | 1.7 | 追踪/狩猎/陷阱 + 环境适应 |
| 制造 | 工匠制作 | 35 | 1.9 | 工具/武器制作品质 + 改造配件 |
| 制造 | 建造拆解 | 35 | 1.9 | 建造耗时缩短 + 拆解返还提高 |
| 制造 | 汽车改造 | 40 | 1.9 | 载具改装解锁 |
| 心智 | 智力 | 50 | 3.3 | 配方解锁门槛 + 高阶制作品质 + NPC说服 |

### 3.2 技能升级消耗公式

（`SkillCostTable.GetCost`）：`Round(baseCost + currentLevel * increment)`

智力是最昂贵的技能（基线50，每级+3.3），资源采集最为经济（基线10，每级+0.6）。这些数值体现了"体力劳动者进步快、高级知识门槛高"的设计哲学。

### 3.3 职业技能加成

`CharacterTemplate` 通过 `List<SkillBoost>` 提供开局直接加级：

```csharp
[Serializable] public class SkillBoost {
    SkillType skillType;
    int bonus;   // +1 / +2 等，直接叠加到技能等级
}
```

## 4. XP 转换系统

### 4.1 经验获取流程

```
全系统行为 → EventBus.Publish(new SurvivalXpGained(amount, source))
    → SurvivalXPSystem.OnSurvivalXpGained() 
    → AddXP(amount) 累积到 _totalXP
```

`SurvivalXpGained` 是 `readonly struct`，包含 `Amount` 和 `Source` 字段，通过全局 `EventBus` 传递。任何系统都可以发布此事件来馈送经验。

### 4.2 兑换为技能点

```csharp
ConvertXPToPoints() → pointsEarned = _totalXP / XP_PER_SKILL_POINT(100)
```

- 兑换比：**100 XP = 1 技能点**
- 未兑换的经验碎片保留在 `_totalXP` 中，不会丢失
- 兑换成功后发布 `CharacterStatsChanged("xp_converted", "", pointsEarned)` 事件

### 4.3 消费技能点

| 方法 | 用途 | 事件 |
|------|------|------|
| `SpendPoint(SkillType)` | 花点升级技能 | `OnSkillLevelUp` delegate + `CharacterStatsChanged("skill_up", ...)` |
| `SpendAttributePoint(AttributeType)` | 花点升级属性 | `OnAttributeUp` delegate + `CharacterStatsChanged("attr_up", ...)` |

两种消费均调用 `SkillCostTable` 查询实时消耗，技能/属性上限均为 10 级。

## 5. 体力系统 (StaminaSystem)

### 5.1 核心数值

定义于 `GameConstants`：

| 常量 | 值 | 说明 |
|------|-----|------|
| `STAMINA_BASE_MAX` | 100 | 基础体力上限 |
| `STAMINA_PER_ENDURANCE` | 10 | 每点耐力增加的上限 |
| `STAMINA_DRAIN_RUN` | 8/秒 | 跑步体力消耗 |
| `STAMINA_DRAIN_MELEE` | 15 | 近战攻击一次性消耗 |
| `STAMINA_DRAIN_LABOR` | 10 | 劳作（伐木/挖矿）一次性消耗 |
| `STAMINA_OVERLOAD_MULT` | 1.5x | 超重时体力消耗倍率 |
| `STAMINA_REGEN_IDLE` | 15/秒 | 静止恢复速率 |
| `STAMINA_REGEN_WALK` | 10/秒 | 走路恢复速率 |
| `STAMINA_REGEN_PER_ENDURANCE` | 2/秒 | 每点耐力的额外恢复 |
| `STAMINA_EXHAUSTED_SPEED_MULT` | 0.6 | 体力耗尽时移速倍率 |

### 5.2 体力模型

```
每帧 Update:
  drain = 0
  if (isRunning) drain += 8
  if (overloaded) drain *= 1.5
  currentStamina -= drain * dt

  if (drain == 0):
    regen = (isWalking ? 10 : 15) + endurance * 2
    currentStamina = min(max, currentStamina + regen * dt)
```

**关键规则**：
- 只有**不消耗**时才能恢复（走路/静止），跑步时不会恢复。
- 体力为零时不可跑步、不可攻击（`IsExhausted` → `CanPerform()` 返回 false）。
- `RecalculateMax()` 根据当前耐力值动态重算 `MaxStamina`。

### 5.3 公共 API

| 方法 | 说明 |
|------|------|
| `CanPerform(float cost)` | 是否有足够体力执行操作 |
| `Consume(float amount)` | 直接扣减体力 |
| `ConsumeMelee()` | 近战攻击消耗（15点） |
| `ConsumeLabor()` | 劳作消耗（10点） |
| `SetRunning(bool)` / `SetWalking(bool)` | 设置跑步/走路状态 |
| `SetOverloaded(bool)` | 设置超重状态（消耗×1.5） |
| `FullRestore()` | 完全恢复（睡觉/使用物品） |

## 6. 负重惩罚系统

实现在 `PlayerCharacter.UpdateWeightPenalty()`：

- 监听 `InventoryChanged` 事件（经由 `EventBus`）
- 当当前重量 ÷ 有效最大重量 > 70% 时，移速修正从 1.0 线性下降
- 公式：`penalty = 1 - (ratio - 0.7) * 1.5`，下限 0.3
- 移速修正被夹在 `[PLAYER_MIN_MOVE_MODIFIER(0.1), PLAYER_MAX_MOVE_MODIFIER(2.0)]` 之间

```csharp
// 伪代码
ratio = currentWeight / effectiveMaxWeight
if ratio > 0.7:
    SetMoveSpeedModifier(max(0.3, 1 - (ratio - 0.7) * 1.5))
else:
    SetMoveSpeedModifier(1.0)
```

## 7. 职业系统 (CharacterTemplate)

### 7.1 ScriptableObject 结构

```csharp
[CreateAssetMenu(menuName = "Game/Character Template")]
public class CharacterTemplate : ScriptableObject {
    string templateName;
    string description;
    List<AttributeMod> attributeMods;      // 属性修正
    List<SkillBoost> skillBoosts;          // 技能加成
    ItemData[] startingEquipment;          // 开局装备（扩容容器）
    ItemRequirement[] startingItems;       // 开局物品
    bool startWithAIBot;                   // 是否开局带AI无人机
    BuildableData startingAIBotBuildable;  // AI机器人建造数据
}
```

### 7.2 职业应用流程

`SurvivalXPSystem.Start()` → `InitFromCharacterData()` → `ApplyProfession(template)` → `ProfessionApplier.ApplyStartingGear(template)`

1. 合并 `attributeMods`：将修正值加到 `RuntimeAttribute.value`，限制在 `[min, max]`
2. 合并 `skillBoosts`：将加成值直接加到 `RuntimeSkill.level`
3. 装备投放（两步顺序不可颠倒）：
   - 先 `EquipItem(装备)` → 扩容容器容量
   - 再 `AddItem(物品,数量)` → 放入已扩容的背包
4. AI无人机开局：在玩家附近寻找 NavMesh 位置，通过 `BuildModeController.PlaceStructureDirect()` 直接放置

## 8. PlayerCharacter 组件

挂载在玩家 GameObject 上，是一切角色数据的统一访问入口。

```csharp
public class PlayerCharacter : MonoBehaviour
```

### 8.1 生命周期

| 阶段 | 行为 |
|------|------|
| `Awake` | `ServiceLocator.Register(this)`、添加 `AIBotPilotInputLock`、加载/创建 `CharacterData` SO、获取 `Inventory` 组件 |
| `Start` | 若 `SurvivalXPSystem.Instance` 为空则自动创建 GameObject + 挂载组件 |
| `OnEnable` | 订阅 `InventoryChanged` 事件 |
| `OnDisable` | 取消订阅 `InventoryChanged` |
| `OnDestroy` | `ServiceLocator.Unregister<PlayerCharacter>()` |

### 8.2 便利属性

```csharp
public int Strength  => SurvivalXPSystem.Instance?.GetAttributeValue(AttributeType.力量) ?? 5;
public int Agility   => SurvivalXPSystem.Instance?.GetAttributeValue(AttributeType.敏捷) ?? 5;
public int Constitution => SurvivalXPSystem.Instance?.GetAttributeValue(AttributeType.体质) ?? 5;
public int Endurance => SurvivalXPSystem.Instance?.GetAttributeValue(AttributeType.耐力) ?? 5;
```

### 8.3 方法

| 方法 | 说明 |
|------|------|
| `GetAttributeValue(AttributeType)` | 获取属性值 |
| `GetSkillLevel(SkillType)` | 获取技能等级 |
| `SetMoveSpeedModifier(float)` | 设置移速修正（自动 clamp） |
| `GetMoveSpeedModifier()` | 获取当前移速修正 |

## 10. 存档系统接入

三个运行时组件均提供存档接口，供 `SaveLoadManager` 统一调用：

### 10.1 SurvivalXPSystem

```csharp
PopulateSaveData(PlayerSaveData pd)  // 导出 totalXP, availablePoints, attributes[], skills[]
RestoreFromSave(PlayerSaveData pd)   // 恢复 totalXP, availablePoints, attributes[], skills[]
```

导出格式：
- `attributes[]` → `List<AttributeSaveData>`（含 `type:string` 和 `value:int`）
- `skills[]` → `List<SkillSaveData>`（含 `skillType:string`, `level:int`, `xp:int`, `xpToNext:int`）

恢复时通过 `Enum.TryParse` 将字符串回映射为枚举。

### 10.2 StaminaSystem

```csharp
PopulateSaveData(PlayerSaveData pd)  // 导出 currentStamina
RestoreFromSave(PlayerSaveData pd)   // 恢复 currentStamina
```

### 10.3 存档数据容器

（定义于 `_Game/Systems/SaveLoad/Data/PlayerSaveData.cs`）

存档数据类包含上述字段，由 `SaveLoadManager` 统一序列化/反序列化（JSON + Newtonsoft.Json）。

## 11. 互连架构图

```
                         ┌──────────────────────────┐
                         │     ServiceLocator        │
                         │  (静态全局服务注册表)       │
                         └──────┬──────────┬────────┘
                    Register/Get  │          │  Register/Get
                ┌─────────────────┘          └──────────────┐
                ▼                                           ▼
    ┌─────────────────────┐              ┌──────────────────────────┐
    │   PlayerCharacter    │              │     SurvivalXPSystem     │
    │  (挂载 on Player GO) │◄─────────────│    (DontDestroyOnLoad)    │
    │                     │ 读取属性/技能 │    全局单例经验池          │
    │ - Strength/Agility  │              │                          │
    │   /Con/Endurance    │              │ - _totalXP, _availablePoints│
    │ - weightPenalty     │              │ - _skills[], _attributes[] │
    │ - moveSpeedModifier │              │ - OnSkillLevelUp delegate │
    └──┬──────────────────┘              │ - OnAttributeUp delegate  │
       │                                  └──┬───────────┬───────────┘
       │ Register                            │           │
       ▼                                  Register       │
    ┌─────────────────────┐                   │           │
    │   StaminaSystem     │        ┌──────────┘           │
    │  (挂载 on Player GO) │        ▼                      ▼
    │                     │    ┌─────────────────┐  ┌─────────────────┐
    │ - currentStamina    │    │ EventBus.Sub    │  │ EventBus.Pub    │
    │ - maxStamina        │    │ SurvivalXpGained│  │CharacterStatsCh │
    │ - isRunning/Walking │    └─────────────────┘  └─────────────────┘
    │ - overloadBonus     │           ▲                      │
    └─────────────────────┘           │                      ▼
                                      │              ┌──────────────┐
                    ┌─────────────────┘              │  UI 面板刷新  │
                    │ 任意系统发布                     │ (CharPanel等)│
                    │ SurvivalXpGained               └──────────────┘
```

**事件流**：
1. **XP 流入**：任何系统 → `EventBus.Publish(SurvivalXpGained)` → `SurvivalXPSystem.AddXP()`
2. **兑换流出**：UI 调用 `ConvertXPToPoints()` → `EventBus.Publish(CharacterStatsChanged("xp_converted"))` → UI 刷新
3. **升级流出**：UI 调用 `SpendPoint()` / `SpendAttributePoint()` → delegate + `EventBus.Publish(CharacterStatsChanged)` → UI 刷新
4. **负重联动**：`Inventory` 变化 → `EventBus.Publish(InventoryChanged)` → `PlayerCharacter.UpdateWeightPenalty()` → 移速修正
5. **体力联动**：`StaminaSystem` 每帧读取 `PlayerCharacter.Endurance` 用于 `RecalculateMax()` 和恢复率计算
